## 战略总纲

```
人设：互联网连续创业者，被产业基金朋友拉进能源行业
角度：不是专家，是翻译者——用互联网语言讲能源行业的机会
```

## 关键原则

### 1. 人设一致性

```
□ 永远不装"我一直都懂"。今天刚学会就说今天刚学会。
□ 被问住了就拍视频承认。被问住的视频可能比科普视频更火。
□ "互联网人"的身份标签永远不丢。这是最核心的差异化。
□ 朋友基金这个资源要持续提（不刻意，但要在内容里自然出现）。
```

### 2. 学习驱动内容

```
每天学习 → 每天有素材 → 每天拍视频
学习 = 内容引擎，不是额外负担。
```

### 3. 真实 > 完美

```
镜头前的紧张、说错话、卡壳，可以不剪掉。
"我重说一遍啊，刚才没讲清楚"比完美的一遍更有温度。
```

### 4. 不跟汤老师比深度

```
他的壁垒是行业经验和专业深度。
你的壁垒是真实、视角、资源。
不在他的战场打。在你的战场，他打不过你。
```

## 每日执行节奏

```
早上 1-2h：学习能源行业知识（政策文件、行业报告、跟行业人聊）
           → 学习过程中自然产生选题

下午 1h：拍摄+剪辑1条视频
          → 就用今天学到的东西
          → 念一遍。卡了就重录。不追求完美

晚上 30min：发布+回复评论+复盘数据+准备明天选题
```

## 风险预案

| 风险 | 应对 |
|------|------|
| 学习速度跟不上日更 | 减少到每周5更。质量>数量。学习笔记可以隔天发 |
| "不专业"的评论 | 不删。回复"确实，我刚学几个月，还在补课。有说错的地方欢迎指正" |
| 跟汤老师等被比较 | 不需要比。你是另一个赛道。互联网人看能源≠能源专家教技术 |
| 基金朋友不愿意被提及 | 提前沟通。泛称"朋友基金"，不点名具体机构和规模 |
| 内容数据不好 | 前30天不做数据焦虑。核心指标是"真不真实"，不是播放量 |
