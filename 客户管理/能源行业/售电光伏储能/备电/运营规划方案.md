> 定位：刚入行，学售电学储能。跟你一样从零开始。不同的是——有人带，看到了机会。
> 更新：2026-06-09 | 最终落点：不是专家，是跟你一样的新人，只是走运有人教

---

## 一、核心定位

### 一句话

**刚入行，学售电学储能。跟你一样从零开始。不同的是——有人带，看到了机会。把学到的分享出来。**

### 信任来源

```
别人说"我懂这行" → 观众：装的吧
你说"我刚学，合伙人教的" → 观众：这人说实话
```

### 人设三角

```
          真金白银在投
           /\
          /  \
         /    \
        /      \
  互联网18年 ──── 刚入行（真实）
  (跨界视角)    (不装大佬)
```

### 竞品差异

| | 汤老师新能源 | 增元AI棱镜 |
|------|------|------|
| 身份 | 能源行业专家 | **刚入行的新人，但有人带** |
| 讲什么 | 技术标准、政策原文 | **合伙人教我的、我们在做的、我刚学懂的** |
| 信任 | 行业经验 | **真实（刚学就刚学）+ 有信息源（合伙人）+ 真项目（巴南山东）** |
| 观众 | 想学能源知识 | 想了解行业机会、想一起入行的人 |

## 四、内容原则

```
□ "我在投" > "我在学" > "我在看" —— 每期内容优先从投资角度切入
□ "刚入行"不是弱点。大佬说自己不懂是谦虚，新人说自己刚学是真实
□ 项目进展持续更新。巴南签约、山东推进——每一条都是信任锚点
□ 政策翻译加"利好什么、利空什么"——投资人视角才有
□ 互联网18年别浪费。"电商、平台、算法"这些词，能源行业的人不会用
```

## 六、概念图谱系列（PPT+录音）

### 定位

每周一期长视频（5-10分钟），用PPT+录音的方式讲透一组能源概念的关系框架。不出镜，零拍摄成本。

### 对标参考

大山（抖音）"源网荷储虚拟电厂绿色微电网"——金字塔框架图+分层拆解+场景串联演示+总结问答。观众看完的感觉是"通透"。

### 与日常短视频的分工

| | 日常短视频 | 概念图谱系列 |
|------|------|------|
| 形式 | 分身出镜+AI画面 | PPT+录音 |
| 时长 | 2-5min | 5-10min |
| 目标 | 拉新、建立人设 | 深度收藏、建立专业信任 |
| 频率 | 日更/隔日更 | 每周1期 |
| 内容 | 单点观点/事件 | 系统性框架拆解 |

### 系列选题（首期10个）

| 期 | 选题 | 核心框架 |
|------|------|------|
| 1 | 源网荷储到底是什么 | 四字拆解 + "源随荷动"为什么不行了 |
| 2 | 虚拟电厂：不拥有设备凭什么叫电厂 | 滴滴/操作系统类比 + 调度逻辑 |
| 3 | 绿色工业微电网：工厂自己的发电站 | 自给自足村庄类比 + 真实案例 |
| 4 | 绿电直供：一条直达工厂的电力高速 | 为什么需要直供 + 谁适合 |
| 5 | 零碳园区：前面几样的集大成者 | 全景图 + 落地四步 |
| 6 | 售电公司代运营储能 | 换庄逻辑（已有脚本） |
| 7 | 分时电价取消后，游戏规则怎么变了 | 旧规则vs新规则对比 |
| 8 | 碳资产/绿证入门框架 | 绿证=碳资产存折类比 |
| 9 | 储能产业链全景 | 上中下游树状图（已有B型图文） |
| 10 | 电力现货市场入门 | 15分钟定价机制 + 对工厂意味着什么 |

### PPT制作规范

- 工具：Canva（搜索"教育培训PPT"模板，选白底+细线+单色点缀风格）
- 比例：9:16竖版（匹配抖音全屏）
- 配色：备电暖调——琥珀[#C89050]标题色，暖深灰[#1a1e24]正文色，白底
- 字号：标题≥48pt，正文≥28pt（手机小屏可读）
- 每页一个核心信息点，不堆砌

### 录音规范

- 剪映录音，逐页念稿
- 节奏：关键概念放慢，过渡段正常语速
- 语气：像在给朋友讲一个你刚学会的东西，不是在上课

## 七、运营节奏

### 第1个月
- 日更。重点打"I线（投资笔记）"——这是别人做不了的内容
- 不追数据。追"真不真实"
- 核心指标：有没有人说"关注了，真投资人"

### 第2-3个月
- 基于数据调整内容线配比
- 私信开始出现工厂客户和居间人
- 搭建培训体系

### 第3-6个月
- 居间人网络扩展
- 代理交易平台跑通
- 人设从"刚入行"自然过渡到"投了多个项目"

## 八、互动策略

```
□ 有人问专业问题：会的就答，不会就说"我也刚学到，我的理解是…"
□ 有人质疑投资人身份：不辩解。发一期"巴南项目是怎么谈下来的"
□ 有人问怎么参与：私信聊，不在评论区推销
```
