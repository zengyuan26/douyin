## 二、投资方向判断

### 核心方向：工业微电网EMC

```
微电网 = 分布式光伏 + 储能 + 充电桩 + AI调度
EMC = 投资方出钱，工厂出屋顶。省了电费分钱。
```

**为什么这个方向最值得投**：
- 工厂零投入，决策门槛低
- 25年合同，现金流稳定
- AI调度多赚15-30%，技术壁垒
- 绿证+碳核算新规，工厂必须绿电

### 延伸方向：代理交易服务

2026年15分钟电价+现货市场，储能投资方和售电公司都需要AI交易能力。代理交易 = 帮他们做交易，收服务费。

## 三、CVC视角 vs 传统PE/VC

| | 传统PE/VC | CVC |
|------|------|------|
| 投资逻辑 | 财务回报 | 产业协同+财务回报 |
| 投后管理 | 派董事、看报表 | 下场做渠道、做交易 |
| 退出周期 | 3-5年 | 更长，但现金流稳定 |
| 在能源行业的优势 | 资金量大 | **懂产业+有产业资源** |

**CVC在能源行业的核心优势**：不只是给钱，是给渠道、给交易能力、给客户。这也是备电做代理交易平台的逻辑——投完之后，帮被投项目跑起来。
