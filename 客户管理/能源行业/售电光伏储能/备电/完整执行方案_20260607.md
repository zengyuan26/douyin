# EMC 居间人渠道平台 · 完整执行方案

> 核心战略：轻资产撮合 | 全国市场 | 居间人渠道网络 | 睡后收入
> 日期：2026-06-07
> 更新：调整为三阶段递进策略——先用行业知识建立信任，逐步过渡到居间人招募

---

## 核心逻辑：信任在前，招募在后

```
阶段1（蓄水）：行业知识 + 工厂省钱内容
  → 观众认知：这个人懂行、靠谱、说的东西有用
  → 潜在居间人暗中观察：这行业好像挺赚钱的？

阶段2（过渡）：行业知识 + 行业机会信号
  → 适当展示居间人案例、佣金模型
  → 观众开始问：我能做吗？怎么加入？

阶段3（爆发）：居间人招募主力
  → 全力推进居间人网络
  → 培训体系+工具包全面分发
```

**核心原则**：一个人不会加一个上来就喊"跟我干"的账号。但他会加一个"说的东西有料、看起来这行业有钱赚"的账号。信任到了，他会主动问你怎么加入。

## 阶段0：内容蓄水期（第1-8周）

### 0.1 目标

```
核心目标：建立"EMC/售电行业专家"的信任认知
次要目标：让观众隐约感受到"这行业有钱赚"
不做的事：不主动推居间人招募
```

### 0.2 内容配比

| 类型 | 占比 | 目的 |
|------|------|------|
| 行业知识/政策解读 | 35% | 建立专业权威 |
| 工厂省钱算账 | 35% | 吸引工业用户+展示业务价值 |
| 行业观点/赚钱信号 | 20% | 埋下"这行业有机会"的伏笔 |
| 个人人设/IP | 10% | 拉近距离，让人记住你 |

### 0.3 核心任务

```
□ 抖音账号从0启动，日更
□ 跑通拍摄-剪辑-发布流程（30min/条）
□ 验证哪些内容类型互动最高
□ 积累1000+粉丝基础
□ 产出第一批工厂咨询线索（不追求签约，先验证内容→咨询的转化）
□ 让观众形成认知：增元 = 懂能源、懂工厂省钱
```

### 0.4 居间人伏笔（只做不说）

阶段0**不主动**招募居间人，但做三件事为后续铺路：
- 每条视频结尾引导评论（"打'学习'领取资料"→ 收集潜在居间人线索）
- 私信回复时，如有工厂资源的询问，简单介绍居间模式但不强推
- 建立"潜在居间人"标签，在微信里分组暂存

## 阶段1：基建期（第9-12周）

### 1.1 目标

在内容信任建立的基础上，**开始释放居间人信号**，同时搭建培训体系。

### 1.2 内容配比

| 类型 | 占比 | 变化 |
|------|------|------|
| 行业知识/工厂省钱 | 50% | ↓ 从70%降到50% |
| 行业机会/居间信号 | 35% | ↑ 新增，展示佣金/案例/行业前景 |
| 居间人招募（轻度） | 10% | 新增，"有兴趣的私信我" |
| 个人人设 | 5% | 维持 |

### 1.3 核心任务

```
□ 录制居间人培训课（8节）——利用阶段0的案例和素材
□ 开发用电诊断计算器
□ 编写居间人操作手册
□ 制作EMC提案模板
□ 招募3-5个种子居间人（从阶段0积累的潜在居间人中筛选）
□ 跑通第一个居间人出单流程
```

### 1.4 居间人冷启动

种子居间人不是从抖音公开招募的，而是从阶段0的私信咨询中**私下筛选**：
- 在评论区/私信里问过"这个怎么合作"的人
- 自我介绍有工厂资源的
- 看过你的内容一段时间、有一定信任基础的

## 阶段2：验证期（第13-20周）

### 2.1 目标

带种子居间人跑出第一批签约，验证整个模式。

### 2.2 内容配比

| 类型 | 占比 |
|------|------|
| 行业知识/工厂省钱 | 35% |
| 居间人案例/佣金展示 | 30% |
| 居间人招募 | 25% |
| 个人人设 | 10% |

### 2.3 核心任务

```
□ 种子居间人各跑3个工厂
□ 增元协助关键成交
□ 至少3个工厂签约
□ 案例资产化（每个签约→文章+短视频+数据卡片）
□ 培训课迭代v2.0
□ 工具包迭代（根据实际使用反馈）
```

## 阶段3：规模期（第21-30周）

### 3.1 目标

从10个居间人到100个居间人。内容全面转向居间人招募。

### 3.2 内容配比

| 类型 | 占比 |
|------|------|
| 居间人招募/赋能 | 50% |
| 居间人案例晒单 | 20% |
| 行业知识/工厂 | 20% |
| 个人人设 | 10% |

### 3.3 核心任务

```
□ 居间人分级体系上线
□ 微信群分层运营
□ 区域扩展（广东、江苏、山东优先）
□ 建立多资金方池
□ 增元逐步抽身日常运营
```

## 资金方策略（全阶段）

```
阶段0-1：确认朋友基金可用额度、投资偏好、决策流程
        → 从第1周就建立备选资金方清单
阶段2：  实际对接项目 → 验证资金方响应速度和放款能力
        → 如有问题，启用备选
阶段3+： 建立2+稳定资金方，平台化运作
```

## 风险预案

| 风险 | 阶段 | 应对 |
|------|------|------|
| 阶段0内容没流量 | 0 | 测试钩子，投小预算豆荚，专注内容质量 |
| 阶段1潜在居间人转化难 | 1 | 不急，继续积累信任；先找已有关系的人做种子 |
| 种子居间人不出单 | 2 | 增元深度参与，手把手带 |
| 基金额度不足 | 2+ | 立即启用备选资金方 |
| 内容信任建立太慢 | 0-1 | 增加政策热点蹭流量，增加算账冲击力内容 |
