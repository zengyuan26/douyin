# 【机会分析】EMC 居间人渠道模式

> 分析框架：纳瓦尔业务分析模型 | 日期：2026-06-07

---

## 一、机会本质

**一句话定义：建立一个可复制的EMC项目分发网络——线上内容吸引居间人，用培训和工具包武装他们去拓客，你只在高价值环节（匹配资金方、协助成交）出现，赚取撮合佣金。**

本质上，你不是在卖电，你是在卖一个"让有工厂资源的人能安全变现"的加盟系统。

## 二、纳瓦尔框架分析

### 1. 定位分析

```
你的定位矩阵：

                    高价值输出                    低价值输出
              ┌──────────────────────┬──────────────────────┐
      擅长    │  ★ 渠道赋能者         │                      │
              │  培训+工具+成交辅助   │                      │
              ├──────────────────────┼──────────────────────┤
      不擅长  │                      │                      │
              │                      │                      │
              └──────────────────────┴──────────────────────┘
```

**你的定位不是"售电专家"，而是"EMC渠道赋能者"。**

这是三个角色中最值钱的那个位置：

```
资金方（朋友基金）  ←→  你（渠道赋能+撮合）  ←→  居间人（地方获客）  ←→  工厂（终端客户）
     │                        │                        │
  出钱、承担风险          你的位置！           有人脉、缺变现方式
```

你不需要是电力工程师（那是AI方案的事），不需要是基金经理（那是朋友的事），你需要的是：**让全国各地有工厂人脉的人，都能用你的体系赚到钱。**

### 2. 价值分析

你的价值层级：

```
层级4：认知差价值 ★★★★★（最高）
  → EMC商业模式的理解、政策解读、算账能力
  → 你知道工厂电费怎么省、光储怎么建、合同怎么签

  → 连接资金方（朋友基金）和项目（居间人找来的工厂）
  → 全国范围内的供需匹配
  → 每撮合成功一笔，你抽佣金

层级2：执行价值 ★★（中等）
  → 培训居间人、协助成交、提供工具包
  → 这部分可以标准化，不需要你长期投入

层级1：体力价值 ☆（不做）
  → 跑工厂、做勘测、安装施工
  → 这些都不是你的事
```

**关键判断：你的核心价值在层级3+4，其他都是别人的事。**

这是最关键的部分——**哪些可以复制？哪些必须你亲自做？**

#### 🔴 必须你做的事（当前阶段，无法复制）

| 事项 | 为什么必须你 | 占时间比例 |
|------|-------------|-----------|
| 内容出镜（个人IP） | 居间人信任的是"增元"这个人脸和判断力 | 20% |
| 复杂项目成交协助 | 大项目谈判、资金方对接、合同把关 | 15% |
| 居间人网络核心关系维护 | 头部的20%居间人产出80%的业绩 | 10% |
| 战略判断（政策/市场方向） | 没人能替代你的判断力 | 5% |

#### 🟢 可以标准化+复制的事（高睡后收入潜力）

| 事项 | 复制方式 | 睡后收入贡献 |
|------|----------|------------|
| **居间人培训体系** | 录播课 + 直播答疑 + 操作手册 | ⭐⭐⭐⭐⭐ |
| **工具包** | 用电诊断计算器、EMC收益测算表、提案模板 | ⭐⭐⭐⭐⭐ |
| **内容创作** | 选题库 + 脚本模板（居间人自己拍本地版） | ⭐⭐⭐⭐ |
| **客户跟进SOP** | 标准化跟进流程、话术库、常见问题FAQ | ⭐⭐⭐⭐ |
| **项目筛选标准** | 什么样的工厂适合做EMC、多大的用电量才值得跑 | ⭐⭐⭐⭐ |
| **合同模板** | 已经有EMC合同模板，根据不同项目类型配置 | ⭐⭐⭐ |
| **成功案例库** | 每成交一单就变成一个可复用的案例 | ⭐⭐⭐ |
| **居间人招募SOP** | 从抖音评论→进群→听课→出单的完整转化路径 | ⭐⭐⭐ |
| **佣金结算体系** | 标准化分钱规则，居间人清楚自己拿多少 | ⭐⭐⭐ |

#### 复制路径：三步走

```
阶段1（1-3个月）：标准化
  ├── 录一套居间人培训课（6-8节，每节15分钟）
  ├── 做出用电诊断计算器（输入月用电量 → 输出省多少钱）
  ├── 写出居间人操作手册（怎么找工厂、怎么开口、怎么跟进）
  └── 验证：先带3-5个居间人跑一遍，完善流程

阶段2（3-6个月）：规模化
  ├── 抖音内容持续引流 → 居间人持续进群
  ├── 群里自动化运营：新人引导 → 课程推送 → 答疑 → 出单激励
  ├── 跑出10个以上的成功案例，每个案例都是复制的素材
  └── 验证：月均稳定5+居间人出单

阶段3（6-12个月）：平台化
  ├── 居间人分级体系（初级/中级/高级，不同分佣比例）
  ├── 区域合伙人制度（省级/市级独家代理）
  ├── 居间人之间互相推荐（A城市的工厂在B城市有分厂 → 推荐给B城市居间人）
  └── 验证：你不用参与日常运营，系统自动运转
```

### 5. 经验复利分析

**这个模式的复利效应极强：**

```
复利来源一：居间人网络的指数增长
  ├── 第1个月：你找到5个居间人
  ├── 第3个月：这5个人各自推荐了2个 → 15个
  ├── 第6个月：网络效应产生，居间人互相推荐 → 50个
  └── 第12个月：品牌效应，居间人主动找来 → 100+

  每个居间人 = 一个24小时在线的本地销售
  他们的经验积累 = 成交率持续提升
  你不给他们发工资，他们赚佣金

复利来源二：案例资产的滚雪球
  ├── 每成交一个工厂 = 一个新案例
  ├── 案例越多 → 信任越强 → 成交越快
  ├── 第1单：没有案例，纯靠你一张嘴
  └── 第50单：50个工厂都在省钱，居间人拿去说服新客户

复利来源三：政策认知的护城河
  ├── 电力市场化改革越深入 → 工厂越看不懂
  ├── 你看得越久 → 判断越准 → 价值越大
  └── 新人入行需要1-2年才能达到你的水平

复利来源四：品牌资产
  ├── "增元AI棱镜" = EMC渠道赋能第一品牌
  ├── 工厂老板自己刷到你的视频 → 主动找本地居间人
  └── 品牌越大 → 获客成本越低 → 居间人越愿意跟你合作
```

**复利判断：✅ 这是典型的"做得越久越值钱"的模式。经验、网络、案例、品牌四个飞轮都在加速。**

## 三、八条关键问题

```
□ 0. 项目通过筛选标准了吗？
   → 决策周期：工厂主决策周期中等（几周），需要通过算账驱动
   → 复购：EMC签约25年，极强的客户锁定
   → 客户群体：全国工厂，客群巨大
   → 可视性：电费单对比 + 光伏板实拍，视觉冲击力强
   → 专业门槛：需要懂EMC，但居间人不需要，你提供培训
   → 客单价：你拿撮合佣金，单项目佣金可观
   ✅ 通过筛选

□ 1. 这个机会的本质是什么？
   → 建立在EMC行业知识+资金供给+渠道网络之上的撮合平台
   → 不是售电公司，是EMC的"贝壳找房"

□ 2. 我的定位是什么？
   → 渠道赋能者 + 撮合者
   → 不做执行，只做赋能和成交辅助

□ 3. 我能提供的独特价值是什么？
   → EMC知识体系（培训） + 资金方对接 + 成交能力

□ 4. 我能用什么杠杆放大价值？
   → 媒体（获客）+ 人力（居间人执行）+ 资本（朋友基金）+ 代码（工具）

□ 5. 子弹能否剥离？
   → 大部分可以：培训、工具、SOP、案例都可以标准化
   → 个人IP暂时无法剥离，但IP本身也在变成资产

□ 6. 关键决策点是什么？
   → ① 资金方的持续供给能力（基金的规模和持续性）
   → ② 第一批居间人的质量和产出（0→1比1→100更难）
   → ③ 佣金分配模型（居间人拿多少、你拿多少、资金方拿多少）

□ 7. 风险是什么？
   → 资金风险：基金额度有限，有项目没钱做
   → 声誉风险：居间人乱承诺导致客户纠纷
   → 政策风险：电力政策变化影响EMC经济模型
   → 竞争风险：模式被复制

□ 8. 退出机制是什么？
   → 平台化后可以转让整个居间人网络
   → 或者保持轻资产运营，持续赚取佣金流
   → 不需要退出，本身就是被动收入系统
```

| 品牌授权 | 持续（区域代理费） | 几乎不需要 | ⭐⭐⭐⭐⭐ | 高（需品牌建立后） |

### 睡后收入路线图

```
当前 ↓
靠增元个人能力成交，一单一单赚佣金
期 ↓（3个月后）
培训课录好了，居间人自己看课就能学会。增元只在大项目出现。
中 ↓（6个月后）
工具包+案例库自动运转。居间人自己用计算器算账、用模板出方案。
期 ↓（12个月后）
居间人网络有100+人，完全自发运转。新人老人互相带。
后 ↓
增元的IP = 品牌 = 自动获客引擎
每新进来一个居间人 → 自动培训 → 自动出单 → 增元只收管理费/分佣
```

### 最关键的三件事

**第一：培训课必须先录出来**

这是所有睡后收入的基础。没有标准化培训，每个居间人都要你手把手教，你永远无法抽身。6-8节课，覆盖：
1. EMC是什么，为什么工厂需要
2. 怎么找到一个工厂的决策人
3. 怎么开口聊电费（开场白话术）
4. 怎么用计算器算账（实操演示）
5. 怎么推进到签合同
6. 怎么处理常见拒绝
7. 成交后怎么拿佣金
8. 怎么发展下级居间人

**第二：用电诊断计算器是钩子**

一个简单的在线工具：输入月用电量 → 自动算出 EMC 模式下一年能省多少钱。这是居间人拓客的核心武器——不需要懂技术，输个数字就能出结果。工厂老板看到数字，信任就建立了一半。

**第三：佣金结构必须清晰且诱人**

居间人的动力来自看到别人赚到钱。佣金结构要：
- 清晰：每单拿多少，什么时候发，一目了然
- 诱人：单项目佣金足够大，值得他花时间
- 层级：推荐其他居间人有额外的推荐奖励（但不建议超过两级，避免传销嫌疑）
