# 备电 · 完整执行方案

> 定位：刚入行，学售电学储能。跟你一样从零开始。不同的是——有人带，看到了机会。
> 更新：2026-06-09

---

## 战略总纲

```
身份：互联网老兵，跟CVC合伙人一起干新能源
分工：他管钱、看方向、带我看项目。我搞品牌渠道、跑落地、搭平台
状态：刚入行，但有人带。学到的东西第一时间分享
壁垒：
  - 真项目（巴南已落地、山东已谈妥）→ 不可伪造
  - 合伙人行业认知 → 通过我的内容传递出来
  - 互联网18年视角 → 独特的跨界洞察
  - 真实（刚入行、有人带、在学）→ 观众信
```

### 定位进化

```
旧：CVC投资人（不完全准确——钱不是我的）
新：跟CVC合伙人一起干（准确——他是船长，我是大副）

观众感知：
  旧 → "这人是不是在装大佬？"
  新 → "这人跟在大佬身边干活，学的都是真东西"
```

## 阶段1：人设建立（第1-4周）

### 核心任务

```
□ Day1 开号：「跟CVC合伙人一起投新能源。他管钱，我干活。刚入行。」
□ 日更30天。重点打"合伙人教的"和"我们在做什么"——这两条线别人做不了
□ 让观众形成认知："这人不是大佬，但跟在大佬身边，学的是真东西"
```

### Day1 开号视频

**「跟CVC合伙人一起投新能源。刚入行。」**

讲清楚：干了18年互联网 → 现在跟CVC合伙人一起干新能源 → 他管钱、看方向；我搞品牌渠道、跑项目 → 刚入行，天天跟在他后面学 → 巴南已落地、山东已谈妥 → 他教我的、我们在做的，分享出来。

## 关键原则

```
□ 永远说"我们"，不说"我"。准确的权力关系，反而更可信。
□ "合伙人说"是内容钩子——这句话意味着接下来是行业干货。
□ 项目进展别藏着。巴南签约、山东推进，每一条都是信任锚点。
□ 互联网18年别浪费。你用互联网语言讲能源，行业老炮做不到。
```
