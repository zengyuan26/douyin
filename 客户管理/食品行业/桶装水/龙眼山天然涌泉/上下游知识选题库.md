## 八、内容创作模板

### 【知识型内容公式】

```
标题：XXX的秘密/XXX的真相/XXX区别

结构：
1. 抛出问题 → 引发好奇
2. 专业知识 → 建立信任
3. 案例展示 → 生动形象
4. 总结建议 → 实用价值
5. 行动引导 → 关注/私信
```

### 【对比型内容公式】

```
标题：XXX和XXX有什么区别？/XXX vs XXX

结构：
1. 引入对比对象 → 明确主题
2. 分别介绍 → 各自特点
3. 对比分析 → 优劣势展示
4. 适用场景 → 给出建议
5. 结论引导 → 选择推荐
```
