### 3.2 B端客户分析

#### 3.2.1 茶楼/茶馆（核心客户）

| 维度 | 内容 |
|------|------|
| **重庆市场** | 茶楼数量约5000+家 |
| **核心需求** | 水质适合泡茶（软水/低硬度）、口感好、稳定配送、性价比 |
| **决策者** | 茶楼老板、茶艺师、采购负责人 |
| **痛点** | 循环桶卫生隐患、二次污染投诉、品牌选择困难、配送不及时 |
| **采购特点** | 长期合作、稳定供应、月结付款 |
| **黑话关键词** | 茶水、茶位、走水、温杯、醒茶、闷泡、回甘、生津、喉韵 |

**茶楼老板画像**：
- 年龄：40-55岁
- 特点：注重茶艺和待客之道，茶水是招牌体验的核心
- 担忧：怕客户说茶不好喝其实是水的问题；怕一次性桶成本太高撑不住
- 转化钩子：水质对了，茶才香；选对水，省心又省钱

| **采购特点** | 批量采购、价格敏感、发票需求 |

#### 3.2.3 企业/写字楼

| 维度 | 内容 |

| **决策者** | 行政、HR、采购 |

| **采购特点** | 年度合同、月结付款、品牌选择需汇报 |

- 转化钩子：泡茶用水选对了，老板满意还省钱
