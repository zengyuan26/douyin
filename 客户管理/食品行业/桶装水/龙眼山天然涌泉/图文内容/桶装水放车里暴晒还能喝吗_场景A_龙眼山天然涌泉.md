# 图文内容模板

> **【强制规格】内容形式：图文 | 图片数量：5张 | 比例：9:16竖版 1080×1920px**

---

## 一、核心内容规则

> **【必读】图文内容三大核心规则，每次创作必须遵守**

### 1.1 人-货-场闭环

> **内容逻辑：人在场景下用产品/服务 → 遇到问题 → 苦恼不能解决 → 情绪变化 → 解决方案**

| 要素 | 说明 |
|------|------|
| **人** | 目标用户：家庭主妇、宝爸宝妈 |
| **货** | 产品/服务：暴晒的桶装水（问题）→ 正确储存的水/一次性桶（解决方案） |
| **场** | 使用场景：接孩子放学、车内高温环境 |

**闭环要求：**
- 每张图都要体现"人-货-场"中的某个环节
- 内容要有代入感，让用户觉得"说的就是我"
- 解决方案要直接对应前面提出的问题

### 1.2 情绪递进（"已经"原则）

> **情绪节奏：无奈 → 疑惑 → 爆发 → 释然 → 希望 → 行动**

| 情绪阶段 | 图号 | 情绪词 | 状态描述 |
|----------|------|--------|----------|
| 初始 | 图1 | 纠结/戳心 | 问题已经发生，用户正在纠结 |
| 代入 | 图2 | 回忆/共鸣 | 用户回想自己也遇到过 |
| 爆发 | 图3 | 担忧/焦虑 | 情绪达到顶点，担心孩子健康 |
| 揭秘 | 图4 | 释然/顿悟 | 揭示真相，学会正确处理 |
| 方案 | 图5 | 放心/行动 | 给出正确方案和替代选择 |

**"已经"原则示例：**
- ❌ "如果桶装水在车里暴晒了"（假设性，语气弱）
- ✅ "车里放了一天的水 我已经纠结了3次"（已完成式，情绪强）
- ❌ "暴晒过的水可能有问题"（猜测性）
- ✅ "暴晒过的水 不是不能喝 是要看桶"（确定性，已揭秘）

### 1.3 大字原则

> **核心要点必须用大字突出，一眼就能看到关键信息**

| 层级 | 字数 | 位置 | 作用 |
|------|------|------|------|
| 主标题 | 7字以内 | 图上1/3处 | 第一眼看到，决定点不点 |
| 痛点词 | 3-5字 | 图中突出位置 | 引发共鸣，一眼戳心 |
| 揭秘语 | 5-8字 | 图中C位 | 颠覆认知，金句炸场 |
| 方案名 | 5字以内 | 图中显眼位置 | 直接给答案 |
| 行动语 | 10字以内 | 底部 | 引导互动/私信 |

**大字规范：**
- ⚠️ **每张图必须有1-2个大字金句**
- ⚠️ **大字用来说"结果"，小字用来说"原因"**
- ⚠️ **禁止整段文字，字数要少，冲击力要强**

## 三、标题设计

> **【标题规范】**
> - 主标题：7个字以内（含有客户问题或问题答案）
> - 副标题：10个字以内（直接给价值或行动引导）
> - 标题要简洁，避免冗长

### 3.1 主标题

> 桶装水放车里晒了一天（8字，略超）
> 车内暴晒的水 还能喝吗（8字，略超）
> 车里晒过的水 能喝吗（7字，可行）

### 3.2 副标题

> 纠结扔了可惜 喝了担心（10字，情绪强）

### 3.3 封面建议

| 要素 | 说明 |
|------|------|

| 副标题 | 10字以内，揭示纠结 |

## 七、制作规范

### 设计规格

```
✓ 图片数量：5张（9:16竖版）
✓ 主标题：7字以内
✓ 副标题：10字以内
✓ 图表为主，文字精简
✓ 每张图文字不超过30字/行
✓ 每张必须有场景图（禁止纯色/渐变背景）
✓ 背景与内容相关
✓ 禁止出现手机外框/手机Mockup
✓ 禁止使用emoji和特殊符号
✓ 严禁堆砌文字，保持简洁易读
```
