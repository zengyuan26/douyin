# 图文内容模板

> **【强制规格】内容形式：图文 | 图片数量：5张 | 比例：9:16竖版 1080×1920px**

---

## 一、核心内容规则

> **【必读】图文内容三大核心规则，每次创作必须遵守**

### 1.1 人-货-场闭环

> **内容逻辑：人在场景下用产品/服务 → 遇到问题 → 苦恼不能解决 → 情绪变化 → 解决方案**

| 要素 | 说明 |
|------|------|
| **人** | 目标用户：泡茶爱好者、茶楼老板、家庭用户、企业行政 |
| **货** | 产品/服务：自来水（问题）→ 矿泉水/泡茶专用水（解决方案） |
| **场** | 使用场景：家庭茶桌、茶楼、办公室茶水间 |

**闭环要求：**
- 每张图都要体现"人-货-场"中的某个环节
- 内容要有代入感，让用户觉得"说的就是我"
- 解决方案要直接对应前面提出的问题

### 1.2 情绪递进（"已经"原则）

> **情绪节奏：无奈 → 疑惑 → 爆发 → 释然 → 希望 → 行动**

| 情绪阶段 | 图号 | 情绪词 | 状态描述 |
|----------|------|--------|----------|
| 初始 | 图1 | 无奈/戳心 | 问题已经发生，用户已经被困扰 |
| 代入 | 图2 | 疑惑/回忆 | 用户回想自己也遇到过 |
| 爆发 | 图3 | 爆发/共鸣 | 情绪达到顶点，说出用户心声 |
| 揭秘 | 图4 | 释然/顿悟 | 揭示真相，用户恍然大悟 |
| 方案 | 图5 | 希望/期待 | 给出解决方案，用户看到希望 |

**"已经"原则示例：**
- ❌ "如果你也有这个问题"（假设性，语气弱）
- ✅ "自来水怪味 我已经忍了5年"（已完成式，情绪强）
- ❌ "可能是因为水不好"（猜测性）
- ✅ "不是茶叶的问题 是水的问题"（确定性，已揭秘）

### 1.3 大字原则

> **核心要点必须用大字突出，一眼就能看到关键信息**

| 层级 | 字数 | 位置 | 作用 |
|------|------|------|------|
| 主标题 | 7字以内 | 图上1/3处 | 第一眼看到，决定点不点 |
| 痛点词 | 3-5字 | 图中突出位置 | 引发共鸣，一眼戳心 |
| 揭秘语 | 5-8字 | 图中C位 | 颠覆认知，金句炸场 |
| 方案名 | 5字以内 | 图中显眼位置 | 直接给答案 |
| 行动语 | 10字以内 | 底部 | 引导互动/私信 |

**大字规范：**
- ⚠️ **每张图必须有1-2个大字金句**
- ⚠️ **大字用来说"结果"，小字用来说"原因"**
- ⚠️ **禁止整段文字，字数要少，冲击力要强**

## 七、制作规范

### 设计规格

```
✓ 图片数量：5张（9:16竖版）
✓ 主标题：7字以内
✓ 副标题：10字以内
✓ 图表为主，文字精简
✓ 每张图文字不超过30字/行
✓ 每张必须有场景图（禁止纯色/渐变背景）
✓ 背景与内容相关
✓ 禁止出现手机外框/手机Mockup
✓ 禁止使用emoji和特殊符号
✓ 严禁堆砌文字，保持简洁易读
```
