# 图文内容模板规则

> **【强制规格】**
> - 形式：9:16 竖图，3～9 张轮播
> - 画面：真实场景图，禁止纯色/渐变背景
> - 文案：大字短句，一行不超过 10 字
> - 风格：口语化、接地气
> - 前 3 秒字幕：大、要戳心

---

## 一、图文基础规范

### 1.1 图片规格

| 项目 | 要求 |
|------|------|
| **尺寸** | 1080×1920px（9:16 竖版） |
| **张数** | 3～9 张（建议 5～7 张） |
| **背景** | 真实场景图，禁止纯色/渐变/抽象背景 |
| **排版** | 竖版阅读习惯，文字在上半部分或底部 |

### 1.2 文案规范

| 项目 | 要求 | 示例 |
|------|------|------|
| **单行字数** | ≤10 字 | 好茶被怪味毁了（8字） |
| **语速感** | 口语化，像说话 | 泡茶那股塑料味 别忍了 |
| **情绪感** | 戳心、有共鸣 | 难怪客人再也不来了 |
| **前 3 秒** | 字幕最大最醒目 | 桶装水有塑料味？ |

### 1.3 画面规范

```
【禁止】
├── ❌ 纯色背景
├── ❌ 渐变背景
├── ❌ 抽象图形背景
├── ❌ 手机/电脑 Mockup
└── ❌ 空白背景

【允许】
├── ✅ 真实场景图（茶席/厨房/办公室/超市）
├── ✅ 产品实拍图
├── ✅ 人物场景图
└── ✅ 图表+场景组合
```

## 三、封面设计规范【最关键】

> **封面是决定点击率的核心！前 3 秒字幕必须大、要戳心！**

### 3.1 封面文案公式

```
【公式】痛点/疑问 + 情绪词 + 好奇钩子

【好示例】
├── 你家桶装水有塑料味？
├── 泡茶那股怪味 其实是这个
├── 难怪客人再也不来了
├── 老板不知道的秘密
└── 用了3年才发现被骗了

【差示例】❌
├── 桶装水知识科普（太专业）
├── 关于水质你需要知道的事（太泛）
└── 产品介绍（太硬）
```

### 3.2 封面画面要求

```
【背景图】真实场景 + 情绪氛围
├── 茶席泡茶场景（茶艺师表情凝重）
├── 家庭饮水机（桶在旁边）
├── 超市货架（矿泉水琳琅满目）
└── 人物表情要贴合情绪

【文字区域】
├── 主标题：≤10 字，要大、要戳心
├── 副标题：≤10 字（可选）
└── 底部：≤6 字（引导往下看）

【禁止】❌ 纯色背景 ❌ 纯产品图
```

## 四、文案写作规范

### 4.1 文案类型

| 类型 | 特点 | 示例 |
|------|------|------|
| **痛点型** | 戳痛点，引发共鸣 | 泡茶那股塑料味 别忍了 |
| **疑问型** | 设疑问，引发好奇 | 为什么你泡的茶没有茶香？ |
| **揭秘型** | 揭秘秘密，引发关注 | 桶装水这个秘密没人告诉你 |
| **对比型** | 对比差异，引发兴趣 | 同一款茶 水不同 差太多 |
| **数字型** | 数字说话，更有说服力 | 3步辨别问题桶装水 |

### 4.2 文案风格

```
【口语化】
├── ❌ 桶装水开封后应尽快饮用
├── ✅ 开封超过7天 别喝了

├── ❌ 水质对泡茶有重要影响
├── ✅ 水不对 茶再好也白费

└── ❌ 选择低硬度矿泉水
    ✅ 泡茶用水这种才对

【情绪化】
├── 痛点：好茶被怪味毁了
├── 震惊：你以为喝的是水？
├── 释然：原来问题出在这
└── 期待：这样选就对了

【场景化】
├── 接水时那股塑料味
├── 泡了好茶客人却不喝
└── 饮水机聪明座里居然这么脏
```

### 4.3 每张图的文案要求

```
【单行字数】≤10 字
├── 好：好茶被怪味毁了（8字）
├── 好：水有怪味 快检查这3处（10字）
└── 差：为什么你家的桶装水喝起来有一股塑料味道（20字）❌

【信息密度】一张图一个核心信息
├── 好：①看 ②闻 ③检查聪明座（三步分开）
└── 差：一张图塞满所有知识点 ❌

【留白】文字占画面≤30%
├── 给画面呼吸感
└── 给字幕清晰度空间
```

## 五、图表设计规范

> **能用图表就不用文字！图表要直观、易懂、有冲击力！**

### 5.1 图表类型

| 类型 | 适用场景 | 示例 |
|------|----------|------|
| **对比表** | 两种方案/产品对比 | 循环桶 vs 一次性桶 |
| **流程图** | 步骤/因果关系 | 污染路径/处理流程 |
| **条形图** | 数据/程度对比 | 健康风险/口感评分 |
| **清单图** | 步骤/要点 | 3步辨别/4招解决 |
| **卡片图** | 优势/特点展示 | 4大优势/5个理由 |

### 5.2 图表设计原则

```
【对比表示例】
┌─────────────┬─────────────┐
│   循环桶    │  一次性桶   │
├─────────────┼─────────────┤
│ ████░░░░░░ │ █████████░  │
│ 材质老化    │ 全新材质    │
├─────────────┼─────────────┤
│ ███░░░░░░░ │ ████████░░  │
│ 有污染风险  │ 0污染      │
└─────────────┴─────────────┘

【条形图示例】
茶汤清亮度：
龙眼山 ████████████
农夫山 ███████░░░░
怡宝   ████░░░░░░░

【清单图示例】
3步辨别问题桶
① 看桶身 → 发黄？划痕？
② 闻桶口 → 异味？
③ 查聪明座 → 污垢？
```
