# 长文内容模板 (Long Text)

## 概述
融合社交媒体4维度设计方法论，生成知乎/公众号风格长文，支持8种结构模板。

## 8种长文结构模板

| 模板ID | 模板名称 | 字数范围 | 适用场景 |
|--------|----------|----------|----------|
| knowledge_article | 知识科普型 | 2000-3000字 | 系统化讲解专业知识 |
| cause_analysis | 原因分析型 | 2500-3500字 | 深度剖析问题原因 |
| industry_reveal | 行业揭秘型 | 3000-4000字 | 揭秘行业潜规则 |
| deep_review | 深度测评型 | 3000-4000字 | 多产品对比测评 |
| experience_share | 经验分享型 | 2000-3000字 | 个人经历复盘 |
| data_insight | 数据解读型 | 2500-3500字 | 数据支撑的洞察 |
| buying_guide | 选购指南型 | 3000-4000字 | 决策全流程指导 |
| solution_guide | 解决方案型 | 2500-3500字 | 问题系统化解决 |

## 4维度设计方法论

### 维度一：情绪动线逻辑
章节级情绪曲线：
- 开篇(P1)：期待/代入
- 痛点(P2)：共情/焦虑
- 获得感(P3-P5)：专业/坚定
- 升华(P6)：温暖/安心
- 转化(P7)：信任/行动

### 维度二：视觉资产一致性
- light_style: High-key lighting（柔和自然光）
- color_gradient: 冷色(痛点) → 暖色(方案) → 品牌色(闭环)

### 维度三：信息层级架构
每章节遵循"1-3-5"原则：
- 1个大字金句（≤15字）
- 3个信息模块
- 5秒阅读闭环

UI组件：
- 勾选框（Checklist）
- 图标（Icon）
- 数字徽章（Badge）
- 时间轴（Timeline）
- 对比图（Comparison）
- 气泡框（Bubble）

### 维度四：高转化排版模板
与图文一致：billboard / problem_solver / matrix / trust_builder

## 文章结构示例
```
开篇钩子（3行内直接给核心价值）
├── 章节1: 引言/背景
├── 章节2: 核心概念/问题分析
├── 章节3: 解决方案/深度解读
├── 章节4: 案例验证/数据支撑
├── 章节5: 总结建议
└── CTA（唯一明确行动号召）
```

## 核心质量要求
1. 开篇直接性：前3行必须直接给出核心结论
2. 信任证据：至少3条具体数据/案例/引用
3. 行动号召：文末唯一明确的下一步行动
4. 禁止抽象词：空洞词/抽象词/模糊词禁用
