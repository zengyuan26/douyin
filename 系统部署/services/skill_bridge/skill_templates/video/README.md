# 短视频脚本模板 (Video Script)

## 概述
基于选题和用户画像，生成短视频脚本，支持10种脚本类型，适用于抖音/快手/视频号等平台。

## 10种脚本类型

| 脚本ID | 脚本名称 | 时长 | 适用场景 |
|--------|----------|------|----------|
| knowledge_teach | 知识讲解型 | 30-60秒 | 专业知识点讲解 |
| story_share | 故事分享型 | 60-120秒 | 个人/他人经历分享 |
| comparison | 对比测评型 | 45-90秒 | 产品/方案对比 |
| tutorial | 教程演示型 | 60-180秒 | 操作步骤演示 |
| qa_answer | 问答解答型 | 30-60秒 | 常见问题解答 |
| list_rank | 榜单排行型 | 45-90秒 | 排行榜/清单推荐 |
| behind_scene | 幕后揭秘型 | 30-60秒 | 行业/产品内幕 |
| emotional_resonance | 情感共鸣型 | 30-60秒 | 情感话题讨论 |
| product_showcase | 产品展示型 | 15-30秒 | 快速产品展示 |
| challenge_trend | 挑战热点型 | 15-60秒 | 热门话题/挑战 |

## 脚本结构

### 标准短视频结构（30-60秒）
```
P1 Hook (0-3秒)：开场吸引
  - 抛出痛点/疑问/反常识
  - 制造悬念

P2 Problem (3-8秒)：问题呈现
  - 展开痛点场景
  - 引发共鸣

P3 Solution (8-40秒)：解决方案
  - 分点讲解（1-2-3）
  - 核心干货输出

P4 CTA (40-50秒)：行动号召
  - 引导关注/评论/收藏
  - 明确下一步动作
```

### 长视频结构（60-180秒）
```
P1 Hook (0-5秒)：开场吸引
P2 Problem (5-15秒)：问题展开
P3 Solution 1 (15-40秒)：方案一
P4 Solution 2 (40-60秒)：方案二
P5 Summary (60-90秒)：总结强化
P6 CTA (90秒+)：行动号召
```

```json
{
  "script": {
    "title": "视频标题",
    "duration": "预估时长",
    "hook": "开场钩子",
    "scenes": [
      {
        "scene_index": 1,
        "time_range": "0-5秒",
        "content_type": "口播 | 画面 | 字幕",
        "script": "台词/画面描述",
        "visual_guide": "画面指导",
        "emotion": "情绪关键词"
      }
    ],
    "cta": {
      "text": "行动号召文案",
      "type": "follow | comment | share | save"
    }
  },
  "shooting_guide": {
    "angle": "拍摄角度",
    "light": "光线要求",
    "scene": "场景要求"
  }
}
```

## 情绪曲线
与图文一致：P1期待 → P2共情 → P3-P5获得 → P6升华 → P7转化

## 拍摄参数
- 比例：9:16（竖版）或 16:9（横版）
- 分辨率：1080x1920 或 1920x1080
- 帧率：30fps 或 60fps
- 光线：柔和自然光为主

## 平台适配
| 平台 | 风格 | 时长 | 重点 |
|------|------|------|------|
| 抖音 | 节奏快、有爆点 | 15-60秒 | 前3秒吸引力 |
| 快手 | 真实接地气 | 30-90秒 | 情感共鸣 |
| 视频号 | 专业有深度 | 60-180秒 | 内容价值 |
| B站 | 知识性强 | 5-15分钟 | 深度讲解 |
