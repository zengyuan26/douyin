## 描述

这是一个**双核驱动的AI智能运营专家skill**，整合了账号运营规划能力与SEO搜索优化能力，为客户提供从账号定位、内容规划到关键词布局、选题生成的全链路运营解决方案。

支持两种客户类型：**卖货类**和**服务类**，帮助客户实现从0到1的账号搭建，或从1到10的账号优化。

## 💡 核心思维：问题在先，人群在后

### ❌ 错误思路（先有人，再找需求）

```
先定义人群：新手妈妈、二胎妈妈
↓ 
找需求：担心假货、不会选
↓ 
卖产品：进口奶粉
```

### ✅ 正确思路（先有问题，再有人群）

```
先发现问题：孩子乳糖不耐受、便秘、过敏
↓ 
人群自动出现：有个这样的宝宝的妈妈
↓ 
提供解决方案：特殊奶粉 + 专业指导
```

### 核心逻辑

```
问题 → 背后站的是人群 → 需要解决方案

孩子有问题 → 妈妈才需要买奶粉
水管漏水 → 业主才需要修水管
房子要装修 → 业主才需要装修公司
```

## 💡 进阶思维：产品服务化

> 💡 **核心理念**：卖产品不如卖服务，产品只是服务的载体

### ❌ 错误思路（单纯卖产品）

```
定位：我是卖XX产品的
思维：产品思维、交易思维
结果：陷入价格战、利润越来越薄
```

### ✅ 正确思路（卖服务化）

```
定位：我是提供XX解决方案/服务的
思维：服务思维、顾问思维、医生思维
结果：建立专业壁垒、难以被替代、定价权
```

### 产品服务化的三种形式

| 形式 | 说明 | 示例 |
|------|------|------|
| **顾问型** | 成为客户的咨询顾问 | 卖锁 → 开锁专家/安防顾问 |
| **医生型** | 像医生一样诊断+方案 | 卖电池 → 电池医生/电瓶诊疗 |
| **服务型** | 产品是服务的载体 | 卖水 → 企业饮水解决方案 |

| 错误 | 正确 |

**问题 → 关键词 → 内容**

```
问题：宝宝拉肚子
    ↓
关键词：宝宝喝奶粉拉肚子怎么办
    ↓
内容：乳糖不耐受科普 + 解决方案

问题：奶粉过敏
    ↓
关键词：牛奶蛋白过敏喝什么奶粉
    ↓
内容：水解奶粉科普 + 推荐
```

## 🔄 工作流程

### Step 1: 读取客户基础信息

从 `client-intake` 或总控获取已收集的客户信息：
- 行业、业务范围、客户名称
- 品牌定位、出镜意愿
- 核心优势、产品详情
- 目标客户画像

### Step 2: 读取行业分析报告（如有）

从 `insights-analyst` 或总控获取行业分析报告：
- 行业趋势、消费者痛点
- 竞争对手动态
- **⭐ 蓝海机会点分析（重点！）**

### Step 3: 关键词矩阵构建

基于客户档案和行业洞察，构建关键词矩阵：

| 关键词类型 | 定义 | 示例（奶粉行业） |
|------------|------|------------------|
| **核心关键词** | 用户搜索最多、流量最大的词 | 奶粉、婴儿奶粉 |
| **产品关键词** | 具体产品/品类相关的词 | 有机奶粉、羊奶粉、A2奶粉 |
| **长尾关键词** | 精准但搜索量较低的词 | 奶粉怎么选、奶粉推荐性价比 |
| **场景关键词** | 与使用场景相关的词 | 送礼奶粉、换奶粉、囤货 |
| **地域关键词** | 本地服务特有（含城市/区域） | 成都奶粉、北京母婴店 |
| **品牌关键词** | 个人/公司品牌相关 | XX营养师、XX公司 |

### Step 4: 选题库构建

根据客户类型和关键词，规划具体选题：

| 客户类型 | 地域范围 | 选题方向 | 核心关键词类型 |
|----------|----------|----------|----------------|
| 卖货类 | 任意 | 产品测评、选购指南、避坑 | 产品词+长尾词 |
| 卖货类 | 本地 | 本地门店、到店体验 | 地域词+产品词 |
| 服务类 | 本地 | 本地服务案例、到店引导 | 地域词+场景词 |
| 服务类 | 跨区域 | 专业科普、行业洞察 | 行业词+长尾词 |

### Step 5: 账号设计与运营规划

基于选择的蓝海机会，做账号设计：
- **昵称**：围绕问题，不围绕产品
- **简介**：强调能解决什么问题
- **内容标签**：#问题关键词
- **内容方向**：围绕问题出内容

### Step 6: 输出完整运营方案

包含以下模块：
1. 市场机会分析
2. 关键词矩阵（≥100个）
3. 选题库（≥100个）
4. 账号设计方案
5. 内容规划
6. 运营阶段规划

## 🏃 起号阶段运营策略

### 起号阶段核心原则

```
新号起号阶段（0-30天）核心目标：

1. 【找蓝海】找到竞争小、流量精准的细分市场
2. 【做权重】提升账号权重，获得平台流量倾斜
3. 【贴标签】让平台精准识别账号定位
4. 【建信任】通过内容建立用户初步信任
5. 【测数据】测试内容方向，找到爆款公式

起号阶段黄金法则：
✅ 蓝海词优先 → 精准流量，竞争小
✅ 垂直内容 → 平台识别，标签精准
✅ 差异化角度 → 避免同质化，突出重围
✅ 高互动内容 → 提升权重，获得推荐
```

### 起号阶段3步走策略

**Step 1：蓝海市场挖掘（0-7天）**
- 目标：找到精准蓝海市场，测试内容方向
- 方法：人群细分、痛点挖掘、场景细分
- 输出：找到3-5个蓝海切入点

**Step 2：内容矩阵搭建（8-21天）**
- 目标：建立内容矩阵，测试爆款公式
- 内容配比：40%蓝海词 + 30%痛点解决 + 20%场景需求 + 10%人设
- 输出：找到3-5个高互动内容方向

**Step 3：爆款复制放大（22-30天）**
- 目标：复制爆款，稳定流量，提升权重
- 方法：爆款特征分析、复制策略、权重提升
- 输出：找到爆款公式，稳定内容输出
