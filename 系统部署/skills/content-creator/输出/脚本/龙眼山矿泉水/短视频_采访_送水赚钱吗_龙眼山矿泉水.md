## 三、即梦视频提示词（直接复制使用）

### 【镜头1：开场钩子】0-3秒

```
男记者手持话筒站在送水仓库前，中近景。切到女老板小周，表情一脸认真地点头，仿佛在说一件很严肃的事。字幕弹出：送水赚钱吗？背景为龙眼山桶装水堆叠。暖色调，自然光。运镜：采访对切。

口播：记者：送水赚钱吗？小周：（一本正经）送水还是很赚钱的。
```

### 【镜头2：炫富堆料】3-6秒

```
小周对着镜头掰手指数，表情认真得像在做年度汇报，越说越笃定。字幕快速闪现：500万、路虎、保时捷、奔驰、两栋别墅。男记者镜头插入，表情逐渐呆滞。节奏渐快。

口播：去年挣了500多万，三辆车，路虎保时捷奔驰，綦江还有两栋别墅。
```

### 【镜头3：转折铺垫】6-9秒

```
小周表情突然收敛，眉头微皱，叹了口气，像在谈一个很现实的经济问题。男记者满脸狐疑。BGM突然收住，画面静止半拍。

口播：不过今年行情不是特别好。
```

### 【镜头4：反转爆点】9-12秒

```
小周凑近镜头，压低声音，表情极其诚恳，像在透露商业机密。说完停顿一帧，自己嘴角微微上扬，憋着笑。字幕大字弹出：主要看睡眠质量。
```

口播：主要还是要看睡眠质量，有时候梦得到，有时候梦不到。

### 【镜头5：两人笑场】12-15秒

```
小周彻底破功，笑出声来。男记者也放下话筒，两人一起哈哈大笑。画面放松自然，像拍摄花絮。字幕弹出：哈哈哈哈。轻松BGM起，画面渐出。
```

口播：（两人哈哈大笑，无台词）
```
