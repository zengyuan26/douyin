## 八、方法论更新说明

### 本次优化核心变化

| 原来 | 现在 |
|------|------|
| 直接讲价值观 | 通过故事带出价值观 |
| 自问自答"为什么" | 用故事回答"为什么" |
| 喊口号 | 用行动展示态度 |
| 讲"我诚信" | 讲"我是怎么做的" |

### 故事线设计

```
P2: 有一天客户打电话来说水有问题（引入故事）
P3: 我追回去查，发现问题不在水（调查真相）
P4: 从那以后我做了一个决定（展示选择）
P5: 其实还有一个客户也遇到过类似的事（验证选择）
P6: 远嫁创业这么多年，我悟出一个道理（升华感悟）
P7: 品牌收尾（引导行动）
```

### 核心公式

```
人设 = 故事 + 细节 + 你的选择
     = 不是我说的 → 是你感受到的
```
