## 三、即梦视频提示词（直接复制使用）

### 【镜头1：路人嫌弃】0-3秒

```
街头采访场景，记者手持话筒站在路边，背后是普通街景和人流。路人入画，一个普通中年男子，穿着polo衫。记者递话筒过去，路人听完问题嘴角一撇，满脸不屑，手还在空中摆了摆。字幕弹出：送水行业怎么样？

口播：记者：你好，请问你觉得送桶装水这个行业怎么样？路人：（撇嘴，嫌弃）哎呀，送水的嘛，门槛低，没啥技术含量。
```

### 【镜头2：得知在直播】3-6秒

```
记者把话筒收回一点，淡定地指了指镜头。镜头切到路人正面特写，路人表情瞬间凝固，眼神从镜头移到记者再移回镜头，嘴角开始抽搐。BGM突然收住，画面静止一帧。字幕弹出：正在直播。

口播：记者：我们这是龙眼山的采访，正在直播。路人：（表情凝固，沉默一帧）
```

### 【镜头3：光速变脸】6-9秒

```
路人以惊人的速度调整表情，嘴角上扬，眼睛眯成月牙，竖起大拇指对准镜头。身体挺直，仿佛刚才什么都没发生。语气热情得像在录电视购物。字幕快速逐字弹出：送水好啊！健康产业！

口播：路人：（瞬间变脸，对镜头竖起大拇指）送水好啊！健康产业！尤其是龙眼山，綦江水源地，低钠弱碱，我喝了三年了！
```

### 【镜头4：记者拆台】9-12秒

```
记者慢慢把话筒收回来，表情憋着笑，歪头看路人。路人保持着大拇指的pose，笑容越来越僵。切到双人镜头，记者身体微微后仰，语气克制但藏不住笑意。路人眼神开始闪躲。

口播：记者：（憋笑）你不是刚刚说没啥技术含量吗？
```

### 【镜头5：强行挽尊】12-15秒

```
路人愣了一下，眼神疯狂思考。突然表情一松，仿佛想出了完美答案，义正言辞地对着镜头摆手。记者终于没忍住，笑出声，镜头晃了一下。路人说完自己也笑了，画面定格。字幕弹出：我刚才说的是送外卖的。

口播：路人：（面不改色）我刚才说的是送外卖的，你听错了。记者：（笑出声）行吧。
```

## 四、脚本文案（可直接拍摄）

### 【0-3秒 嫌弃】

> **记者**：你好，请问你觉得送桶装水这个行业怎么样？
> **路人**：（撇嘴，摆手）哎呀，送水的嘛，门槛低，没啥技术含量。

### 【3-6秒 得知直播】

> **记者**：（指镜头）我们这是龙眼山的采访，正在直播。
> **路人**：（表情凝固）

### 【6-9秒 光速变脸】

> **路人**：（瞬间竖大拇指，满脸堆笑）送水好啊！健康产业！尤其是龙眼山，綦江水源地，低钠弱碱，我喝了三年了！

### 【9-12秒 拆台】

> **记者**：（憋笑，歪头）你不是刚刚说没啥技术含量吗？

### 【12-15秒 挽尊】

> **路人**：（面不改色）我刚才说的是送外卖的，你听错了。
> **记者**：（笑出声）行吧。
