> ⚠️ **【强制规格】内容形式：图文 | 图片数量：5张 | 比例：9:16竖版 1080×1920px**

## 一、核心内容规则

> ⚠️ **【必读】图文内容三大核心规则，每次创作必须遵守**

### 1.1 人-货-场闭环

> **内容逻辑：人在场景下用产品/服务 → 遇到问题 → 苦恼不能解决 → 情绪变化 → 解决方案**

|| 要素 | 说明 |
|------|------|------|
| **人** | 目标用户 | 目标人群的具体画像和痛点 |
| **货** | 产品/服务 | 解决用户问题的具体方案 |
| **场** | 使用场景 | 用户在什么场景下遇到问题 |

**闭环要求：**
- 每张图都要体现"人-货-场"中的某个环节
- 内容要有代入感，让用户觉得"说的就是我"
- 解决方案要直接对应前面提出的问题

### 1.2 情绪递进（"已经"原则）

> **情绪节奏：无奈 → 疑惑 → 爆发 → 释然 → 希望 → 行动**

| 情绪阶段 | 图号 | 情绪词 | 状态描述 |
|----------|------|--------|----------|
| 初始 | 图1 | 无奈/戳心 | 问题已经发生，用户已经被困扰 |
| 代入 | 图2 | 疑惑/回忆 | 用户回想自己也遇到过 |
| 爆发 | 图3 | 爆发/共鸣 | 情绪达到顶点，说出用户心声 |
| 揭秘 | 图4 | 释然/顿悟 | 揭示真相，用户恍然大悟 |
| 方案 | 图5 | 希望/期待 | 给出解决方案，用户看到希望 |
| 行动 | 图6 | 行动 | 引导用户采取行动 |

**"已经"原则示例：**
- ❌ "如果你也有这个问题"（假设性，语气弱）
- ✅ "桶装水那股怪味 我已经忍了3年"（已完成式，情绪强）
- ❌ "可能是因为水不好"（猜测性）
- ✅ "不是水的问题 是桶的问题"（确定性，已揭秘）

### 1.3 大字原则

> **核心要点必须用大字突出，一眼就能看到关键信息**

| 层级 | 字数 | 位置 | 作用 |
|------|------|------|------|
| 主标题 | 7字以内 | 图上1/3处 | 第一眼看到，决定点不点 |
| 痛点词 | 3-5字 | 图中突出位置 | 引发共鸣，一眼戳心 |
| 揭秘语 | 5-8字 | 图中C位 | 颠覆认知，金句炸场 |
| 方案名 | 5字以内 | 图中显眼位置 | 直接给答案 |
| 行动语 | 10字以内 | 底部 | 引导互动/私信 |

**大字规范：**
- ⚠️ **每张图必须有1-2个大字金句**
- ⚠️ **大字用来说"结果"，小字用来说"原因"**
- ⚠️ **禁止整段文字，字数要少，冲击力要强**

**大字示例：**
```
❌ 错误示范（密密麻麻）：
"很多人都有这样的困惑，为什么桶装水会有怪味呢？
其实这不是水的问题，而是桶的问题..."

✅ 正确示范（大字金句）：
"不是水的问题
 是桶的问题"

大字：揭秘真相（8字）
小字：循环桶用久了 材质老化析出物质
```

### 视觉设计核心规则

#### 1. 构图与比例
- [ ] 图片必须为9:16竖版比例（1080x1920px）
- [ ] 元素上下排列，禁止左右横排
- [ ] 文字区域占比不超过画面40%

#### 2. 场景融合（关键！）
- [ ] 每张图必须有适配的场景图（虚化背景）
- [ ] **严禁使用：纯色背景、渐变背景、纹理背景**
- [ ] 背景与内容相关，辅助理解但不喧宾夺主

#### 3. 色彩搭配（关键！）
- [ ] 避免大面积色块（禁止纯红、纯黄、纯绿等鲜艳色块）
- [ ] 使用淡雅色彩：浅绿、浅蓝、白色、米色为主
- [ ] 用标记（X/✓）代替颜色区分

#### 4. 排版整洁
- [ ] 元素对齐，间距合理
- [ ] **文字简洁，每张图不超过30字/行**
- [ ] **严禁堆砌文字，保持简洁易读**
- [ ] 无杂乱感，画面干净

#### 5. 标题规范
- [ ] **主标题：7字以内**
- [ ] **副标题：10字以内**
- [ ] 标题含有客户问题或问题答案
- [ ] 封面直接给到行动引导或好处

#### 6. 品牌信息出现位置规范
- [ ] 封面：仅底部行动引导
- [ ] 内容页（2-4）：无品牌信息（纯内容）
- [ ] 结尾页：副标题（仅1处，不要加底部品牌信息）
