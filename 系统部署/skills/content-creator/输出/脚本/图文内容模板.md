## 三、内容规划（5张图片）

> ⚠️ **【封面是重中之重】首页决定图文的生死！生成后必须先由以下专家重点评审通过后再制作其他页面：**
> - **消费心理学专家**：评审标题吸引力、行动引导效果、消费心理驱动
> - **视觉设计专家**：评审9:16比例、场景图适配、排版美观、字体颜色

> ⚠️ **【重要】AI生图使用"文字内容"部分，画面描述和设计规格是给人类设计师看的参考**

> ⚠️ **强制要求：所有图片必须为 9:16 竖版比例 1080×1920px**

> ⚠️ **【核心原则】每张图片都必须有丰富的图文内容，形成完整的内容逻辑链：封面吸引→问题引入→原因分析→解决方案→行动引导**

> ⚠️ **【视觉设计核心规则】每次生成图文内容必须遵守**

### 视觉设计核心规则

#### 1. 构图与比例
- [ ] 图片必须为9:16竖版比例（1080x1920px）
- [ ] 元素上下排列，禁止左右横排
- [ ] 文字区域占比不超过画面40%

#### 2. 场景融合（关键！）
- [ ] 每张图必须有适配的场景图（虚化背景）
- [ ] **严禁使用：纯色背景、渐变背景、纹理背景**
- [ ] 背景与内容相关，辅助理解但不喧宾夺主

#### 3. 色彩搭配（关键！）
- [ ] 避免大面积色块（禁止纯红、纯黄、纯绿等鲜艳色块）
- [ ] 使用淡雅色彩：浅绿、浅蓝、白色、米色为主
- [ ] 用标记（X/✓）代替颜色区分

#### 4. 排版整洁
- [ ] 元素对齐，间距合理
- [ ] **文字简洁，每张图不超过30字/行**
- [ ] **严禁堆砌文字，保持简洁易读**
- [ ] 无杂乱感，画面干净

#### 5. 标题规范
- [ ] **主标题：7字以内**
- [ ] **副标题：10字以内**
- [ ] 标题含有客户问题或问题答案
- [ ] 封面直接给到行动引导或好处

#### 6. 品牌信息出现位置规范
- [ ] 封面：仅底部行动引导
- [ ] 内容页（2-4）：无品牌信息（纯内容）
- [ ] 结尾页：副标题（仅1处，不要加底部品牌信息）

```
✓ 图片数量：5张（9:16竖版）
✓ 主标题：7字以内
✓ 副标题：10字以内
✓ 图表为主，文字精简
```
