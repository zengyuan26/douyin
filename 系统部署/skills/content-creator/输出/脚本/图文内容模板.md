## 三、内容规划（5张图片）

> ⚠️ **【封面是重中之重】首页决定图文的生死！生成后必须先由以下专家重点评审通过后再制作其他页面：**
> - **消费心理学专家**：评审标题吸引力、行动引导效果、消费心理驱动
> - **视觉设计专家**：评审9:16比例、场景图适配、排版美观、字体颜色

> ⚠️ **【重要】AI生图使用"文字内容"部分，画面描述和设计规格是给人类设计师看的参考**

> ⚠️ **强制要求：所有图片必须为 9:16 竖版比例 1080×1920px**

> ⚠️ **【核心原则】每张图片都必须有丰富的图文内容，形成完整的内容逻辑链：封面吸引→问题引入→原因分析→解决方案→行动引导**

#### 图片1：封面 ⚠️【最关键！重点评审】

> ⚠️ **【简洁原则】封面文字不要过多，点出问题+行动引导即可，文字太多会流失观众**

**【内容功能】**吸引点击，简洁有力

**【画面描述】**
- 背景：虚化场景图（根据主题定制）
- 布局：竖版9:16，简洁大气
- ⚠️ 禁止：手机外框、手机Mockup、任何电子设备外壳

**【文字内容】**
```
主标题：【问题+解决方案，如：豆芽发苦？1招解决】
行动引导：【引导收藏/点赞，如：收藏起来以后用】
```

**【设计规格】**
- 尺寸：1080×1920px（9:16）
- 背景：虚化场景图
- 排版：简洁大气，突出重点

---

> 💡 **封面优化要点（消费心理学+视觉设计重点评审）**：
> - [ ] 标题是否有悬念/好奇/价值感？
> - [ ] 行动引导是否引发损失厌恶？（如"错过就后悔"）
> - [ ] 排版是否大气、层次分明？
> - [ ] 字体颜色是否清晰、与背景对比度够？

#### 图片2：问题引入/痛点揭示 ⚠️强制9:16

**【内容功能】**
- 抛出用户最关心的问题/痛点
- 引发用户共鸣和思考
- 为后续内容铺垫

**【画面描述】（人类设计师参考）**
- 背景：白色简洁背景或浅色场景图
- 布局：上方大标题+中间痛点描述+底部悬念引导
- 元素：问号图标、警示图标、人物/场景图
- 风格：引发共鸣、引发思考
- ⚠️ 禁止：手机外框、手机Mockup、任何电子设备外壳

**【文字内容】（给AI生图用，详细描述画面场景，让图片图文并茂）**
```
画面：【根据选题定制场景，如：整洁的婴儿房，妈妈抱着宝宝表情困惑】
背景：浅色温馨背景，干净整洁的家庭环境
文字：
- 上方标题："奶粉结块了，还能给宝宝喝吗？"
- 中间核心问题：
  ❌ 继续喝=害了宝宝
  ❌ 直接扔=太浪费
  ✅ 正确答案往下看→
- 底部悬念：90%的家长都做错了！
风格：温馨但带有疑问感，让人想知道答案
```

**【设计规格】**
- 尺寸：1080×1920px（9:16）
- 背景：白色或浅色温馨风
- 排版：标题醒目，要点突出

> ### 视觉设计核心规则

> #### 1. 构图与比例
> - [ ] 图片必须为9:16竖版比例（1080x1920px）
> - [ ] 元素上下排列，禁止左右横排
> - [ ] 文字区域占比不超过画面40%

> #### 2. 场景融合（关键！）
> - [ ] 每张图必须有适配的场景图（虚化背景）
> - [ ] 禁止使用：纯色背景、渐变背景、纹理背景
> - [ ] 背景与内容相关，辅助理解但不喧宾夺主

> #### 3. 色彩搭配（关键！）
> - [ ] 避免大面积色块（禁止纯红、纯黄、纯绿等鲜艳色块）
> - [ ] 使用淡雅色彩：浅绿、浅蓝、白色、米色为主
> - [ ] 用标记（X/✓）代替颜色区分

> #### 4. 排版整洁
> - [ ] 元素对齐，间距合理
> - [ ] 文字简洁，每张图不超过30字/行
> - [ ] 无杂乱感，画面干净

> #### 5. 人物出镜原则（关键！非常重要！）
- [ ] **核心理念**：让用户看到"用这个产品能产生什么效果"，而不是"产品本身是什么"
- [ ] 第2张图（问题引入/痛点揭示）：建议有人物和产品一起出镜，展示使用场景，让用户感受到"别人家的效果"
- [ ] 第3张图（深度分析）：建议用实物对比图，有产品图更真实可信
- [ ] 第4张图（解决方案）：建议有人物举产品出镜，展示实际使用效果
- [ ] 第5张图（结尾引导）：建议人物举产品出镜，形成"看了就想行动"的冲动
- [ ] **参考案例**：春游野炊场景 - 一家三口举定制水拍照，这才是用户想要的效果

#### 6. 品牌信息出现位置规范
> - [ ] 封面：仅底部行动引导
> - [ ] 内容页（2-4）：无品牌信息（纯内容）
> - [ ] 结尾页：副标题（仅1处，不要加底部品牌信息）

## 五、制作规范

### 设计规格

```
✓ 图片数量：5张（9:16竖版）
✓ 文字大小：16pt+
✓ 核心观点要醒目
✓ 案例数据要具体
✓ 互动引导要明确
✓ 每张必须有场景图（禁止纯色/渐变背景）
✓ 禁止出现手机外框/手机Mockup/电子设备外壳
✓ 内容页（2-5）建议人物出镜，展示使用效果
```

### 色彩方案

| 元素 | 颜色参考 | 使用场景 |
|------|----------|----------|
| 主色调 | 蓝色系（#1A73E8）/蓝绿色（#00897B） | 主标题、强调元素 |
| 辅助色 | 白色（#FFFFFF） | 文字背景、留白 |
| 警示色 | 橙色（#FF9800） | 行动引导 |
| 信任色 | 绿色（#4CAF50） | 正面示例、成功案例 |
| 文字深色 | 深灰（#333333） | 浅色背景时 |
| 文字浅色 | 白色（#FFFFFF） | 深色/场景图背景时 |

> 💡 **字体颜色使用规则**：
> - 浅色背景（白色/浅色渐变）：标题用深灰#333333，正文用深灰#333333
> - 深色背景/场景图：标题用白色#FFFFFF，正文用白色#FFFFFF或浅灰#EEEEEE
> - 行动引导：根据背景深浅选择橙色#FF9800或深灰#333333

### 字体规范

> 💡 **字体规范**：正文使用黑体/宋体，标题醒目，正文清晰即可，无需指定具体字号（上下15%）

## 📌 模板使用说明

1. 所有图片强制9:16竖版比例
2. 先确认内容形式（短视频/图文），再按模板生成
3. 标签从关键词库提取，不要自己编造
4. 品牌信息从客户档案读取
5. **【重要】内容页（2-5）建议人物出镜**：让用户看到"用这个产品能产生什么效果"，而不是"产品本身是什么"
6. **【核心】每张图片都必须有实质内容**：
   - 图片1封面：吸引点击的标题+悬念引导
   - 图片2问题引入：抛出用户痛点/问题，引发共鸣，建议人物+产品出镜
   - 图片3深度分析：原因/原理/专业知识，建立信任，建议实物对比
   - 图片4解决方案：具体步骤/要点/注意事项，实用价值，建议人物举产品出镜
   - 图片5行动引导：关注引导+互动问题+品牌信息，建议人物举产品出镜
