# 图文内容模板

> ⚠️ **【强制规格】内容形式：图文 | 图片数量：5张 | 比例：9:16竖版 1080×1920px**

---

## 一、核心内容规则

### 1.1 人-货-场闭环

| 要素 | 说明 |
|------|------|
| **人** | 目标用户 | 制造业老板、生产厂长、车间主任 |
| **货** | 产品/服务 | 车间自动化解决方案 |
| **场** | 使用场景 | 制造业车间、工厂生产线 |

### 1.2 情绪递进（"已经"原则）

| 情绪阶段 | 图号 | 情绪词 | 状态描述 |
|----------|------|--------|----------|
| 初始 | 图1 | 无奈/戳心 | 问题已经发生，用户已经被困扰 |
| 代入 | 图2 | 疑惑/回忆 | 用户回想自己也遇到过 |
| 爆发 | 图3 | 爆发/共鸣 | 情绪达到顶点，说出用户心声 |
| 揭秘 | 图4 | 释然/顿悟 | 揭示真相，用户恍然大悟 |
| 方案 | 图5 | 希望/期待 | 给出解决方案，用户看到希望 |

**【内容功能】**吸引点击，简洁有力，直接给价值

- 背景：虚化车间里停工的设备场景

## 六、制作规范

```
✓ 图片数量：5张（9:16竖版）
✓ 主标题：7字以内
✓ 副标题：10字以内
✓ 每张必须有场景图（禁止纯色/渐变背景）
✓ 背景与内容相关：必须是制造业车间场景
✓ 禁止出现手机外框
✓ 禁止使用emoji和特殊符号
✓ 严禁堆砌文字，保持简洁易读
```

## 八、检查清单

#### 核心规则检查
- [ ] **人-货-场闭环**：每张图是否体现了人-货-场中的某个环节
- [ ] **情绪递进**：情绪是否按"无奈→疑惑→爆发→释然→希望→行动"递进
- [ ] **"已经"原则**：痛点描述是否使用已完成式
- [ ] **大字原则**：每张图是否有1-2个大字金句

#### 内容规范检查
- [ ] 主标题7字以内，含问题或答案
- [ ] 副标题10字以内，直接给价值
- [ ] 严禁堆砌文字，保持简洁
- [ ] 禁止纯色/渐变背景，必须有场景图
- [ ] 场景必须是制造业车间/工厂环境

#### 禁止出现
- [ ] 无品牌暴露
- [ ] 无具体产品名称
- [ ] 无行业黑话
- [ ] 使用自然语言
