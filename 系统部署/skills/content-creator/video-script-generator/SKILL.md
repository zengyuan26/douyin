# 短视频脚本生成专家

> **版本**：1.3.0
> **定位**：基于短视频脚本生成系统方案，提取核心方法论，形成可直接操作的创作规范

---

## ⚠️ 第一步：人设确认（必须先执行）

> 人设决定了内容的语气、风格、讲述角度，是内容创作的起点。
> 生成任何短视频脚本前，必须先确认人设！

### 5种人设定位

| 人设 | 核心态度 | 行为特征 | 适用人群 | 内容风格 |
|------|---------|---------|---------|---------|
| **陪伴者** | 我懂你 | 共情、理解、陪伴 | 焦虑型、迷茫型 | 温暖、走心、像朋友聊天 |
| **教导者** | 我教你 | 专业、权威、教导 | 理性型、谨慎型 | 专业、有深度，知识感 |
| **崇拜者** | 秀自己 | 热情、种草、分享 | 追求型、跟风型 | 积极，感染力、种草感 |
| **陪衬者** | 不如你 | 真实、共鸣、分享 | 经验型、分享型 | 接地气、真实感、代入感 |
| **搞笑者** | 逗笑你 | 幽默、娱乐、反差 | 年轻型、猎奇型 | 轻松、有趣，反转多 |

### ⚠️ 第二步：三要素线设计（必须选择）

> 内容至少要满足「有趣」「有用」「有共鸣」三要素中的至少2条。
> 三条线贯穿全程，控制节奏，引发互动。

### 三要素线说明

| 要素线 | 核心作用 | 带来的数据 |
|--------|---------|-----------|
| **有趣** | 留住人，让观众看下去 | 完播率、点赞率 |
| **有用** | 留住人，让观众觉得值 | 收藏率、转发率 |
| **有共鸣** | 带走情感，让观众认可你 | 评论率、关注率 |

### 四种组合选择

| 组合 | 适用场景 | 节奏设计 |
|------|---------|---------|
| **有用 + 有趣** | 知识科普、教程 | 有趣开路，有用收尾 |
| **有用 + 有共鸣** | 解决方案、案例 | 有用铺垫，共鸣爆发 |
| **有趣 + 有共鸣** | 人设故事、观点 | 有趣切入，共鸣升华 |
| **有用 + 有趣 + 有共鸣** | 高质量内容 | 三线交织，全程高潮 |

### ⚠️ 第三步：三要素线节奏设计（内容制作规范）

> 三条线贯穿全程，不是孤立的，而是交织进行。
> 每个时间段都有"主线+辅线"，共同控制节奏。

#### 组合1：「有用 + 有趣」节奏设计

```
节奏设计：有趣开路 → 有用支撑 → 双重满足

0-3秒：有趣线启动（主线）
    │
    ├── 抛出悬念/反差/冲突
    └── "你们猜怎么着..."

3-15秒：有趣线持续（主线）+ 有用线铺垫（辅线）
    │
    ├── 有趣：冲突展开、情绪起伏
    └── 有用：开始暗示"有用信息即将揭晓"

15-30秒：有用线爆发（主线）
    │
    ├── 干货输出、知识点密集
    └── 金句提炼、收藏点设置

30-60秒：双线收尾
    │
    ├── 有趣：反转/彩蛋
    └── 有用：总结+行动引导

【互动设计】
├── 评论区：你遇到类似问题吗？
├── 收藏引导：先收藏起来，以后用得上
└── 关注引导：关注我，持续分享
```

#### 组合2：「有用 + 有共鸣」节奏设计

```
节奏设计：有用铺垫 → 共鸣爆发 → 信任建立

0-3秒：有用线钩子（主线）
    │
    ├── 痛点问题、利益承诺
    └── "这个问题不解决，你会..."

3-15秒：有用线展开（主线）
    │
    ├── 分析原因、拆解问题
    └── 数据支撑、案例铺垫

15-30秒：有用 + 共鸣交织
    │
    ├── 有用：解决方案逐步揭示
    └── 共鸣：情感连接开始建立

30-60秒：共鸣线爆发（主线）
    │
    ├── 说出用户心声
    ├── 替用户表态
    └── 价值观输出

【互动设计】
├── 评论区：你是不是也这样觉得？
├── 追问引导：你们还遇到过什么？
└── 分享引导：转给需要的朋友
```

#### 组合3：「有趣 + 有共鸣」节奏设计

```
节奏设计：有趣切入 → 共鸣深化 → 人设升华

0-3秒：有趣线爆发（主线）
    │
    ├── 反差/反转/意外
    └── 制造悬念，吸引眼球

3-15秒：有趣 + 共鸣交织
    │
    ├── 有趣：冲突故事、反转情节
    └── 共鸣：代入感、熟悉感

15-30秒：共鸣线主导（主线）
    │
    ├── 情感高潮点
    ├── 观点输出
    └── 价值观表达

30-60秒：人设升华
    │
    ├── 总结核心观点
    └── 金句收尾

【互动设计】
├── 评论区：你认同吗？
├── 观点引导：你觉得呢？
└── 关注引导：喜欢这种风格的关注我
```

#### 组合4：「有用 + 有趣 + 有共鸣」节奏设计

```
节奏设计：三线交织 → 全程高潮 → 全方位满足

0-3秒：三线同时启动
    │
    ├── 有趣：强钩子（反差/悬念）
    ├── 有用：利益承诺
    └── 共鸣：情感触点

3-15秒：有趣 + 有用 并行
    │
    ├── 有趣：冲突展开
    └── 有用：信息传递

15-30秒：有用 + 共鸣 交织
    │
    ├── 有用：干货密集区
    └── 共鸣：情感高潮

30-45秒：三线汇聚
    │
    ├── 有趣：反转/彩蛋
    ├── 有用：最终价值点
    └── 共鸣：价值观升华

45-60秒：三线收尾
    │
    ├── 金句总结
    ├── 行动引导
    └── 关注号召

【互动设计】
├── 点赞：觉得有用的点个赞
├── 收藏：先收藏起来
├── 评论：你遇到过吗？
└── 关注：关注不迷路
```

### 三要素线设计检查清单

```
生成内容时，检查是否满足：

□ 1. 选择了至少2条线（有用/有趣/有共鸣）
□ 2. 每条线在内容中都有明确的"触发点"
□ 3. 节奏设计符合所选组合的特点
□ 4. 三条线不是孤立存在，而是交织进行
□ 5. 每个时间段都有主导线+辅助线
□ 6. 结尾处所有线都得到满足
□ 7. 互动引导与主导线匹配
```

## 一、核心方法论

### 1.1 游戏化内容设计三角原则

```
┌─────────────────────────────────────────────────────────────────┐
│                    短视频脚本设计三角原则                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ⚡ 前3秒法则                    ⚡ 即时反馈递减               │
│  ┌───┐                          ┌───┐                       │
│  │3秒│ ─── 留住用户 ───→       │奖励│                       │
│  └───┘      必须强钩子             └───┘                       │
│      ↓                                 ↓                        │
│  每3-5秒一个奖励点              前密 → 后疏                    │
│  (前10秒密集)                   (10秒后逐渐拉长)               │
│                                                                  │
│                          ↓                                        │
│  📈 渐进式难度                                                │
│  ┌───┐                                                         │
│  │难度│                                                         │
│  └───┘                                                         │
│      ↓                                                         │
│  低门槛 → 中复杂度 → 高价值结论                                 │
│  (容易理解) (层层递进)   (认知刷新)                            │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 三要素定位

| 要素 | 核心问题 | 目的 | 时间位置 |
|------|---------|------|---------|
| **有趣** | "为什么要看？" | 留住人，前3秒抓住注意力 | 视频前段 |
| **有用** | "学到了什么？" | 提供价值，让观众觉得没白看 | 视频中段 |
| **共鸣** | "为什么关注他？" | 带走情感，认可这个人 | 视频后段 |

### 1.3 三要素时间分布

```
0-3秒    3-10秒    10-30秒   30-60秒   60秒+
 ┌───┐    ┌───┐     ┌───┐     ┌───┐    ┌───┐
 │有趣│ → │有趣│  → │有用│  → │共鸣│ → │金句│
 └───┘    └───┘     └───┘     └───┘    └───┘
  ↑                        ↑            ↑
 钩子                      干货          认同
 悬念                      知识          观点
 冲突                      经验          价值观
```

### 4.1 六维均衡器

| 维度 | 范围 | 影响效果 | 调节建议 |
|-----|-----|---------|---------|
| **信息密度** | 30%-100% | 关键词出现频率、干货密度 | 知识类→高 / 情感类→低 |
| **问题悬念** | 30%-100% | 问句密度、钩子强度 | 问题类→高 / 方案类→中 |
| **情绪波动** | 30%-100% | 情绪词密度、情绪转折次数 | 情感类→高 / 专业类→低 |
| **互动频率** | 30%-100% | 互动点密度、CTA出现次数 | 粉丝运营→高 / 品牌→中 |
| **奖励分布** | 30%-100% | 奖励点"前密后疏"程度 | 前密→100% / 均匀→30% |
| **难度递进** | 30%-100% | 内容难度提升速度 | 深度内容→高 / 浅层内容→低 |

### 4.2 奖励分布详解

```
奖励分布 = 即时反馈递减原则的具体实现

30%（均匀分布）
●──────●──────●──────●──────●──────●
均匀分配奖励点，不符合注意力递减规律

60%（适度前密）
●●●─●─────────●──────●─────────●───────●
前10秒密集，后续逐渐拉长

100%（强烈前密）
●●●●●●───────●───────────●───────────●
前10秒极高密度，后续快速拉长

推荐配置：热点类→80-100%，知识类→60-80%，故事类→40-60%
```

### 4.3 难度递进详解

```
难度递进 = 渐进式难度原则的具体实现

30%（平缓递进）
理解难度：■■■□□□□□□□
低────→低→中─────→中→高
适合：简单科普、轻松话题

60%（标准递进）
理解难度：■■■□□□□■□□■
低────→低→中─────→高────→高
适合：方案类、观点类内容

100%（快速递进）
理解难度：■■■□□■□□■□■
低────→中→高────→更高
适合：深度观点、认知刷新类内容

推荐配置：知识科普类→70-90%，观点输出类→70-100%
```

### 4.4 预设模式

| 模式 | 信息密度 | 问题悬念 | 情绪波动 | 互动频率 | 奖励分布 | 难度递进 | 适用场景 |
|------|---------|---------|---------|---------|---------|---------|---------|
| **沉稳型** | 80% | 50% | 40% | 40% | 50% | 60% | B端、企业服务、专业咨询 |
| **活力型** | 60% | 80% | 90% | 80% | 90% | 40% | 快消品、年轻品牌、促销引流 |
| **专业型** | 90% | 60% | 50% | 60% | 70% | 80% | 知识付费、教育、培训 |
| **情感型** | 40% | 50% | 95% | 50% | 60% | 40% | 母婴、情感、心理咨询 |
| **犀利型** | 70% | 90% | 85% | 70% | 85% | 60% | 吐槽、对比、避坑类 |
| **温情型** | 40% | 40% | 80% | 60% | 50% | 30% | 陪伴、治愈、暖心内容 |

## 七、评分体系

### 7.1 总分计算

```
总分 = 情绪得分 × 30% + 节奏得分 × 30% + 互动得分 × 40%
```

### 7.2 情绪评分（0-100分）

| 评分维度 | 分值 | 优秀标准 | 合格标准 |
|---------|-----|---------|---------|
| 情绪词密度 | 25分 | 每15秒1个情绪词 | 每30秒1个情绪词 |
| 情绪词多样性 | 25分 | 使用3种以上情绪词 | 使用2种情绪词 |
| 情绪转折设计 | 25分 | 有明显情绪起伏 | 有1-2个情绪变化 |
| 代入感设计 | 25分 | 多次使用"你"、设问 | 偶尔使用"你" |

### 7.3 节奏评分（0-100分）

| 评分维度 | 分值 | 优秀标准 | 合格标准 |
|---------|-----|---------|---------|
| 钩子数量 | 30分 | 达到时长推荐数量 | 达到推荐的80% |
| 钩子时机 | 30分 | 前3秒有钩子 + 中间有钩子 | 前3秒有钩子 |
| 节奏峰值 | 20分 | 每个节奏段落有1个峰值点 | 有2-3个峰值点 |
| 时长匹配度 | 20分 | 时长与内容饱满度匹配 | 时长略长或略短 |

### 7.4 互动评分（0-100分）

| 评分维度 | 分值 | 优秀标准 | 合格标准 |
|---------|-----|---------|---------|
| 互动点数量 | 30分 | 达到推荐的互动点数量 | 达到推荐的60% |
| 互动点时机 | 30分 | 互动点设计在价值点后 | 互动点时机合理 |
| CTA多样性 | 20分 | 使用3种以上CTA类型 | 使用2种CTA类型 |
| 引导自然度 | 20分 | 互动引导融入内容，不生硬 | 引导略生硬 |
