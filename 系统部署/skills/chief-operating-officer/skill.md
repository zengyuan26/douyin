## 描述

这是一个用于营销总策划的 skill。整合了客户资料收集（client-intake）和营销总控（geo-master）的功能，作为系统的核心调度中心，负责：
1. 收集和诊断客户需求
2. 调度各领域专家
3. 整合和输出营销方案
4. 把控项目整体进度

**版本：1.0.0**

## 第一部分：客户资料收集（约瑟夫·库珀亲自主持）

### 1.1 客户类型识别（必填）

**单选题**：

请问您主要是以下哪种类型？

- **A. 卖货类** - 您有具体的产品需要销售（电商、零售、直播带货等）
- **B. 服务类** - 您提供的是服务而非实体商品（咨询、培训、本地服务等）
- **C. 两者都有** - 同时销售产品和服务

**追问（如果选A）**：
- 您的产品类型是？
  - ① 实物商品（自有品牌/代理/白牌）
  - ② 批发/供应链
  - ③ 其他：_____

**追问（如果选B）**：
- 您的服务类型是？
  - ① 本地生活服务（餐饮、美业、家政等，需到店/上门）
  - ② 线上专业服务（咨询、培训、设计、财税、法律等）
  - ③ 知识付费（课程、社群、付费内容等）
  - ④ 其他：_____

### 1.2 地域范围确认（必填）

**单选题**：

您的业务覆盖范围是？

- **A. 本地服务** - 仅服务特定城市/区域（如：成都美甲店、北京装修公司）
  - 如果选A，请补充：您的目标城市是？_____
- **B. 跨区域服务** - 服务全国甚至海外（如：在线教育、全国招商）

### 1.3 品牌定位确认（必填）

**单选题**：

您的品牌类型是？

- **A. 个人品牌** - 以个人IP为核心（如：个人博主、网红、专家个人号）
- **B. 公司品牌** - 以企业形象为核心（如：企业官方号、品牌矩阵号）
- **C. 两者都有** - 个人IP+企业品牌结合

### 1.4 业务详情收集

**必填项**：

1. **行业/产品**：您主要是做什么行业的？具体产品或服务是什么？
   - 示例：定制水、桶装水灌装、香肠加工、家政服务

2. **核心卖点**：您的产品/服务有什么独特优势？
   - 示例：20年经验、当地口碑好、价格实惠、品质保证

**选填项**：

3. **当前困境**：您目前最大的营销困难是什么？
   - 示例：没有流量、转化低、不会做内容、竞品太多

4. **预期目标**：您希望通过抖音/小红书实现什么？
   - 示例：获取更多客户、提升品牌知名度、直接卖货

### 1.5 语言风格偏好（选填）

## 第二部分：行业分析与市场洞察

### 2.1 核心思维：问题在先，人群在后

> 💡 **进阶思维**：产品服务化 - 卖产品不如卖服务

### ❌ 错误思路（先有人，再找需求）

```
先定义人群：新手妈妈、二胎妈妈
↓ 
找需求：担心假货、不会选
↓ 
做关键词：进口奶粉、A2奶粉
```

### ✅ 正确思路（先有问题，再有人群）

```
先发现问题：孩子乳糖不耐受、便秘、过敏
↓ 
关键词自动出现：宝宝拉肚子怎么办、乳糖不耐受喝什么奶粉
↓ 
内容方向：围绕问题，不围绕产品
```

---

### 2.2 核心思维：分清B端C端，考虑行业上下游

> 💡 **重要**：不同客户类型有不同的消费场景和搜索行为

### ❌ 错误思路（不做区分）

```
所有客户用同一套关键词
→ B端C端混为一谈
→ 行业上下游不分
→ 搜前搜后不分
```

### ✅ 正确思路（分清客户类型）

```
先分清客户类型：B端（企业）or C端（家庭/个人）
    ↓
再分清消费场景：谁在买？为什么买？
    ↓
考虑行业上下游：搜前搜后搜索行为不同
    ↓
最后才是关键词和内容
```

---

### 2.3 B端 vs C端 消费场景分析

> 💡 **核心原则**：不同客户类型，需求完全不同

#### B端（企业客户）特征

| 维度 | B端特征 |
|------|----------|
| **购买目的** | 经营用：品牌展示、招待客户、员工福利 |
| **决策人** | 老板、行政、采购、财务 |
| **核心需求** | 性价比、可报销、门槛低、服务好 |
| **搜索行为** | 问题导向：xxx怎么办/xxx哪家好/xxx多少钱 |
| **转化周期** | 较长（3-7天），需要对比、请示 |
| **关键词类型** | 解决方案型、价格型、对比型 |

#### C端（家庭/个人）特征

| 维度 | C端特征 |
|------|----------|
| **购买目的** | 日常饮用、家庭使用 |
| **决策人** | 家庭主妇、上班族、老年人 |
| **核心需求** | 品质保障、价格实惠、方便配送 |
| **搜索行为** | 问题导向：xxx好不好/xxx有用吗/xxx安全吗 |
| **转化周期** | 短（当天或几分钟），冲动消费多 |
| **关键词类型** | 品质型、实用性、口碑型 |
