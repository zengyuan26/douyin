## 三、GEO模式匹配

> GEO（Growth-Engine-Optimization）模式是内容的SEO优化框架

### 3.1 GEO模式定义

| GEO模式 | 核心结构 | 适用关键词 | 适用阶段 |
|---------|----------|-----------|----------|
| **问题诊断型** | 问题→原因→方案 | 为什么、怎么回事 | 痛点放大 |
| **对比答案型** | A vs B→优缺点→推荐 | 哪个好、区别、对比 | 方案对比 |
| **场景故事型** | 场景→经历→转折→结果 | 经历、故事、终于 | 受众锁定、愿景勾画 |
| **顾虑消除型** | 顾虑→方案→保障→行动 | 放心、靠谱、保障 | 顾虑消除 |
| **促销引导型** | 利益点→限时→行动 | 优惠、活动、限时 | 顾虑消除、愿景勾画 |
| **知识科普型** | 概念→原理→应用 | 原理、为什么、科普 | 痛点放大、方案对比 |
| **技巧教程型** | 目标→步骤→注意→总结 | 怎么、技巧、方法 | 痛点放大、愿景勾画 |
| **情感共鸣型** | 共鸣点→故事→升华 | 终于、没想到、原来 | 受众锁定、愿景勾画 |

### 3.2 五段式 → GEO模式映射

| 五段式阶段 | 推荐GEO模式 | 优先级 |
|------------|------------|--------|
| **受众锁定** | 场景故事型、情感共鸣型 | ★★★ |
| **痛点放大** | 问题诊断型、知识科普型、技巧教程型 | ★★★ |
| **方案对比** | 对比答案型、知识科普型 | ★★ |
| **愿景勾画** | 技巧教程型、情感共鸣型、场景故事型 | ★★ |
| **顾虑消除** | 顾虑消除型、促销引导型 | ★★★ |

### 3.3 内容类型 → GEO模式推荐

| 内容类型 | 推荐GEO模式 |
|----------|------------|
| 图文种草 | 对比答案型、问题诊断型 |
| 知识科普 | 知识科普型、问题诊断型 |
