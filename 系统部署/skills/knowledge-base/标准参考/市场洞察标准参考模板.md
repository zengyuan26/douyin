# 市场洞察标准参考模板

> **用途**：AI 在进行市场分析、识别问题、生成人群画像时的统一参考标准
> **核心思维**：问题在先，人群在后 | 信任佐证 + 竞争优势
> **维护**：当分析逻辑有更新时，同步更新本文件

---

## 一、分析流程（7步法）

> **必须按顺序执行，完成后才进入信任佐证与竞争优势**

```
1️⃣ 行业分析 → 了解行业规模、趋势、季节性特征
2️⃣ 找蓝海 → 发现竞争小的细分市场
3️⃣ 人群细分 → 区分不同客户群体（付费人 vs 使用人）
4️⃣ 长尾需求 → 发现细分人群的独特需求
5️⃣ 知识技能解决 → 用专业知识技能满足需求
6️⃣ 搜前搜后 → 分析客户搜索前后的需求
7️⃣ 行业关联 → 分析上下游关联行业
```

## 二、识别问题（核心：使用方 + 付费方）

### 2.1 判断：付费者与使用者是否分离

| 情况 | 示例 | 分析策略 |
|------|------|----------|
| **付费者 = 使用者** | 灌香肠加工、建材销售、本地服务 | 传统分析：直接需求 + 顾虑 + 场景 |
| **付费者 ≠ 使用者** | 奶粉、童装、礼品、养老服务 | 分离分析：使用者问题 + 付费者顾虑 |

### 2.2 使用方问题（使用者面临的问题）

| 维度 | 问题 |
|------|------|
| **核心问题** | 产品/服务要解决什么问题？ |
| **具体表现** | 这个问题怎么表现出来的？ |
| **严重程度** | 高/中/低 |
| **用户意识** | 有意识（知道自己有问题）vs 无意识（不知道问题所在） |

### 2.3 付费方顾虑（购买决策时的心理障碍）

| 顾虑类型 | 具体顾虑 |
|----------|----------|
| **真假顾虑** | 会不会买到假货/不靠谱/没效果？ |
| **价格顾虑** | 值不值这个价？有没有更便宜的？ |
| **选择顾虑** | 哪个更适合我？怎么判断优劣？ |
| **信任顾虑** | 商家说的是真的吗？别人用过吗？ |

## 三、人群画像（十维度 + 分批规则）

### 3.1 十个维度

| 维度 | 限定词示例 | 适用场景 |
|------|------------|----------|
| **发展阶段** | 刚起步 / 天使轮 / A轮 / 成熟期 / 转型期 | 创业者、企业家 |
| **营收规模** | 500万以下 / 3000万-2亿 / 5亿以上 | 企业家、B2B服务 |
| **团队规模** | 10人以下 / 10-50人 / 50-400人 | 企业家、管理者 |
| **工作年限** | 3-5年 / 5-10年 / 10年以上 | 职场人、专家 |
| **行业背景** | 传统制造 / 互联网 / 金融 / 教育 | 所有人群 |
| **地域** | 一线城市 / 新一线 / 三四线 / 海外 | 本地服务、区域业务 |
| **年龄段** | 25-35岁 / 35-45岁 / 45岁以上 | 个人服务 |
| **痛点状态** | 遇到瓶颈 / 想要转型 / 寻求突破 | 所有人群 |
| **目标诉求** | 想要增长 / 想要转型 / 想要变现 | 所有人群 |
| **职位角色** | 创始人 / CEO / VP / 总监 / 经理 | B2B服务 |

### 3.2 分批与组合规则

1. **每次选 3-4 个非「职位角色」维度**，再 **必选「职位角色」**
2. **组合顺序**：按上表顺序排列，**「职位角色」固定排在最后**
3. **同一批画像**：以共性锚点（痛点状态、目标诉求）为核心，保证痛点一致

**示例**：`成熟期 × 3000万-2亿 × 遇到瓶颈 × 想要增长 × 总监`

## 四、信任佐证与竞争优势

### 4.1 信任佐证（解决"能不能帮我做好"）

> **公式**：专业知识技能 + 环境 + 过程 + 案例 = 信任佐证

| 信任方向 | 客户问题 | 佐证内容 |
|----------|----------|----------|
| **专业知识技能** | 能不能帮我做好？ | 经验、技术、配方 |
| **环境** | 卫生吗？干净吗？ | 制作条件、现场 |
| **过程** | 怎么做的？ | 制作流程、步骤 |
| **案例** | 别人做得好不好？ | 客户反馈、口碑 |

### 4.2 竞争优势（解决"为什么要选你"）

> **竞争对手有两种**：1. 直接同行  2. 潜在客户自己动手

| 竞争优势 | vs 同行 | vs 自己动手 |
|----------|---------|------------|
| **省心** | 做得更专业 | 不用自己动手 |
| **省事** | 一站式服务 | 不用准备材料工具 |
| **省钱** | 品质好价格合理 | 避免失败浪费 |
| **放心** | 品质有保障 | 卫生看得见 |
