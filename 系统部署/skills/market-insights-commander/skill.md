## 描述

这是一个**双核驱动的市场洞察与舆情监控 skill**。整合了市场洞察分析能力与社交媒体舆情监控能力，为客户提供从宏观市场分析到微观舆情感知的全维度服务。

支持两种客户类型：**卖货类**和**服务类**，同时提供**社交媒体趋势监控接口**，支持抖音、快手、小红书、大众点评四大平台的全维度监控分析。

## 💡 核心思维：问题在先，人群在后

### ❌ 错误思路（先有人，再找需求）

```
先定义人群：新手妈妈、二胎妈妈
↓ 
找需求：担心假货、不会选
↓ 
卖产品：进口奶粉
```

### ✅ 正确思路（先有问题，再有人群）

```
先发现问题：孩子乳糖不耐受、便秘、过敏
↓ 
人群自动出现：有个这样的宝宝的妈妈
↓ 
提供解决方案：特殊奶粉 + 专业指导
```

### 市场洞察逻辑

```
问题 → 背后站的是人群 → 需要解决方案

孩子有问题 → 妈妈才需要买奶粉
水管漏水 → 业主才需要修水管
```

## 💡 进阶思维：产品服务化

> 💡 **核心理念**：卖产品不如卖服务，产品只是服务的载体

### ❌ 传统思路（单纯卖产品）

```
定位：我是卖XX产品的
思维：产品思维、交易思维
结果：陷入价格战、利润越来越薄
```

### ✅ 服务化思路（卖解决方案）

```
定位：我是提供XX解决方案/服务的
思维：服务思维、顾问思维、医生思维
结果：建立专业壁垒、难以被替代、定价权
```

### 产品服务化的三种形式

| 形式 | 说明 | 示例 |
|------|------|------|
| **顾问型** | 成为客户的咨询顾问 | 卖锁 → 安防顾问/开锁专家 |
| **医生型** | 像医生一样诊断+方案 | 卖电池 → 电池医生/电瓶诊疗 |
| **服务型** | 产品是服务的载体 | 卖水 → 企业饮水解决方案 |

## 🔥 新增：用户失败经历分析（高权重⭐）

> 💡 **核心洞察**：爆款内容往往来自用户真实的失败经历
> 
> **案例**：灌香肠爆款选题"为什么你灌的香肠一切就散，一晒就酸"（70万播放）
> - 这不是从"产品角度"想到的问题
> - 这是从"用户制作失败经历"中挖掘的问题

### 失败经历挖掘维度

| 维度 | 说明 | 示例 |
|------|------|------|
| **制作失败类** | 用户自己制作时遇到的问题 | 灌的香肠一切就散、一晒就酸 |
| **使用失败类** | 用户使用产品时遇到的问题 | 香肠咬不动、太硬、發霉 |
| **选择失败类** | 用户选择时犯的错误 | 买错了品牌、买错了口味 |
| **储存失败类** | 用户储存不当导致的问题 | 香肠变质、有异味 |


### 四大监控维度

| 维度 | 功能 | 核心价值 |
|------|------|----------|
| **用户需求类** | 监控问题咨询、使用场景、需求表达、痛点抱怨 | 直接反哺内容创作 |
| **舆情风险类** | 监控负面投诉、虚假宣传、质量/服务问题 | 及时预警响应 |
| **竞品动态类** | 追踪竞品新视频、互动数据、转化策略、内容创新 | 找差异化 |
| **内容热点类** | 监控热门话题、衰退话题、新兴话题、季节性趋势 | 借势流量 |

### 监控对象

| 监控类型 | 说明 | 示例 |
|----------|------|------|
| **话题标签** | 监控指定#话题标签下的内容 | #奶粉、#婴儿奶粉、#送礼奶粉 |
| **关键词** | 监控关键词相关内容 | "奶粉怎么选"、"奶粉推荐" |
| **账号** | 追踪竞品账号动态 | 竞品官方账号、行业KOL |
| **视频类型** | 筛选特定类型视频 | 探店视频、DIY教程、测评视频 |

## 指令

### 卖货类客户策略
- **优化**：差异化长尾需求挖掘 - 从细分长尾市场找机会，以长尾小词配合大词
- **优化**：蓝海词矩阵构建 - 新号起号专用，问题型+痛点型+场景型+地域型
- 挖掘细分市场机会点
- 识别淡旺季消费特征
- 提供差异化产品定位建议
- 分析竞品关键词布局策略
- **舆情**：调用社交媒体趋势监控，分析用户真实需求
- **舆情**：监控竞品内容策略，找到差异化切入点
- **舆情**：借势热点流量，提升内容曝光
- **舆情**：预警负面舆情，保护品牌形象

### 服务类客户策略
- 分析目标客户痛点和需求
- 识别高转化服务场景
- 挖掘行业关键词机会
- 提供内容营销策略建议
- 评估服务市场容量和增长空间
- **舆情**：监控本地服务话题，借势地域热点
- **舆情**：追踪本地竞品动态，优化服务定位
- **舆情**：舆情预警，保护品牌形象
