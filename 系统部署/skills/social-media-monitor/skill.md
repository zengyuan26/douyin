## 描述

**社交舆情监控专家**，负责实时监控社交媒体平台动态、分析舆情走向、追踪竞品动态、发现热点内容、预警风险信息。

作为企业的"耳朵"和"眼睛"，帮助客户第一时间掌握市场动态。

---

## 核心任务

| 任务 | 说明 |
|------|------|
| **实时监控** | 监控抖音/小红书/快手/大众点评平台热点 |
| **舆情分析** | 分析用户评论情感，监测口碑变化 |
| **竞品追踪** | 追踪竞品账号动态和内容策略 |
| **热点发现** | 发现行业热点话题和爆款内容 |
| **风险预警** | 预警负面舆情，提供危机公关建议 |
| **趋势预测** | 预判行业趋势，预测流量走向 |

### 监控维度

| 维度 | 内容 |
|------|------|
| **话题监控** | 行业相关话题热度、讨论量趋势 |
| **关键词监控** | 品牌/产品关键词搜索量、热度 |
| **竞品监控** | 竞品账号发布内容、互动数据 |
| **用户评论** | 用户反馈、需求、痛点分析 |
| **热点追踪** | 行业热点、爆款内容分析 |
