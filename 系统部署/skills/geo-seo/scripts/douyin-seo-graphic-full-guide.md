## 7. Hashtag矩阵规则

### 7.1 标签组合公式

```
#[核心关键词] + #[产品关键词] + #[长尾/场景关键词] + #[行业词] + #[品牌词]
   ✅必选          ✅必选              建议           建议         可选
```

### 7.2 示例（奶粉行业）

```markdown
#奶粉选购 #婴儿奶粉 #奶粉怎么选 #母婴 #品牌名
   ↑核心        ↑产品       ↑长尾       ↑行业    ↑品牌
```

### 7.3 标签数量建议

- **必选标签**：2个（核心+产品）
- **建议标签**：2-3个（长尾+行业）
- **可选标签**：1个（品牌）
- **总计**：5-6个为最佳
