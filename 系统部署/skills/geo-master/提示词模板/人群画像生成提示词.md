# 人群画像生成提示词模板

> 核心思维：**问题在先，人群在后**
> 目标：精准长尾细分市场，每批人群有统一的痛点状态
>
> ⚠️ 已集成到系统：访问 `/admin/persona` 使用可视化界面

---

## 核心逻辑

```
第一步：问题挖掘
├── 使用方问题 → 产品/服务要解决的核心问题
│   ├── 具体表现
│   ├── 严重程度
│   └── 用户意识（有意识 vs 无意识）
│
├── 付费方顾虑 → 购买决策时的心理障碍
│   ├── 真假顾虑
│   ├── 价格顾虑
│   ├── 选择顾虑
│   └── 信任顾虑
│
└── 购买方与使用方关系
    ├── 购买方 = 使用方（本人决策）
    └── 购买方 ≠ 使用方（分开决策）

第二步：按问题分批生成人群画像
└── 每个问题 = 一个细分人群 = 统一的痛点状态
```

---

## 第一阶段：问题挖掘

### 使用方问题（产品/服务要解决的核心问题）

```
是谁的问题？什么问题？
├── 具体表现是什么？（症状/场景）
├── 严重程度/紧急程度
└── 用户自己知道问题所在吗？（有意识 vs 无意识）
```

### 付费方决策顾虑（购买时的心理障碍）

```
├── 真假顾虑：会不会买到假货/不靠谱/没效果？
├── 价格顾虑：值不值这个价？有没有更便宜的？
├── 选择顾虑：哪个更适合我？怎么判断优劣？
└── 信任顾虑：商家说的是真的吗？别人用过吗？
```

### 特殊情况判断

```
□ 购买方 = 使用方（本人使用，本人购买）
□ 购买方 ≠ 使用方（分开的，如：家长买给宝宝用、子女买给老人用、下属买给领导用）
```

```
- 痛点程度：

### 决策顾虑（付费方）
- 真假顾虑：
- 价格顾虑：
- 选择顾虑：

### 用户旅程
[问题产生] → [意识到问题] → [寻找解决方案] → [评估选项] → [购买] → [使用反馈]

### 内容触点
- 主动搜索：[用户会搜什么]

```

## 输出格式示例

### 示例：奶粉行业

```markdown
## 第一阶段：问题梳理

### 使用方问题
| 问题 | 具体表现 | 严重程度 |
|------|----------|----------|
| 乳糖不耐受 | 喝奶后腹泻、胀气 | 高（直接影响健康） |
| 牛奶蛋白过敏 | 湿疹、吐奶、便血 | 高（过敏反应） |
| 消化不良 | 便秘、上火、不长肉 | 中（慢性问题） |

### 付费方顾虑
| 类型 | 具体顾虑 |
|------|----------|
| 真假 | 是不是正品？渠道正规吗？ |
| 价格 | 比国产贵这么多，值得吗？ |
| 选择 | 太多品牌，不知道选哪个 |
| 信任 | 朋友圈代购靠谱吗？ |

### 特殊情况
☐ 购买方 ≠ 使用方
- 使用方：宝宝（0-3岁）
- 付费方：妈妈/爸爸/长辈

---

### 第二阶段：分批生成

#### 批次1：乳糖不耐受人群

**使用方**：乳糖不耐受的宝宝
**付费方**：宝宝拉肚子治不好的妈妈

**使用方问题**：
- 核心问题：喝了含乳糖的奶粉就腹泻
- 具体表现：大便稀、有泡沫、哭闹
- 痛点程度：急迫（影响生长发育）

**付费方顾虑**：
- 真假：深度水解/无乳糖奶粉怎么辨别真假？
- 价格：特殊奶粉比普通奶粉贵好多，值吗？
- 选择：爱他美、惠氏、雀巢……到底选哪个？

---

#### 批次2：过敏体质人群

**使用方**：牛奶蛋白过敏的宝宝
**付费方**：四处求医问药却找不到合适奶粉的妈妈

**使用方问题**：
- 核心问题：普通奶粉导致过敏反应
- 具体表现：湿疹反复、吐奶、生长发育迟缓
- 痛点程度：非常急迫（长期影响健康）

**付费方顾虑**：
- 真假：氨基酸奶粉怎么验真伪？
- 价格：一个月要花多少钱？
- 选择：适度水解、深度水解、氨基酸……到底喝哪个？
- 信任：母婴店推荐的会不会是为了利润？

---

### 内容选题方向

| 人群 | 用户会搜 | 内容方向 |
|------|----------|----------|
| 乳糖不耐受 | "宝宝喝奶粉拉肚子怎么办" | 问题解决型 |
| 牛奶蛋白过敏 | "宝宝湿疹是奶粉的原因吗" | 原因分析型 |
| 消化不良 | "奶粉上火便秘怎么办" | 解决方案型 |
```
