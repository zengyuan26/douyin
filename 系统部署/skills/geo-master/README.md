# 📚 Geo Master Skill - 使用指南

## 概述

`geo-master` 是整个本地化内容运营Skill系统的**总控专家**，负责接收用户需求、收集客户资料、分析业务场景、调度领域专家、整合输出完整解决方案。

作为中枢大脑，统筹协调抖音运营全链路专家团队。

## 使用流程

### 第一阶段：需求接收与资料收集

```
用户首次咨询 → geo-seo-master → 调度 client-intake → 生成客户档案
```

### 第二阶段：需求分析与专家调度

```
客户档案 → geo-seo-master分析 → 确定专家组合 → 调度各专家
```

### 第三阶段：方案生成与整合

```
各专家输出 → geo-seo-master整合 → 生成完整方案 → 输出给用户
```

---

## 专家调度规则

### 调度公式

```
必选专家 = geo-seo（所有项目）

可选专家 =
  - 若需市场洞察 → 调度 insights-analyst
  - 若需内容创作 → 调度 content-expert
  - 若需心理优化 → 调度 consumer-psychology-expert ⭐必选
  - 若需客户收集 → 调度 client-intake（或内置）
```

### 客户类型与专家组合

| 客户类型 | 地域范围 | 品牌类型 | 推荐专家组合 |
|----------|----------|----------|--------------|
| 卖货类 | 任意 | 任意 | geo-seo + content-expert + psychology-expert |
| 卖货类 | 本地 | 任意 | geo-seo + insights-analyst + psychology-expert |
| 服务类-本地 | 本地 | 个人IP | geo-seo + content-expert + psychology-expert |
| 服务类-本地 | 本地 | 公司品牌 | geo-seo + insights-analyst + psychology-expert |
| 服务类-跨区域 | 跨区域 | 任意 | geo-seo + insights-analyst + psychology-expert |
| 品牌建设 | 任意 | 任意 | content-expert + insights-analyst + psychology-expert |

## 快速模式

当用户需求简单时，可使用快速模式：

```
用户：我是卖奶粉的，帮我做抖音

AI（快速模式）：
✅ 已识别：卖货类 + 跨区域 + 可选品牌

📋 快速方案：

【关键词建议】
- 核心词：奶粉、婴儿奶粉
- 产品词：有机奶粉、羊奶粉
- 长尾：奶粉怎么选、奶粉推荐

【选题建议】
1. 奶粉怎么选？90%妈妈踩过的坑
2. 有机奶粉真的好吗？真相来了
3. 2025年奶粉红黑榜

【内容规划】
- 日更/周更：_____
- 模板类型：问题解决型

需要我生成完整的执行方案吗？
```
