## 一、10项内容质量检查清单

> ⚠️ **每次内容生成后，必须使用此清单进行自检**

### 检查项汇总表

| 序号 | 检查项 | 10分标准 | 核心问题 |
|------|--------|----------|----------|
| 1 | 标题 | 问题句或价值承诺 | 标题平淡无吸引力 |
| 2 | 开篇 | 前3句话有核心答案 | 开篇冗长抓不住人 |
| 3 | 结构 | 有小标题+列表 | 结构混乱无层次 |
| 4 | 模块化 | 段落独立完整 | 段落依赖性强 |
| 5 | 信任证据 | 有案例/数据/引用 | 缺乏说服力 |
| 6 | 品牌锚点 | 开头+结尾+中间有IP | IP植入不足 |
| 7 | 关键词 | 2-3次自然出现 | 关键词堆砌/不足 |
| 8 | 可读性 | 短段+清爽+接地气 | 段落冗长难读 |
| 9 | 行动号召 | 唯一+明确+低门槛 | 行动模糊 |
| 10 | 改造潜力 | 可改结构图+短视频 | 无法二次利用 |

## 二、详细评分标准

### 2.1 标题检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 问题句开头 + 悬念感强 + 7字以内 |
| 8分 | 问题句开头 + 悬念感一般 |
| 6分 | 有吸引力但非问题句 |
| 4分 | 平铺直叙，无吸引力 |
| 0分 | 标题与内容不符 |

**优化标准**：
```
✅ 优秀标题：问题句/价值承诺/颠覆认知
❌ 问题标题：陈述句/平淡无奇/太长
```

**优化方向**：
- 改为问题句："XX怎么办？"、"XX你做对了吗？"
- 改为价值承诺："1招解决XX"、"XX必看的3个方法"
- 改为颠覆认知："不是XX的问题，是XX的问题"

---

### 2.2 开篇检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 前3句话直接给出核心答案 |
| 8分 | 前5句话有核心答案 |
| 6分 | 前10句话有核心答案 |
| 4分 | 需读完全篇才能理解核心 |
| 0分 | 无核心观点 |

**优化标准**：
```
✅ 优秀开篇：开门见山，前3句有答案
❌ 问题开篇：铺垫太长，用户流失
```

**优化方向**：
- 删除无关背景铺垫
- 前3句话直接说结论/答案
- 使用"已经"原则："我已经XX"、"这个问题已经XX"

---

### 2.3 结构检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 有小标题 + 多级列表 + 逻辑清晰 |
| 8分 | 有小标题 + 简单列表 |
| 6分 | 有分段但无小标题 |
| 4分 | 大段文字无结构 |
| 0分 | 全文无分段 |

**优化标准**：
```
✅ 优秀结构：小标题 + 列表 + 层次分明
❌ 问题结构：堆砌文字 + 无层次感
```

**优化方向**：
- 每300字添加一个小标题
- 使用数字列表（1/2/3）
- 使用项目符号（• - ★）

---

### 2.4 模块化检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 每段独立完整，可单独提取 |
| 8分 | 80%段落独立 |
| 6分 | 60%段落独立 |
| 4分 | 段落依赖性强 |
| 0分 | 全文强依赖，无法拆分 |

**优化标准**：
```
✅ 优秀模块化：每段独立 + 承上启下
❌ 问题模块化：段落强依赖 + 无法拆分
```

**优化方向**：
- 每段开头有独立论点
- 每段结尾有独立结论
- 添加过渡句连接段落

---

### 2.5 信任证据检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 有具体案例 + 数据支撑 + 权威引用 |
| 8分 | 有案例 + 数据或引用之一 |
| 6分 | 有简单案例 |
| 4分 | 无具体案例 |
| 0分 | 纯主观陈述 |

**优化标准**：
```
✅ 优秀信任：案例 + 数据 + 引用
❌ 问题信任：空洞描述 + 无证据
```

**优化方向**：
- 添加具体案例："XX客户/用户"
- 添加数据："90%的人"、"3个方法"
- 添加引用："专家说"、"数据显示"

---

### 2.6 品牌锚点检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 开头+结尾+中间均有IP/品牌 |
| 8分 | 开头+结尾有，中间缺少 |
| 6分 | 仅结尾有品牌信息 |
| 4分 | 无品牌信息 |
| 0分 | 提及竞品过多 |

**优化标准**：
```
✅ 优秀锚点：开头+中间+结尾均有IP
❌ 问题锚点：品牌存在感不足
```

**优化方向**：
- 开头：IP自我介绍或背景铺垫
- 中间：IP亲身经历或观点输出
- 结尾：IP总结或行动引导

---

### 2.7 关键词检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 核心关键词自然出现2-3次 |
| 8分 | 关键词出现1-2次但自然 |
| 6分 | 关键词出现但生硬 |
| 4分 | 关键词堆砌 |
| 0分 | 无关键词或堆砌明显 |

**优化标准**：
```
✅ 优秀关键词：自然出现 + 2-3次
❌ 问题关键词：堆砌/不足/生硬
```

**优化方向**：
- 标题包含核心关键词
- 正文自然嵌入2-3次
- 标签包含关键词

---

### 2.8 可读性检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 短段落 + 排版清爽 + 语言接地气 |
| 8分 | 段落较短 + 排版清晰 |
| 6分 | 有长段落但可读 |
| 4分 | 长段落多 + 排版混乱 |
| 0分 | 难以阅读 |

**优化标准**：
```
✅ 优秀可读性：每段≤3行 + 语言接地气
❌ 问题可读性：长段落 + 生硬术语
```

**优化方向**：
- 每段≤3行
- 删除专业术语，改用口语
- 添加表情符号区分层次（适量）

---

### 2.9 行动号召检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 唯一行动 + 明确具体 + 门槛极低 |
| 8分 | 有行动 + 较明确 |
| 6分 | 有行动但模糊 |
| 4分 | 行动不明确 |
| 0分 | 无行动号召 |

**优化标准**：
```
✅ 优秀行动：唯一 + 明确 + 低门槛
❌ 问题行动：模糊/多重/门槛高
```

**优化方向**：
- 只留一个核心行动
- 明确告诉用户做什么
- 降低行动门槛：收藏 > 评论 > 私信

---

### 2.10 改造潜力检查（10分）

| 分数 | 标准 |
|------|------|
| 10分 | 可提炼结构图 + 可改短视频 |
| 8分 | 可提炼结构图 |
| 6分 | 结构较清晰 |
| 4分 | 无法提炼 |
| 0分 | 内容混乱 |

**优化标准**：
```
✅ 优秀改造：结构清晰 + 可多形式利用
❌ 问题改造：形式单一 + 无法复用
```

**优化方向**：
- 内容结构化（有编号/列表）
- 可提炼为思维导图
- 可改写为短视频脚本

## 四、自动优化流程

### 4.1 优化触发条件

```
评分 < 80分 → 触发自动优化流程
评分 ≥ 80分 → 直接输出
```

### 4.2 优化对应关系

| 问题项 | 问题描述 | 优化操作 |
|--------|----------|----------|
| 标题不通过 | 平淡无吸引力 | 改为问题句或价值承诺 |
| 开篇不通过 | 铺垫太长 | 删除铺垫，前3句给答案 |

| 品牌锚点不通过 | IP不足 | 增加开头/中间/结尾IP |
| 关键词不通过 | 堆砌/不足 | 自然出现2-3次 |
| 可读性不通过 | 难读 | 拆分长段落 + 接地气 |
| 行动号召不通过 | 模糊 | 明确唯一低门槛行动 |
