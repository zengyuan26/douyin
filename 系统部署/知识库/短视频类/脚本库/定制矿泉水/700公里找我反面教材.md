## 二、问题根源

### 厂家思维的极致表现

| 错误思维 | 后果 |
|----------|------|
| "来找我订" | 用户是来求你？ |
| "700多公里" | 远＝厉害？关我什么事 |
| "源头工厂" | 用户不关心你是不是源头 |
| "小批量定制" | 供应链术语，用户听不懂 |

### 用户视角 vs 厂家视角

| 厂家视角（错） | 用户视角（对） |
|---------------|----------------|
| 你愿意700公里来找我吗？ | 我帮你找到最近的定制水服务商 |
| 我是源头工厂 | 帮你省30%成本的定制水 |
| 小批量定制 | 1件起订，不用囤货 |
| 来找我订 | 在家就能定制，送货上门 |

## 五、学到的教训

| 教训 | 说明 |
|------|------|
| ❌ 不要问用户"来找我" | 你是服务方，不是甲方 |
| ❌ 不要强调距离 | 远≠好，用户关心的是方便 |
| ❌ 不要用供应链术语 | 用户听不懂 |
| ❌ 2.5秒太短 | 什么都说不清楚 |
| ✅ 要站在用户角度 | 用户关心的是"能帮我解决什么" |
| ✅ 要说用户语言 | 简单直白 |
