## 三、问题根源

### 为什么这个视频做不起来？

| 错误思维 | 后果 |
|----------|------|
| **厂家思维** | "我是源头厂家"——用户不关心 |
| **产品思维** | "我的水好"——没有差异化 |
| **抽象价值** | "仪式感"——谁都会说 |
| **自嗨式表达** | 没有站在用户角度 |

### 对比：用户思维 vs 厂家思维

| 厂家思维（错） | 用户思维（对） |
|---------------|----------------|
| "我是源头厂家" | "订婚宴用这个水更有面子" |
| "定制水有仪式感" | "一张婚纱照成就一瓶有意义的定制水" |
| "我们价格便宜" | "结婚就这一次，一定要特别" |
| "产品品质好" | "用这个水，亲朋好友都夸有创意" |

## 五、学到的教训

| 教训 | 说明 |
|------|------|
| ❌ 不要说正确的废话 | "仪式感"谁都会说，没有差异化 |
| ❌ 不要站在厂家角度 | 用户不关心你是不是厂家 |
| ❌ 不要只讲产品 | 要讲场景、故事、情感 |
| ✅ 要具体不要抽象 | "婚纱照"比"仪式感"具体100倍 |
| ✅ 要人群召唤 | 告诉谁应该看 |
| ✅ 要场景具体 | 订婚宴 > 结婚 > 仪式感 |
