## 🔍 分析维度框架

### 通用分析维度
| 维度 | 说明 |
|------|------|
| 标题 | 标题类型、关键词、吸引力 |
| 标签 | 话题标签、SEO关键词 |
| 内容结构 | 整体框架、逻辑顺序 |
| 选题 | 选题方向、痛点/痒点 |
| 关键词布局 | 核心关键词、长尾词分布 |
| 内容形式 | 呈现方式、表现形式 |

### 各类型专项维度

#### 图文类专项
- 封面设计：构图、配色、文案
- 排版风格：字体、间距、图片搭配
- 互动引导：评论引导、点赞引导

#### 短视频类专项
- 开头钩子：前3秒吸引力
- 节奏把控：时长、节奏、情绪曲线
- 结尾引导：互动话术、下期预告
- BGM/音效：背景音乐选择
