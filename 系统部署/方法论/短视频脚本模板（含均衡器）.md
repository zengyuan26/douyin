## 四、符号说明

| 符号 | 类型 | 说明 | 密度计算 |
|------|------|------|----------|
| 🪝 | 钩子 | 3秒内抓住注意力 | 固定密度 |
| 🎯 | 干货 | 有用的知识点 | =有用配置值 |
| 💡 | 梗点 | 有趣的段子/反转 | =有趣配置值 |
| 😂 | 共鸣 | 情绪触动的点 | =共鸣配置值 |
| 📚 | 收藏点 | 值得收藏的内容 | 固定1-2个 |
| CTA | 行动引导 | 关注/评论引导 | 固定1个 |

## 七、使用流程

### Step 1：选择模板或自定义配置

```
1. 选择预设模板（A/B/C/D/E）
2. 或自定义调整均衡器参数
```

### Step 2：生成时间轴

```
系统根据配置自动计算：
- 干货点数量和位置
- 梗点数量和位置
- 共鸣点数量和位置
- 钩子位置和密度
```

### Step 3：填充内容

```
根据时间轴位置，填充具体内容：
- 钩子位置 → 放最强痛点/震惊点
- 干货位置 → 放实用知识点
- 梗点位置 → 放幽默反转
- 共鸣位置 → 放情感触点
```

### Step 4：生成即梦提示词

```
根据内容生成对应的视频提示词
```
