# 图文内容模板

> **【选题】** 餐饮行业瓷器修复通用方案
>
> **【核心文案】** 这一招 酒店餐厅茶楼都在用 一年省下10万+
>
> **【目标客户】** 餐饮全行业（高端酒店/会所餐厅/茶楼茶室）
>
> **【行业】** 餐饮行业通用
>
> **【强制规格】** 图文 | 不限张数 | 9:16竖版 1080×1920px
>
> **【核心要求】** 严禁纯色背景！每张图背景必须与内容相关

---

## 一、选题策略

### 1.1 内容定位

| 项目 | 内容 |
|------|------|
| **核心痛点** | 餐饮行业瓷器/茶具损耗大，成本居高不下 |
| **心理策略** | 行业通用 + 降本刚需 - 各行各业都在用的省钱方案 |
| **内容类型** | B端获客 + 干货价值 + 行业覆盖 |
| **目标人群** | 酒店/会所/餐厅/茶楼管理者、老板 |
| **内容目的** | 行业痛点 → 通用方案 → 信任建立 → 咨询转化 |

### 1.2 选题逻辑

```
餐饮行业通用 → 各业态痛点 → 省钱方案 → 行业案例 → 咨询转化
```
