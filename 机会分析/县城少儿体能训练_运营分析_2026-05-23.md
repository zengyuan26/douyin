## 一、核心问题：一个人 2.5 天的收入天花板

```
时间槽（一个人上课，每槽1个班 × 10-12人）：

  周五晚：2 槽
  周六：  6 槽（上午2 + 下午2 + 晚上2）
  周日：  6 槽
  ─────────────
  合计：  14 槽/周

周总人次：14 × 10人 = 140 人次
每个孩子每周来 1-2 次，取 1.5：
  活跃学员上限：140 / 1.5 ≈ 90 人

但这是"体力极限"——每周连上14节课，不可能长期维持。
实际舒适线：50-60 个学员。
```

| 场景 | 学员数 | 月收入 | 备注 |
|------|--------|--------|------|
| 当前（推测） | 30-40 | 14,000-19,000 | |
| 舒适满员 | 60 | ~29,000 | 可持续 |
| 体力极限 | 90 | ~43,000 | 不可持续 |

**单人模式的天花板就是月入 3 万左右**。要突破，只有一个办法：招教练。

## 二、定价分析

当前定价在县城不算离谱，但课包之间的折扣梯度有问题：

```
少儿体能：
  10节 980  (98/节)  ← 基准价，OK
  30节 2380 (79/节)  ← 降19%，降幅偏大
  50节 3280 (66/节)  ← 比10节便宜33%，太狠了

中考体育：
  10节 1280 (128/节)
  30节 2880 (96/节)
  50节 4280 (86/节)
```

**问题**：买大课包的家长单价只有66元，太低了。这不是"跑量"，是"自己压自己价"。

**建议**：收窄折扣幅度。

| 课包 | 现价 | 建议价 | 新单价 | 涨幅 |
|------|------|--------|--------|------|
| 10节 | 980 | 1080 | 108 | +10% |
| 30节 | 2380 | 2780 | 93 | +17% |
| 50节 | 3280 | 3980 | 80 | +21% |

**涨价的抓手**：入训体能评估报告 + 阶段进步对比报告。县城其他培训班基本不做。多一张 A4 纸的报告，家长就觉得"值这个价"。

老学员续费给一次原价过渡期。

## 三、招教练 = 唯一突破天花板的路

```
现状死循环：
  一个人 → 最多带60人 → 月入3万封顶 → 不敢招人 → 继续一个人

打破方式：
  先招兼职教练 → 周末可以同时开2个班 → 容量翻倍 → 收入翻倍
```

### 县城兼职教练的真实成本

| 来源 | 课时费 | 适合带 |
|------|--------|--------|
| 本地体校学生/毕业生 | 50-80/节 | 基础体能、中考体育 |
| 幼师专业毕业生 | 60-80/节 | 3-6岁感统/体适能 |
| 学校体育老师兼职 | 80-100/节 | 中考体育 |

### 算一笔账

```
她一节课（10个孩子 × 80均价）= 800 元收入
教练课时费 = 80 元
教练成本占比 = 10%

10% 的成本换 100% 的容量翻倍。这个杠杆必须用。
```

### 教练会跑路单干吗？

不用太担心。教练只会上课，不会做三件事：
- 体能评估 + 个性化方案设计
- 和家长深度建立信任
- 品牌内容输出

这三件事始终握在她手里。教练离开她，只能做"带练的"，没有差异化。

**关键是：她必须先把评估体系 + 课程SOP 做出来。** 这是招教练的前提——新人来了知道每节课怎么上，而不是靠"看感觉"。

## 四、媒体杠杆：她现在最该做但可能没做的事

她有三个天然的内容优势，县城里很难复制：

```
1. "健身达人的少儿体能课"
   → 她自己的身材和训练状态就是活广告
   → 妈妈们想的是："她这么专业，带孩子肯定也靠谱"

2. "孩子训练前后对比"
   → 跳绳从20个到80个 / 体态明显改善 / 中考项目成绩提升
   → 家长看了直接想报名，这种内容转化率极高

3. "30+女性创业日常"
   → 县城里的同龄女性天然共鸣
   → 从"知道她" → "信任她" → "把孩子交给她"
```

**操作方式**：每节课顺手拍3-5段素材，不用精剪，手机直出就行。每周发3-5条抖音/视频号，打同城标签。

县城传播链路：同城页刷到 → 转发到班级家长群 → 体验课 → 成交。

这个动作成本为0，但可能是她获客效率最高的渠道。

## 五、收入模型：单人 vs 加教练

### 方案：招1个兼职教练，周末两个班并行

```
周末时间槽（她 + 教练各带一个班）：

  周五晚：2 槽 → 2 个班（她 + 教练）
  周六：  6 槽 → 12 个班
  周日：  6 槽 → 12 个班
  ─────────────
  周末合计：26 班次

每班 10 人，每人每周来 1.5 次：
  活跃学员上限：26 × 10 / 1.5 ≈ 170 人

但实际不会全满。按 120 学员算。
```

| | 单人模式（现在） | 加1教练 |
|------|------|------|
| 活跃学员 | 40-60 | 100-120 |
| 月收入 | 19,000-29,000 | 48,000-58,000 |
| 教练成本 | 0 | ~4,000/月 |
| 月利润 | 19,000-29,000 | 44,000-54,000 |

**加一个教练，利润接近翻倍。她从"主要上课的人"变成"上一半课 + 做评估 + 做获客"。**

## 六、路线图（修正版）

### 近期（1-2个月）：打好基础

```
□ 做入训体能评估表 + 阶段性进步报告模板
□ 写3-6岁 / 6-9岁 / 9-12岁 / 中考体育 各一套课程SOP
□ 每节课拍素材，启动抖音/视频号（周3-5条）
□ 老学员继续服务，稳定转介绍
```

### 中期（第2-3个月）：招教练 + 提价

```
□ 物色兼职教练（体校生/幼师优先，女性优先）
□ 教练跟课2周，熟悉SOP后独立带班
□ 新学员启用新价格（评估表已成型，涨价有抓手）
□ 周末容量从1个班 → 2个班
```

### 后期（第4-6个月）：稳定运转

```
□ 她自己上课时间降到总课时的40%以下
□ 主要精力：评估+方案+家长沟通+内容输出
□ 学员数稳定在100+
□ 考虑开第二个教学点或增加周中少儿课
```

## 七、总结

```
核心矛盾：   一个人 × 2.5天 = 天花板卡在月入3万

解法：       招教练（10%成本换100%容量）+ 提价（评估报告支撑）
             媒体杠杆（0成本获客）

房租不是问题 → 不用分心做成人课 → 专注把少儿这件事做透

她的优势：
  ├── 健身达人（专业背书，县城稀缺）
  ├── 女性（妈妈们天然信任）
  ├── 300平场地（容量充足，房租便宜）
  └── 已有学员基础（不是从零开始）

她缺的：
  ├── 课程SOP（没法交给别人）
  ├── 评估体系（涨价的抓手 + 留住教练的壁垒）
  └── 第一个教练（打破单人瓶颈）

优先级：
  1. 做评估表 + SOP
  2. 启动抖音拍素材
  3. 招兼职教练
  4. 新学员涨价
```
