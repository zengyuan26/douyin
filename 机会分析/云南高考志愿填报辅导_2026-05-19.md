# 【机会分析】云南昆明高考志愿填报辅导业务

## 一、机会本质

**一句话定义**：利用信息差和经验积累，在高考出分关键窗口期为云南考生家庭提供志愿填报决策服务的本地化教育咨询生意。

---

## 二、纳瓦尔框架分析

### 1. 定位分析

```
                    高价值输出              低价值输出
              ┌─────────────────┬─────────────────┐
      擅长    │  ★ 核心角色      │    执行角色      │
              │  咨询+决策      │   资料整理      │
              ├─────────────────┼─────────────────┤
      不擅长  │    不适合        │    边缘角色      │
              │  学校对接        │   发传单        │
              └─────────────────┴─────────────────┘
```

**建议定位**：**志愿填报策略顾问**（而非中介或销售）

### 4. 杠杆分析

| 杠杆类型 | 可用性 | 优先级 | 说明 |
|---------|--------|--------|------|
| 代码杠杆 | 低 | ❌ | 志愿填报需要人工判断，标准化难 |
| 媒体杠杆 | 高 | ★★★ | 抖音/小红书/微信 内容获客 |
| 人力杠杆 | 中 | ★★ | 雇兼职做执行，核心咨询自己做 |
| 资本杠杆 | 低 | ❌ | 不需要大资金，适合轻启动 |

**核心策略**：以**媒体杠杆**为主，获客；靠**经验复利**提升定价。

---

### 5. 子弹剥离分析

```
阶段1：子弹在身体里（当前阶段）⚠️
├── 靠时间和体力赚钱
├── 出分那几天=黄金窗口，错过就没
├── 停下来 = 收入停止
└── 天花板：一天最多服务X个家庭

阶段2：部分剥离（2.0模式）✓ 可以做到
├── 标准化咨询流程、模板化方案
├── 雇兼职做信息收集，自己专注判断
└── 可以同步服务更多客户

阶段3：完全剥离（3.0模式）⚠️ 有挑战
├── 开发录播课/工具 → 降低边际成本
├── 建立品牌口碑 → 被动流量
└── 但核心判断还是需要人来做
```

**关键洞察**：经验是"子弹"，但这个子弹不会自动脱离身体。**你需要主动设计系统**，把经验沉淀下来，否则你就是那个永远离不开的"专家"。

## 三、八条关键问题

| 问题 | 答案方向 |
|------|----------|
| 1. 机会本质 | 本地化教育咨询，卖的是"信息+经验+安心" |
| 2. 定位是什么 | 志愿填报策略顾问，不是卖资料的 |
| 3. 独特价值 | 云南本地政策熟悉度 + 见过足够多案例的判断力 |
| 4. 杠杆用什么 | 抖音/小红书内容（最重要）+ 口碑推荐 |
| 5. 子弹能否剥离 | 部分可以（流程标准化），但核心判断还得自己做 |
| 6. 关键决策点 | 定价多少？线上还是线下？如何建立信任？ |
| 7. 风险是什么 | 时间窗口短、口碑风险（报不好被骂）、竞争加剧 |
| 8. 退出机制 | 这个业务本身可以随时停，没有沉没成本 |

## 四、风险评估

| 风险类型 | 等级 | 应对策略 |
|---------|------|----------|
| **时间风险** | 🔴 高 | 提前布局内容，错峰接单 |
| **经验无法传承风险** | 🟡 中 | 主动沉淀SOP和案例库，否则永远是"一个人的生意" |
| **声誉风险** | 🟡 中 | 签服务协议，明确边界（"建议权"非"决定权"） |
| **竞争风险** | 🟡 中 | 差异化：本地化 + 专业度 + 经验年限 |
| **政策风险** | 🟢 低 | 政策每年变，反而是壁垒 |

## 五、最终建议

```
✅ 值得做的理由：
├── ★★★ 复利业务：做得越久越值钱
├── 云南高考市场稳定，年年有需求
├── 媒体杠杆可用，内容可以低成本获客
├── 门槛不高，可以轻启动验证
└── 10年后你可以定价3000-5000/单

⚠️ 需要谨慎的：
├── 时间窗口极短，提前3个月蓄客
├── 经验是壁垒，但也是陷阱（成为离不开的专家）
├── 不要只做执行者，要做经验沉淀者
└── 保护好个人声誉（一两个失败案例影响大）

💡 最佳策略：
├── 第一步：先做1-2个免费/低价案例，积累素材
├── 第二步：抖音/小红书发内容，建立本地认知
├── 第三步：标准化流程，雇人做执行，自己专注判断
└── 第四步：沉淀经验资产（案例库、方法论、口碑）

核心心法：把"经验"变成"可传承的资产"，而不是只有你会的秘密。
```

## 六、行动路线图

### 阶段1：蓄客期（现在-5月底）

```
目标：建立种子客户，积累内容素材

□ 找2-3个今年高考的家庭，做免费/低价咨询
□ 全程记录案例（脱敏后可用于内容）
□ 开通抖音/小红书账号，发布第一条内容
□ 收集常见问题，整理成FAQ

里程碑：3个真实案例 + 10条内容储备
```

### 阶段2：收割期（6月1日-7月15日）

```
目标：最大化高考出分窗口的咨询量

□ 密集发布内容（出分前焦虑期流量最大）
□ 开放预约名额（建议控制每天上限）
□ 使用标准化流程提效
□ 收集评价和案例

里程碑：完成20-30个咨询，客单价验证
```

### 阶段3：经验沉淀（7-12月）

```
目标：把"经验"变成"可传承的资产"

□ 整理今年的成功/失败案例，形成案例库
□ 提炼自己的方法论（判断逻辑、决策框架）
□ 开发辅助工具（对比表、流程图）
□ 规划明年内容矩阵

里程碑：案例库 + 方法论文档 + 明年计划
```

### 阶段4：规模化（第2年起）

```
目标：从"一个人做"到"系统运转"

□ 提高客单价（基于你的经验积累）
□ 雇兼职做信息收集和执行
□ 你专注做高价值判断
□ 考虑开发录播课/会员产品

里程碑：客单价提升 + 被动收入占比 > 20%
```
