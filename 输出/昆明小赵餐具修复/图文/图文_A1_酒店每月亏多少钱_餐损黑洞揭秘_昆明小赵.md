# 图文内容模板

> **【选题】** 酒店每月亏多少钱？餐损黑洞揭秘
>
> **【核心文案】** 酒店后厨餐损控制 从食材到餐具全揭秘
>
> **【目标客户】** 酒店管理者、餐饮总监、后厨主管、老板
>
> **【行业】** 酒店餐饮 / 后厨管理
>
> **【强制规格】** 图文 | 6张 | 9:16竖版 1080×1920px
>
> **【核心要求】** 严禁纯色背景！每张图背景必须与内容相关 | 数据驱动 | 专业分析

---

## 一、选题策略

### 1.1 内容定位

| 项目 | 内容 |
|------|------|
| **【核心痛点】** | 酒店后厨餐损严重但难以量化，管理者不知从何下手 |
| **心理策略** | 数据震撼 + 理性算账 + 专业分析 - 帮酒店人找到最优解 |
| **内容类型** | 行业揭秘 + 干货价值 + B端获客 |
| **目标人群** | 酒店老板、餐饮总监、后厨主管、成本核算人员 |
| **内容目的** | 痛点认知 → 数据震撼 → 双轨对比 → 引导选择 → 咨询转化 |

### 1.2 选题逻辑

```
酒店餐损现状 → 行业权威数据 → 双轨算账对比 → 哪个更划算 → 行动引导
```
