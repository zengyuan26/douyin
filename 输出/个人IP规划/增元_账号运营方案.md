## 📅 第二阶段：内容运营（7-30天）

### 2.1 发布节奏

| 阶段 | 频率 | 形式 | 目的 |
|------|------|------|------|
| 第1周 | 每天1条 | 口播 | 跑通流程、测试数据 |
| 第2周 | 每天1条 | 口播+问答 | 优化内容、找感觉 |
| 第3周 | 每天1条 | 口播+案例 | 积累代表作 |
| 第4周 | 每天1条 | 混合 | 建立节奏 |

### 2.2 内容日历

#### 周一：干货类
```
主题：方法论/技巧
标题公式：xxx的方法/技巧/公式
时长：45-60秒
结构：痛点引入 → 核心方法 → 总结
```

#### 周二：观点类
```
主题：反常识/争议
标题公式：为什么xxx？/xxx其实是个坑
时长：30-45秒
结构：抛观点 → 给理由 → 结案
```

#### 周三：故事类
```
主题：经历/案例
标题公式：我是怎么xxx的/那个xxx被我
时长：60-90秒
结构：背景 → 过程 → 结果 → 感悟
```

#### 周四：干货类
```
主题：工具/AI
标题公式：xxx工具/AI的正确用法
时长：45-60秒
结构：问题 → 工具 → 演示 → 总结
```

#### 周五：互动类
```
主题：问答/讨论
标题公式：粉丝问：xxx/你怎么看xxx？
时长：30-60秒
结构：问题 → 回答 → 引导评论
```

#### 周六：案例类
```
主题：实战案例
标题公式：xxx是怎么做到的/xxx的破局
时长：60-90秒
结构：问题 → 方案 → 结果 → 方法论
```

#### 周日：休息/直播
```
不發布 或 直播答疑
```

### 2.3 短视频结构模板

#### 黄金3秒开头
```
❌ 不要：大家好，今天讲...
✅ 要：90%的人做抖音都错了...
✅ 要：年收入100万的人，都在用这个方法
✅ 要：我最后悔的一件事...
```

#### 内容结构（45秒模板）
```
0-3秒：钩子（痛点/悬念/反问）
3-15秒：背景（为什么讲这个）
15-35秒：干货（核心观点/方法）
35-42秒：案例（证明你说的是对的）
42-45秒：引导（点赞/关注/评论）
```

#### 结尾引导模板
```
✅ "觉得有用，点个赞"
✅ "关注我，下期讲..."
✅ "有问题评论区见"
✅ "有收获的扣1"
```

## 📈 第三阶段：流量获取（持续）

### 3.1 抖音流量来源

| 流量类型 | 占比 | 策略 |
|----------|------|------|
| 推荐流量 | 60% | 完播率、互动率 |
| 搜索流量 | 30% | 关键词布局 |
| 私域流量 | 10% | 评论区引导 |

### 3.2 推荐流量优化

#### 关键指标（按权重）
```
1. 完播率（40%）→ 开头要抓人，内容要紧凑
2. 互动率（30%）→ 评论区引导、提问
3. 转发率（15%）→ 价值感、认同感
4. 收藏率（10%）→ 实用干货
5. 点赞率（5%）→ 认同感
```

#### 提升完播率技巧
```
✅ 语速稍快，不废话
✅ 每5秒一个转折/干货
✅ 时长控制在45-60秒
✅ 画面清晰、声音清楚
```

#### 提升互动率技巧
```
✅ 结尾提问："你怎么看？"
✅ 评论区置顶问题
✅ 回复有代表性的评论
✅ 每周做1条问答类内容
```

### 3.3 搜索流量优化

#### 关键词布局
```
标题：必须含关键词
内容：自然提到关键词
话题：带1-2个相关话题
评论区：可补充长尾词
```

#### 选题关键词库
```
核心词：线上获客、抖音获客、小红书获客
长尾词：
- 中小企业怎么做抖音
- 个体户怎么线上获客
- 抖音搜索流量怎么玩
- 小红书新手怎么起号
- AI做内容靠谱吗
```

### 3.4 评论区运营

#### 黄金1小时法则
```
发布后1小时内回复所有评论
前20条评论决定算法推荐
```

#### 评论回复模板
```
✅ 有价值的评论：认真回复+补充观点
✅ 提问：简短回答+引导私域
✅ 杠精：不反驳+用事实说话
✅ 同行：友好+但我的方法不同
```

#### 引导加微话术
```
✅ "这个问题展开讲太多，感兴趣的+V：xxxx"
✅ "想了解更多，V上聊"
✅ "有具体问题+V问我"
```

## 💰 第五阶段：变现转化

### 5.1 咨询服务流程

```
1. 初步沟通（微信）
   → 了解基本情况
   → 判断是否匹配
   
2. 需求诊断（语音15分钟）
   → 9.9元/免费
   → 明确问题
   
3. 方案报价
   → 根据问题复杂度报价
   → 1999-3999元/次
   
4. 执行陪跑
   → 1周-1个月
   → 定期沟通
```

### 5.2 报价策略

| 服务 | 价格 | 周期 | 适合 |
|------|------|------|------|
| 语音诊断 | 99元 | 15分钟 | 轻度问题 |
| 1对1咨询 | 1999元 | 2小时 | 明确需求 |
| 方案制定 | 3999元 | 3-5天 | 需要方案 |
| 月度陪跑 | 9999元 | 30天 | 深度服务 |
| 年度陪跑 | 3-9万/年 | 长期 | 全面合作 |

### 5.3 转化话术

```
问：增元老师，你能帮我做xxx吗？
答：
1. 先说说你的情况
2. 我评估一下能不能帮到你
3. 能帮：给方案、谈价格
```
