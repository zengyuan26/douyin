# 字幕断行规则

## 输入输出

输入：纠错后的文本（每句一行）
输出：字幕行纯文本（markdown，无时间戳、无标点）

## 核心原则

**语义优先，形态辅助**。只做格式化——不改字、不删字、不加字。

## 字数

每行 ≤ 16 字。能单行完整就不拆。

## 断点优先级

1. 句子/从句自然边界
2. 连词前（但是、所以、因为、然后、而且）
3. 话题转折处（其实、另外）
4. 动词前（主语留上行，谓语下行）

## 绝不拆分的单元

同一行内必须保持完整：
- 动宾搭配、偏正短语、介词短语、数量词+名词、专有名词
- "的"字结构（不在"的"前断）

## 形态

两行不等时下行更长（梯形）。避免上行只有两三字。

## 冲突解决

超过 16 字时换断点或加一行，**绝不删词来缩短**。

## 标点 / 空格 / 分段

- 删除所有中英文标点（保留英文连字符如 GPT-4）
- 中英/中数之间不加空格
- 英文单词之间保留空格
- 话题切换空行分隔，不插入 `---`

## 语言

简体中文。英文专有名词保持原文，数字用阿拉伯数字。
