# 趣味性参考：口播/视频脚本

> 说人话解决"不假"。这份文档解决"不无聊"。
> 适用：口播脚本、视频文案、public-writing 中需要观众留存的内容。

说人话的清洗流程完成后，文本已经自然了。但口播/视频还需要观众不划走。这份文档补上趣味性维度。

## 使用顺序

1. 先完成说人话标准流程（判场景 → 删AI腔 → 保真回读）
2. 再按本文做趣味性检查
3. 两者不冲突：说人话是底线，趣味性是上限

## 一、前3秒铁律

第一句话必须能让人停下来。以下三种直接判 FAIL：

- "大家好。这一期我们来聊XX" → 删掉，直接砸故事或数字
- "XX行业最近发生了三件事" → 删掉，挑最炸的一件事开头
- 任何以"在当今""随着""众所周知"开头的 → 删掉

合格开头至少满足一样：具体的人、具体的数字、具体的画面。

```
✅ "一个山东做铝材出口的朋友，上个月给我看了他的绿证账单。"
✅ "深圳，一度电五块。不是电价，是调度补偿。"
✅ "傍晚六点。太阳落山。光伏十分钟归零。"
```

## 二、全篇趣味性检查

### 2.1 乐教交替

连续 2 页以上纯"教"（定义/原理/政策时间线）→ 插入一个乐点：

| 乐点类型 | 例子 |
|---------|------|
| 问观众 | "你是哪种？""你自己在哪？" |
| 给画面 | "风在西北吹，电在东部用" |
| 数字炸弹 | "深圳实时五块钱一度" |
| 直白判断 | "说白了就是糊弄" |
| 叫停警告 | "内陆偏远工业园？门关了。" |

### 2.2 画面感优先于概念名

解释概念时，先给画面，再给名字。

```
❌ "第一层是资源卡——风电光伏集中在西北，用电大户在东部。"
✅ "风在西北吹，电在东部用。隔两三千公里。这叫资源卡。"
```

### 2.3 个人化

每页至少让观众对号入座一次。泛泛的行业分析 = 别人的事。指向"你" = 自己的事。

```
❌ "出口企业被碳关税卡着，只能继续买绿证。"
✅ "大多数人在第二种——被碳关税卡着，又不够条件跳到第三种。明年价格翻10倍，你还得买。"
```

### 2.4 情绪起伏

全篇匀速 = 催眠。检查每一页的情绪定位：

- 紧张（涨价/关门/卡脖子）→ 用短句、重复句号、数字
- 释压（出路/方法/案例）→ 用稍长句、解释、对比
- 意外（反直觉/新角度）→ 用单字句、破折号、停顿

禁止连续 3 页都是同一情绪。

### 2.5 句长变化

念一遍。以下两道检查：

- 有没有连续 5 句差不多长的？→ 拆一句成三个字，或并两句成一句
- 有没有每句 15-25 字均匀分布的？→ 插入 2-3 字的短句制造节奏

```
❌ 全是中长句：
"绿证分三种。前两种花钱买。第三种不花这笔钱。普通绿证几毛钱一个。绿电交易绿证涨最凶。"

✅ 长短交错：
"绿证分三种。你是哪一种？
第一种。几毛钱一张。年度统计，大概绿色。国内考核凑合用。
第二种。绿电交易绿证。从三五块涨到七块多。涨最凶。"
```

### 2.6 评论引导

结尾不给观众一个评论的理由 = 白送互动机会。

```
❌ "下期见。"
❌ "关注我，带你了解新能源。"
✅ "你现在在哪个阶段？评论区聊聊。"
✅ "你卡在哪一档？"
```

## 三、念稿测试

```
□ 念一遍。有没有地方念着喘不过气？→ 拆句
□ 念一遍。有没有连续 10 秒同一个语调？→ 在该处插入短句/问句/停顿
□ 念一遍。观众会在哪笑/愣/发弹幕？→ 不知道 = 没情绪。回去加情绪锚点
□ 有没有连续 5 句以上同样长度？→ 打断节奏
```

## 四、与说人话的关系

说人话的 positive-style.md 追求"具体动作、真主语、节奏自然"。这些是趣味性的基础——去掉AI腔自然会有一些趣味性提升。但不够。

趣味性额外要求：
- 主动设计情绪起伏（不是说人话的"不要整段一样齐"）
- 主动制造意外和反差（不是说人话的"允许普通句子"）
- 主动引导互动（不是说人话的范畴）

两者按顺序执行：先说人话清理，再做趣味性增强。不要混着做——清理时过度追求趣味会引入新的假。
