# 电影叙事工艺 · 创作自查

> 从抖音电影拉片博主（165集）提取的叙事工艺。每次生成短视频脚本或图文时强制执行。
> 详细理论见：`知识库/方法论库/电影叙事工艺.md`

---

## 执行时机

生成**短视频（即梦）脚本**或**A型图文**时，在输出前强制走完本检查清单。

---

## 10 项强制自查

### 1. 类型契约

```
□ 这个视频跟观众签的是什么约？
  → 概念拆解（平行世界故事） / 个人经历（来时路） / 框架知识（清单体）
□ 只签了一份约吗？（混合类型 = 混淆预期 = 划走）
```

### 2. 激励事件（前5秒）

```
□ 前5秒打破了观众的什么平衡？
  → 不是"今天讲XX"。是替观众说出他们心里已有的困惑
□ 真实困惑够具体吗？
  → "为什么92/95/98号汽油价格不一样" ✓ / "你知道汽油标号吗" ✗
```

### 3. 因果链（1→X）

```
□ 每个镜头之间是"所以"还是"然后"？
  → 因为规则定了，所以阿大自己挑。因为阿大自己挑，所以腰不行了。
  → 不是：阿大自己挑，然后阿二雇人，然后阿三买泵……
□ 删掉中间任何一步，后面的还能成立吗？
  → 能成立=那一步多余，删除
```

### 4. 条件约束（只有…才…所以）

```
□ 规则/条件够具体吗？
  → "水一块钱一桶，一个来回半小时，起步都是两只手"
  → 条件越具体，"才"的选择越有重量
```

### 5. 无废镜（过场戏不存在）

```
□ 每一个镜头/每一页都服务于核心问题吗？
□ 删掉镜X，观众还能理解这个故事吗？→ 能 = 必须删
```

### 6. 心理→精神（顺序不能错）

```
□ 先满足了观众的心理需求（故事/人物/悬念/情绪），再上价值吗？
  → 平行世界法天然满足：前80%故事→翻译→反问
  → 不能一上来就讲道理、列框架、上深度
```

### 7. 在场 > 奇观

```
□ 让事实自己说话了吗？
  → "但水管还在。水还在流。" ✓
  → "你能想象吗！水管居然还在流！" ✗
□ 有没有替观众激动？（"震惊""万万没想到""惊呆了"）
  → 有 = 删除这些词。相信观众自己能判断
```

### 8. 视觉合理优先

```
□ 情绪转折处，视觉上缓冲了吗？
  → P4→P5（乐观→清醒）：必须叠化/意象延续，不能硬切
  → 镜4→镜5（沉默→打通）：构图变化不能跳跃
□ 相邻镜头/页面之间，颜色/形状/情绪有衔接吗？
```

### 9. 逆向推导

```
□ 先确定了结尾感受，再倒推每一镜的吗？
  → 结尾：观众自照（"我加的是哪一种"）
  → 倒推：必须有翻译→必须有命运对比→必须有规则→必须有世界
  □ 结尾感受和开场困惑是闭环的吗？
```

### 10. 结构筛选（系统决定元素）

```
□ 主线（一个概念）定了吗？
□ 每一个出现的人/物/信息，是结构要求的吗？
  → 不是=不放。"不讲纳瓦尔生平、不讲复利、不讲金融史"
```

## 快捷输出格式

自查完成后，在脚本末尾附加：

```
## 电影工艺自查

□ 类型契约: [类型名] | 唯一: [✓/✗]
□ 激励事件: [前5秒的具体困惑] | 打破平衡: [✓/✗]
□ 因果链: [1→X简述] | 无多余步: [✓/✗]
□ 条件约束: [规则简述] | 够具体: [✓/✗]
□ 无废镜: [总镜数]镜 | 删任何一镜仍成立: [是/否]
□ 心理→精神: 前80%故事→翻译→反问 | 顺序正确: [✓/✗]
□ 在场>奇观: 无"震惊/惊呆了/你能想象吗"等词: [✓/✗]
□ 视觉优先: 情绪转折有视觉缓冲: [✓/✗]
□ 逆向推导: 结尾感受←倒推每一镜: [✓/✗]
□ 结构筛选: 无结构外元素: [✓/✗]
```
