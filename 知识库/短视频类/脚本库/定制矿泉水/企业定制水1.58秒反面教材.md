## 五、学到的教训

| 教训 | 说明 |
|------|------|
| ❌ 视频不是越短越好 | 1.58秒什么都说不清楚 |
| ❌ 不要说正确的废话 | "满足您的需求"谁都会说 |
| ❌ 不要用空洞的词 | "专属定制"没有意义 |
| ✅ 要有具体场景 | 告诉用户能做什么 |
| ✅ 要有具体人群 | 告诉用户给谁用 |
| ✅ 合理的时长 | 15-30秒为宜 |
