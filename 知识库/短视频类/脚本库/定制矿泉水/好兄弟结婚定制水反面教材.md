## 三、问题根源

### 同样是厂家思维

| 问题 | 说明 |
|------|------|
| **自嗨式表达** | 站在厂家角度写内容 |
| **没有场景化** | 没说具体什么场景用 |
| **没有差异化** | 和其他定制水没有区别 |
| **没有情感** | 有"好兄弟"但没有展开 |

### 如果是好案例，会怎么写？

| 错误写法 | 正确写法 |
|----------|----------|
| 好兄弟过年要结婚了，婚礼用水都是结婚定制水 | 好兄弟结婚，我送了他一件"特别"的婚礼定制水 |
| 婚礼用水都是定制水 | 婚礼敬酒时，亲朋好友喝的都是印着新人照片的水 |
| #江南农泉定制水 | #好兄弟婚礼 |

## 五、学到的教训

| 教训 | 说明 |
|------|------|
| ❌ 有好兄弟还不够 | 要讲具体故事 |
| ❌ 有时间节点还不够 | 要有具体场景 |
| ❌ 站在厂家角度 | 用户不关心你是谁 |
| ✅ 要场景化 | 敬酒、迎宾、婚房... |
| ✅ 要有故事 | 好兄弟→为什么送→怎么送→什么反应 |
| ✅ 要有价值 | 定制水让婚礼更有意义 |
