# Cinema DNA 视觉方法论

> 来源：cinema-dna-21x9x3 skill v1.2.1 + V4 anti-AI patch。提取适用于短视频（即梦/Seedance）和图文创作的视觉判断框架。
> 核心原则：**构图不是装饰，是人物与空间的权力关系。色彩不是滤镜，是事件发生方式的一部分。电影感不是颗粒和色散，是摄影机在正确的位置拍到了正确的停顿。**

---

## 一、构图压力库（7类）

构图不由"好看"决定，由人物与空间之间的**权力关系**决定。先判断压力类型，再选构图。

### A. 被观看（Observed）
人物被监视/被审视/有秘密。
- 空间线索：裂缝、玻璃、窗帘、人群中的视线、镜面反射
- 机位：透过玻璃拍、门缝视角、人群缝隙、监控视角
- 构图特征：前景遮挡物占画面1/3以上，人物不完全可见

### B. 被困（Trapped）
人物被空间约束，无法逃脱。
- 空间线索：工位隔板、走廊尽头、楼梯间、机构空间（医院/警局/办证大厅）
- 机位：极端高角度俯拍（人物<空间的1/6）、门框压缩、天花板压低
- 构图特征：几何边界切割人物身体，头部空间极度压缩

### C. 关系距离（Relational Distance）
两个人之间的物理距离 = 心理距离。
- 空间线索：空桌面、玻璃隔断、地板、床的两侧
- 机位：侧面拍两人之间的空桌面、从第三人肩膀拍过去
- 构图特征：画面重心在两人之间的空白，不在人物脸上

### D. 权力不对称（Power Asymmetry）
谁高谁低、谁大谁小、谁在前面谁在后面。
- 空间线索：大墙、楼梯、旗杆位置、讲台高度、座位安排
- 机位：低角度仰拍权力方、高角度俯拍弱势方
- 构图特征：人物与建筑元素的比例极度悬殊

### E. 心理失衡（Psychological Imbalance）
人物内心不稳定，但外表不表现。
- 空间线索：人物被挤到画面边缘、过度头部空间、轻微倾斜的地平线
- 机位：偏移中心构图、略倾斜的荷兰角（≤3°）
- 构图特征：画面重心偏移，"看着不舒服但说不出为什么"

### F. 事后状态（Post-event State）
事件已经发生完了，只留下痕迹。
- 空间线索：缺席的人物、空椅子、湿地面、开着的门、没喝完的水
- 机位：固定机位，不移动。摄影机是"晚来一步的目击者"
- 构图特征：画面最重的位置是空的（空椅子/空门口）

### G. 感官插入（Sensory Insert）
不是拍人脸，是拍身体局部——因为那一刻身体比脸更诚实。
- 空间线索：嘴、手、脖子、鞋、头盔边缘、装备扣带
- 机位：极近特写，浅景深但不糖水
- 构图特征：局部占画面60%以上，背景模糊但有信息

### 使用规则

```
□ 每镜/每页只用一个主导压力类型。不堆叠。
□ 压力类型必须从选题/人物状态推导，不能随意选。
□ 同一套内容的三个关键画面（开场/转折/收尾）各用不同的压力类型。
□ 自检：删掉画面后，人物和空间的关系还能被理解吗？能 → 构图失败。
```

## 二、即梦 prompt 反 CG / 反 AI 词汇黑名单

以下词汇会让即梦产出游戏概念图、CG 海报、AI 壁纸——而非实拍感画面。**直接拉黑，用右侧替代。**

### 禁用 → 替代

```
❌ "cinematic"               → 描述具体机位和焦距
❌ "dramatic lighting"        → 描述光源方向和色温
❌ "film grain"               → 换成拍摄介质（见第三节）
❌ "teal and orange"          → 描述画面内的实际颜色来源
❌ "shallow depth of field"   → 描述焦点位置和光圈感受
❌ "epic composition"         → 描述机位高度和人物与空间的比例
❌ "beautiful / poetic"       → 删掉。用可见事实替代。
❌ "emotional / mysterious"   → 删掉。情绪来自构图压力，不是形容词。
❌ "atmospheric / masterpiece"→ 删掉。
❌ "volumetric light"         → 描述光源：window light, desk lamp, overhead fluorescent
❌ "rich detail / highly detailed / intricate" → 删掉。限制每镜2-3个场景细节。
❌ "glowing" + "particles"    → 删掉。除非光源有物理来源。
❌ "neon" + "cyberpunk"       → 删掉。除非选题本身就是赛博朋克题材。
❌ "cinematic color grading"  → 描述画面内实物颜色来源（墙壁/衣服/天气/灯光）
❌ "HDR" / "sharp focus"      → soft highlight roll-off, moderate microcontrast
```

### 即梦 prompt 英文基础公式

每条 prompt 按此顺序写（紧凑段落，80-120词）：

```
1. 画幅 + 拍摄介质
2. 具体时间、地点、人物
3. 人物正在做什么（一个主要动作 + 未完成状态）
4. 机位（高度、焦距、距离、景别、构图压力类型）
5. 实际光源（类型+方向+色温，必须有物理来源）
6. 色彩命题（一主一次+一个点缀色，来源自画面内实物）
7. 真实材质 + 光学限制
8. 负向约束（简短）
```

### 英文 prompt 底版（每条 prompt 末尾粘贴）

```
live-action feature-film still, practical location, real actors, physically plausible set and props, soft highlight roll-off, medium-low microcontrast, natural skin texture, no CGI concept art, no game key art, no glossy AI rendering, no HDR, no plastic skin, no excessive particles, no fantasy poster composition, no teal-orange grading, no artificial rim light, no commercial beauty lighting, no television-drama blocking
```

## 三、拍摄介质选择

每期短视频选一种拍摄介质锁定画面质感。选定后全部镜头共用同一种。

| 介质 | 画面特征 | 适用选题 |
|------|---------|---------|
| **35mm 发行拷贝** | 柔和高光过渡、中低微反差、细腻不均匀颗粒 | 默认。硬核拆解/行业分析 |
| **16mm 电视转拷** | 低分辨率、较粗自然颗粒、轻微色渗、纪录片即时感 | 现场记录/工厂实拍/工地 |
| **MiniDV/早期数码** | 小传感器实用光线、有限动态范围、轻微边缘锐化 | 怀旧题材/个人记忆 |
| **监控/屏幕翻拍** | 几何畸变、屏幕纹理、眩光、黑位压缩 | 揭秘/调查/真相还原 |
| **长焦电影压缩** | 克制浅景深、人物被空间压缩 | 人物肖像/情感表达 |

```
□ 本期选定介质：______。全部镜头 prompt 统一标注。
□ 介质选定后不在同一期中切换。
```

## 四、色彩命题系统

每期内容必须有一句色彩命题。不是"用什么颜色好看"，而是**颜色如何改变观众对事件的判断**。

### 命题格式

```
"______（主色调）在 ______（环境）中 ______（动词）______（情绪/判断）"
```

### 命题示例

```
"褪色的朱红在潮湿的蓝灰中坚持不散" → 旧工业/怀旧/不屈
"午后的奶油黄被室内的病态绿一口口吞掉" → 时间流逝/健康衰退
"仪表盘的冷绿在混凝土灰中微弱但不停" → 技术/坚持/孤独
"琥珀台灯的光圈外，一切沉入不可辨认的黑" → 知识/局限/探索
```

### 每镜色彩规则

```
□ 每镜颜色限制：1个主色域 + 1个次色域 + 1个点缀色（占画面5-15%）
□ 每种颜色必须有画面内的物理来源：衣服/墙壁/天气/实物灯/植物/泥土/水/材质
□ 禁止凭空加色温滤镜。禁止"冷色调""暖色调"这类无来源描述。
□ 同一期所有镜头共享同一个色彩命题。每镜可以改变主/次色域的比例。
```

### 色彩连续性（穿镜变化方式）

- **稳定色域**：全镜同色 → 适合仪式/秩序/冷静分析
- **室内→室外过渡**：色域随空间切换 → 适合发现问题/走出舒适区
- **点缀色移动**：同一颜色在不同镜出现在不同实物上 → 适合追踪/线索
- **色域渐褪**：饱和度逐镜降低或色相偏移 → 适合情绪下沉/时间流逝

## 五、每镜构图自检（强制）

每个镜头输出前逐项检查：

```
□ 这一镜的构图压力类型是什么？从人物与空间的关系推导了吗？
□ 主要构图决策是什么？（只做一个。不是中轴对称+俯拍+门框+虚焦全堆上）
□ 前景遮挡有叙事功能吗？（观察者肩膀/门框/窗帘/车窗——不是随机虚化）
□ 视线流量写了吗？（进入→慢→落→出，四个节点）
□ 这一镜的景别/机位高度/构图机制跟前一镜不同吗？（至少4项变化）
□ 人物在做动作还是摆拍？（必须有一个进行中的、未完成的动作）
□ 画面中有未被解释的信息吗？（有 = 观众会想看下一镜）
□ 删掉这个镜头，故事还成立吗？（成立 = 废镜，删）
```

## 六、反模板扩展检查

在现有反模板基础上追加：

### 构图模板化检测

```
□ 这一期的开场镜构图跟上一期开场镜一样吗？（都是中景+桌面+台灯？）→ 换压力类型
□ 这一期的金句镜是不是又用了"正面特写+盯镜头"？→ 换。试侧面/背面/手部特写
□ 这一期的收尾镜是不是又用了"空房间+遗留物"？→ 禁止。末镜必须有人物在动或关系在变
□ 三个关键画面的景别是否至少有一次极端变化？（中景→特写→远景）
```

### 即梦 prompt 模板化检测

```
□ 是否每镜都用"warm amber desk lamp"？→ 换光源。试窗外光/荧光灯/仪表盘光/水面反射
□ 是否每镜都用"medium shot, eye level"？→ 换机位高度。试低角度/高角度/过肩
□ 是否每镜都用"looking at camera"？→ 换视线方向。试看纸/看窗外/看手/看远处的人
```

## 七、图文专用：视觉权力关系

A型图文的 P2（对比页）和 P6（收尾）适用。

### P2 对比页构图规则

```
□ "有判断的一方"视觉占优：大图/前景/亮区 → 你支持的选择
□ "被否定的"缩小/后置/暗区 → 你不推荐的选择
□ 不是中分对比。是视觉上的偏袒。
```

### P6 收尾页禁止

```
□ 禁止跟上一期 P6 用同一种构图骨架（如：全是居中大字+底部小字）
□ 禁止"金句浮在纯色底上"（除非文字刻在实物载体上）
□ 画面中必须有至少一个实物道具承载情绪（黄铜铭牌/手写便签/蓝图/旧书页）
```
