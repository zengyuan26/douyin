# 情绪线设计

> A型7页标准情绪弧 + 3种变体 + 选色规则。全A型强制。

---

## 标准情绪弧

7页的情绪不是平的。是一条有起有伏的曲线：

```
P1 好奇  →  "这里面有事"
P2 恍然  →  "原来是这样"
P3 专注  →  "有道理，接着看"      ← 情绪爬升
P4 乐观  →  "机会不小"            ← 情绪高点
P5 清醒  →  "但没那么简单"        ← 情绪回落，制造张力
P6 确信  →  "还是有机会的"        ← 回升，给出路
P7 判断  →  "我知道该怎么看了"    ← 收束
```

**核心机制**：P4拉高 → P5打下来 → P6托起来。一路乐观=你在吹。一路悲观=划走。起伏才有说服力。

---

## 三种变体

| 变体 | 适用选题 | P4-P5-P6 关系 | 示例选题 |
|------|---------|--------------|---------|
| **标准弧** | 机会分析、政策解读 | 乐观→清醒→确信 | 新能源车下乡、虚拟电厂 |
| **警示弧** | 风险警示、利空分析 | 焦虑→确认→出路 | 分时电价取消、工商业光伏暴跌 |
| **反问弧** | 争议观点、反常识判断 | 意外→理解→认同 | 强制配储取消反而更赚钱 |
| **共鸣弧** | 个人故事、行业观察 | 共鸣→理解→温暖 | 入行第一月踩坑、工厂老板问住了我 |

## 每页情绪 → 视觉映射

| 情绪 | 卡片强调色 | 视觉密度 | 规则 |
|------|-----------|---------|------|
| 好奇 | 不设答案，只有问号 | 极疏 | 封面只放标题+副标题 |
| 恍然 | 琥珀 [#C89050] | 疏 | 一个核心信息，大留白 |
| 专注 | 琥珀 | 中 | 允许对比面板（左右双卡） |
| 乐观 | 绿 [#5A8A6A] | 中 | 绿色左沿强调，3张卡 |
| 焦虑/清醒 | 红 [#cc5555] / 琥珀 | 中密 | 红色左沿强调，卡更紧凑 |
| 确信 | 绿 | 中 | 绿色回归，3张卡 |
| 判断 | 琥珀 | 疏 | 时间线+行动项，大留白 |

**铁律**：乐观页用绿色、警示页用红色、结论页用琥珀。情绪和颜色不能打架。

## 情绪转折规则

```
P3→P4 转折：为什么→所以机会在哪。自然爬升。不需要过渡。
P4→P5 转折：机会看完→但坎也在。必须用一句话过渡，承认前面"是真的"，
         再翻面。"信号很亮。但坎也在。"（6字过渡）
P5→P6 转折：坎看完→还是有机会。必须给出路，不能停在焦虑。
         "信号是真的，坎也是真的。机会在缝隙里。"（一句话翻转）
P6→P7 转折：机会看完→回到当下。收束，不延伸新信息。
```

P4→P5 是最关键的转折点。过渡太少=翻车太快，读者没准备好。过渡太多=泄了悬念。

---

## 自检

```
□ 这条情绪弧是四选一（标准/警示/反问/共鸣）？
□ P4→P5 有过渡句吗？（不超过12字）
□ P5→P6 给出路了吗？（不能停在焦虑）
□ 颜色跟情绪匹配了吗？（绿=乐观，红=警示，琥珀=结论）
□ P5 有没有制造真正的张力？（不是"有点难但还好"的不痛不痒）
```
