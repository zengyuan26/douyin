# 锚点法则

> 封面公式。全类型强制。没有锚点不发。

---

## 五类有效锚点

| 类型 | 示例 | 适用场景 |
|------|------|---------|
| 热搜 | 百度热搜第一的话题 | 蹭热点扩圈 |
| 政策 | 五部门联合发文 | 政策解读 |
| 大公司 | 滴滴入局、特斯拉联盟 | 行业信号 |
| 大数字 | -61.5%、23.36亿 | 数据冲击 |
| 反直觉 | 暴跌vs暴涨、表面卖车实质不是 | 认知翻转 |

## 两类无效锚点（禁止当封面主角）

- 行业术语（"新型电力系统"）
- 纯科普标题（"虚拟电厂是什么"）

---

## 封面公式

```
事件锚 + 数字锚 + 悬念锚（三选二即可，三全最佳）

示例：
事件锚：五部门联合发文 6.18
数字锚：155款车 + 23.36亿已下拨
悬念锚：卖车？还是。
```

---

## 数字锚分级

| 级别 | 放哪 | 示例 |
|------|------|------|
| 绝对大数字 | 封面 | -61.5%、23.36亿、+89% |
| 行业单价/比例 | 内页P2以后 | 0.3-0.5元/Wh、消纳率80%→50% |

---

## 发布前决策树

```
封面有事件锚吗？ → 没有 → 不加不发
封面有数字锚吗？ → 没有 → 不加不发
封面有悬念吗？   → 没有 → 改成问句或"实际是？"
三个全没有？     → 不发。重做封面。
```
