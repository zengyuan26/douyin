# 情绪锚点

> 一条视频的"魂"。观众看完心里留下的那一个感觉。
> 选题定方向，人设定语气，情绪锚点定重心。所有内容——开场钩子、中间金句、结尾收束——都服务于这一个锚点。

---

## 一、定义

**情绪锚点** = 观众看完这条视频，心里留下的那一个感觉。

它不是选题、不是人设、不是情绪曲线。它是这些上面的那层。先定情绪锚点，再定情绪曲线——曲线服务于锚点，不是反过来。

---

## 二、来时路（职业线）

| 情绪锚点 | 观众心里那句话 | 适用场景 |
|------|------|------|
| 破除恐惧 | "原来转行不是疯了，是算过的" | 转行决定、从零开始 |
| 被理解的踏实 | "他也踩过这个坑，我不是一个人" | 踩坑经历、低谷故事 |
| 我也能 | "他能做到，我说不定也行" | 翻身过程、学习成果 |
| 这人真实 | "不是装的，他说的是真的" | 失败经历、脆弱时刻 |
| 敬佩 | "这人真有韧性，摔不碎" | 14年起起落落，四次从零开始 |
| 被看见 | "他说的就是我。我也40岁，我也迷茫" | 同龄转行者共鸣 |
| 释然 | "原来不需要完美准备好了再开始" | 从零学投资、边做边学 |
| 期待 | "我想看他下一步怎么走" | 阶段性复盘、未来规划 |

## 三、项目路（商业理念/直接解决）

| 情绪锚点 | 观众心里那句话 | 适用场景 |
|------|------|------|
| 刷新认知 | "原来是这样看的，我之前的理解不对" | 投行视角、行业判断 |
| 实用惊喜 | "这个真能帮我省钱" | 算账、避坑 |
| 紧迫感 | "别人都在动了，我不能掉队" | 基金暗战、行业趋势 |
| 后怕 | "还好知道了，不然亏大了" | 电费单上的坑、合同陷阱 |
| 优越感 | "我懂了，别人还不懂" | 政策解读、行业信号 |
| 被尊重 | "他没把我当傻子，不绕弯子" | 直接算账、实话实说 |
| 心安 | "有人帮我想清楚了，不用自己琢磨" | 框架拆解、概念图谱 |
| 兴奋 | "这事儿有搞头" | 新机会、新模式 |

## 四、项目路（育儿/家庭）

| 情绪锚点 | 观众心里那句话 | 适用场景 |
|------|------|------|
| 会心一笑 | "这人跟我一样，也是当爹的" | 带娃日常、教育困惑 |
| 被理解 | "他说出了我说不出来的焦虑" | 孩子教育、工厂接班 |
| 温暖 | "这人不错，想跟他喝一杯" | 陪母亲、带孩子、在家办公 |

---

## 五、使用规则

1. **先定锚点，再动笔**。导演分析第一步就是选情绪锚点，选完才能往下走
2. **一条视频一个锚点**。不贪多。如果有两个，说明选题还没聚焦
3. **开场第一秒到结尾最后一秒，都服务于这个锚点**。中间跑偏了就想：这个镜头/这句话服务于锚点吗？
4. **锚点来自内容线**。来时路和项目路的锚点池不同，别串
5. **自检**：生成完问自己——观众看完心里留下的感觉，是你选的那个吗？

---

## 六、常见错误

| 错误 | 为什么不对 |
|------|------|
| 选了两个锚点 | 观众看完什么都记不住 |
| 锚点和内容线不匹配 | 来时路选了"紧迫感"，观众会觉得你在推销 |
| 锚点和开场钩子矛盾 | 锚点是"破除恐惧"，开场却在制造焦虑 |
| 锚点选了但中间跑了 | 前半段温暖，后半段变成说教——观众困惑 |
| 把选题当锚点 | "微电网怎么省电"是选题不是锚点。锚点是观众的感受——"实用惊喜" |
