# 口播方法论

> 口播写作五大原则 + 开头设计 + 超出预期节奏。全类型强制。

---

## 五大原则

### 1 呼吸感
- 每句≤25字
- 句号断句，制造停顿
- 自问自答制造节奏

### 2 问题链替代目录
- 禁止"好往下走""扣1扣2"
- 每段末尾抛一个问题，下一段回答
- 3-4分钟最多讲2-3个概念

### 3 口语化
- 带情绪，不是只说人话
- 情绪词库：竟然、没想到、但是、说白了、更关键的是
- 删除：在当今、随着、众所周知、因此、然而

### 4 深入浅出
- 三层翻译：概念 → 场景 → 结论
- 每个专业词 3 秒内必须跟生活场景

### 5 密度控制
- 3分钟 ≤ 2个概念
- 5分钟 ≤ 3个概念
- 讲一个转折点，不讲五个概念

---

## 开头设计（强制）

开头决定前 3 秒。详见 [开场方法论](开场方法论.md)。

### 三个万能公式

| 公式 | 模板 | 适合选题 |
|------|------|---------|
| **反常识断言式** | "你以为X。但Y。" | 硬核拆解/观点输出 |
| **身份标签式** | "如果你也在[做某件事]——" | 故事分享/个人经历 |
| **静默中断式** | (停顿) + 一句话推翻常识 | 有信任基础后的进阶 |

### 开头铁律

- 3 秒内必须完成：粉碎默认认知 / 让观众觉得"这说的就是我" / 给认知缺口（三选一）
- 禁止："大家好""今天我们来聊""你知道吗"
- 禁止：开头承诺的东西后面没兑现（标题党伤信任）

## 超出预期节奏模型

整条视频的预期管理节奏：

```
0-3秒     打破预期（反常识钩子）        → 停下来
3-15秒    建立信任（为什么我有资格说）   → 留下来
15-45秒   层层剥茧（2-3个信息模块）     → 每15秒切换节奏
45-55秒   Wow Moment（核心反转）        → 超出预期的主菜
最后5秒    收束 + 判断 + 互动钩子        → 获得感确认
```

### Wow Moment 设计

每条口播必须有一个"天灵盖时刻"，在 75-85% 位置。四种类型：
- **认知反转**："扩产=看好？正相反。"
- **框架赠予**："三个领先指标——下次不看新闻看这三个数。"
- **概念折叠**："2018光伏=2026锂电，同一个剧本。"
- **诚实时刻**："这个判断有一个前提。"

## 脚本规范

- 时长：3-4分钟
- 叙事视角：分析师（"我注意到..."）
- 开头：三个万能公式选一（反常识断言/身份标签/静默中断）
- 结尾：CTA温和，不推销。用开放式问题引导评论

## 常见反模式

- 罗列现象开场（最弱）→ 换成反常识断言或身份标签
- 自我介绍开场 → 直接讲事
- 连续三句陈述句 → 插入问句或感叹
- 开头抛完所有信息 → 留一半，制造信息缺口
- 全篇匀速无起伏 → 15 秒节奏切换 + Wow Moment 集中释放
