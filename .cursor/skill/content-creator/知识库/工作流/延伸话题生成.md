# 延伸话题生成

> 每篇内容生成后，自动推荐下一篇。4种延伸方向 + 决策树。

---

## 四种延伸方向

| 方向 | 跟本篇关系 | 什么时候推 | 示例 |
|------|-----------|-----------|------|
| **深度延伸** | 把某一页展开成独立一篇 | 某页数据/概念评论反响大 | P4 "绿证涨600%" → 单独做绿证全拆解 |
| **对立延伸** | 反过来讲，同一个主题的反面 | 本篇拆利好 → 下篇拆风险 | 新能源车下乡机会 → 为什么乡村项目落地这么慢 |
| **跨圈延伸** | 换目标人群，同一件事不同账本 | 本篇资方视角 → 下篇工厂老板 | 资方看虚拟电厂 → 工厂老板看虚拟电厂：跟你有什么关系 |
| **系列延伸** | 同一个问题链的下一问，串成系列 | 问题链自然延伸，下一篇是逻辑后续 | 车下乡 → 乡村光储充怎么落地 → 落地后运营账怎么算 |

---

## 决策树

```
生成完内容后：

1. 本篇有没有某一页特别"爆"的数据/概念？
   → 有 → 延伸方向：深度延伸。推荐标题："[那页的核心数据]，一篇拆透"
   → 没有 → 往下

2. 本篇有没有明显没讲的反面？
   → 有，且反面对目标人群有价值 → 延伸方向：对立延伸
   → 没有 → 往下

3. 同一件事，换一个人群会不会有完全不同的关心点？
   → 会 → 延伸方向：跨圈延伸
   → 不会 → 往下

4. 本篇是不是一个问题链的起点？
   → 是，且下一篇逻辑自然承接 → 延伸方向：系列延伸
   → 不是 → 回到选题库，不硬推
```

## 提示模板

生成完内容后，根据决策树输出延伸建议：

```
深度延伸：
  "这篇 P[X] 提到的 [某数据/概念]，可以展开一篇独立的深度拆解。做吗？"

对立延伸：
  "这篇讲了 [利好/机会]。反过来——[风险/困难] 值得单独拆一篇。做吗？"

跨圈延伸：
  "这篇是 [人群A] 视角。同一件事，[人群B] 的账完全不一样。做一篇 [人群B] 版的？"

系列延伸：
  "这一篇是 [当前问题]。逻辑上下一步是 [下一个问题]。串成系列？"
```

---

## 什么时候不推

```
□ 本篇已经是完整闭环（P7没有预告下一问）→ 不硬推
□ 延伸方向跟已有内容重复 → 不推，标注"已覆盖"
□ 延伸方向太冷（目标人群不关心）→ 不推
□ 用户刚生成完，明显要歇 → 只标一句，不追问
```

## 跟系列追踪模板的关系

如果用户接了系列延伸建议 → 从 [系列追踪模板](系列追踪模板.md) 新建系列进度文件。
如果用户接了深度/对立/跨圈 → 单篇独立，不纳入系列。
