# 内容类型路由

> 选题→判断类型→加载对应规则。生成前必看。

---

## 决策树

```
选题来了 →
  是新闻事件/政策解读？ →
    有可交易标的（股票/赛道/概念）？ → 投资向 → 黄金六页图文（已验证：钠电、比亚迪）
    无可交易标的？ → 认知向 → ⚠️ 无成熟模版，谨慎投入
  是知识框架/产业链拆解？ → B型 1张信息图
  需要深度讲解+个人观点？ → 口播脚本（可配PPT）
  以上都不是 → 回到选题池，不硬做
```

### 投资向 vs 认知向 判断标准

| | 投资向 | 认知向 |
|------|------|------|
| 有可交易标的？ | 有（股票/赛道/概念股） | 无 |
| 观众看完能行动？ | 买/关注/盯 | 知道了 |
| 评论区驱动力 | 站队吵对错 | 无自然争议 |
| 已验证模版 | 黄金六页 ✓ | **空白** |
| 成功案例 | 钠电CVC、比亚迪储能 | — |
| 失败案例 | — | 国常会运营商篇 |

**铁律：投资向模版不能直接套给认知向。** 黄金六页需要利益锚驱动。认知向需要不同的驱动力（收藏价值/社交货币/故事共鸣）。

## 类型联动

```
A型事件分析 →
  "这篇适合展开一个B型深度拆解吗？"
  "需要配一段口播吗？"
  "建议下篇选题：___"

B型知识图谱 →
  "可以配一段2分钟口播。做吗？"
  "已经有同主题的A型吗？要不要串成系列？"

口播脚本 →
  "需要配套PPT/图文吗？"
  "可以做成图文系列吗？"
```

## 加载规则

```
确定类型 →
  加载 规则库/合规规则.md + 格式规则.md + 设计规则.md
  加载 方法论库/锚点法则.md + 问题链设计.md
  加载 工作流/发布前自检清单.md
  如果是系列 → 加载 工作流/系列追踪模板.md
```
