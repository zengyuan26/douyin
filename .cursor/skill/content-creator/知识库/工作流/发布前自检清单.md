# 发布前自检清单

> 内容生成完毕后，逐项检查。全类型强制。

---

## 一、合规检查

```
□ 公司全名 ≤5 次？
□ 无暴增/暴涨/翻倍/满产/不够用/做不过来/下注？
□ 无收藏/关注/点赞/转发/点击/扫码？
□ 无公司名+财务数据紧挨的句子？
□ 每页有免责声明？
□ P1 封面无「所以呢」结论行？
□ 配文和首评也过了以上检查？
```

## 二、内容质量五维检测

**维度1 · 钩子强度**
```
□ 封面有没有悬念？（不是陈述句）
□ 前3秒有没有数字/具体的人/反直觉判断？
□ 标题删掉前三个字，意思变了吗？（变了 = 前三个字是废话）
```

**维度2 · 问题链完整**
```
□ 7张图能不能串成一条线？
□ 有没有哪页是孤立的知识点？
□ P7 是不是给了判断，不是重复？
```

**维度3 · 口语化（口播专用）**
```
□ 有没有超过25字的句子？
□ 有没有"在当今""随着...的发展""众所周知"？
□ 有没有"因此""然而""我们可以看出"？
□ 念出来像不像人在说话？
```

**维度4 · 数字冲击**
```
□ 封面数字是绝对大数字吗？
□ 每个数字有对比参照物吗？
□ 每张数据图底部有"所以呢"结论行吗？
```

**维度5 · 标签态度**
```
□ 标签是判断不是中性分类？
□ 颜色有情绪映射？（红=警示/绿=机会/琥珀=结论）
```

## 三、可拿走性检查（翻车案例库 5 支血的教训）

```
□ 观众看完能拿走什么？用一句话说出来
□ 答案是"了解了X"？ → 重写。了解 = 没拿到
□ 答案是"能判断/能算出/能避开"？ → 合格
□ 结尾最后一句是问号吗？ → 不是 → 大概率没做互动设计
```

## 四、趣味性检测（口播/视频专用）

> 抖音算法判"趣味性差" = 观众留存率低。内容再对，无聊就划走了。

**维度6 · 全篇趣味性**
```
□ P1 前3秒：第一句有没有具体的人/数字/画面？（不是"大家好""这一期我们来聊"）
□ P2-P4：有没有连续2页以上纯"教"？（有 → 插入一个乐点：问观众、给画面、放数字炸弹）
□ 每页都有"你"吗？至少1次让观众对号入座（"你是哪种？""你自己在哪？"）
□ 概念解释用画面还是用定义？（"风在西北吹，电在东部用" > "资源分布不均"）
□ 情绪有起伏吗？全篇匀速 → 找一个页做紧张（涨价/关门），下一页做释压（出路/方法）
□ 句子长度有变化吗？有没有连续5句一样长的？（有 → 拆一句成三个字，或并两句成一句）
□ 有没有让观众意外的地方？（反直觉数字/没想到的角度/一个"说白了就是糊弄"的直白判断）
□ 结尾有没有给观众一个评论的理由？（问开放式问题，不是"关注我下期见"）
```

**口播念稿测试**
```
□ 念一遍。有没有地方念着喘不过气？→ 拆句
□ 念一遍。有没有连续10秒同一个语调？→ 在该处插入短句/问句/停顿打破节奏
□ 念一遍。观众会在哪笑/愣/发弹幕？ → 不知道 → 没情绪，加情绪锚点
```

## 五、AI 味检测

```
□ 有没有"在当今""随着...的发展""众所周知"？ → 删
□ 有没有"首先...其次...最后..."？ → 换口语转折
□ 有没有"建议""推荐""值得关注"？ → 删
□ 有没有连续三句都是陈述句？ → 插入一个问句或感叹
□ 念一遍，像不像人在说话？ → 不像 → 加语气词、停顿
```
