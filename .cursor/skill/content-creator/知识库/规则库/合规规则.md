# 合规规则

> 抖音广告检测规避。图文/口播全类型强制。生成内容后必须逐项检查。

---

## 一、公司/品牌名密度控制

- 公司全名单篇 ≤5 次
- 首次用全名，后续用"该公司""这家头部公司"
- 股票代码绝对禁止

## 二、收益/财务敏感词替代表

| 禁用词 | 替换为 |
|--------|--------|
| 暴增、暴涨、飙升 | 大幅预增、增长、同比+xx% |
| 赚、赚钱、赚翻了 | 增长、利润、出货、需求拉动 |
| 翻倍、翻三倍 | 同比+xx%、+200% |
| 满产满销 | 产能紧张、供不应求 |
| 不够用了、抢不到 | 还在增长、持续增长 |
| 做不过来 | 供不应求 |
| 暴利、躺赚、稳赚 | 禁止使用，无法替换则删除整句 |

## 三、CTA 禁止项

- 禁用：收藏、关注、点赞、转发、点击、扫码
- 禁用：任何行动指令（包括"翻出来对一遍"）
- 允许：无 CTA。末尾只放"行业观察笔记 | 不构成任何建议"

## 四、公司名+财务数据隔离

```
禁止：亿纬锂能净利+110%，产能紧张     （公司+数据紧挨）
允许：头部电池厂净利+110%              （去公司名）
允许：亿纬锂能发布半年报。行业数据显示出货增长68%（公司+行业数据分离）
```

规则：同一句话里，公司名和财务数据不能同时出现。

## 五、全类型敏感词库

```
投资、研判、推荐、买入、卖出、目标价、估值、股价、
暴增、暴涨、飙升、翻倍、产能紧张、赚、
收藏、关注、点赞、转发、点击下方、
不够用了、抢、最后机会、马上、独家、唯一、第一、
下注、赌
```

## 六、发布前自检

```
□ 公司全名 ≤5 次？
□ 无暴增/暴涨/翻倍/满产/不够用/做不过来/下注？
□ 无收藏/关注/点赞/转发/点击/扫码？
□ 无公司名+财务数据紧挨的句子？
□ 每页有免责声明"行业观察笔记 | 不构成任何建议"？
□ P1 封面无「所以呢」结论行（封面只制造悬念）？
□ 配文也过了以上所有检查？
□ 首评无诱导词？
```
