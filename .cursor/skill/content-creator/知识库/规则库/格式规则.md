# 格式规则

> A/B型图文强制规格。生成前确定类型，按对应模板执行。

---

## A型（分析类图文）

| 项目 | 规格 |
|------|------|
| 图片数 | 7张（可扩展至8张） |
| 比例 | 9:16 竖版 1080x1920px |
| 文字量 | 每页≤60字，总计≤420字 |
| 核心驱动 | 转发 + 评论 |
| 创作者角色 | 分析师 |
| 结构 | P1悬念→P2本质→P3-P6拆解→P7判断 |
| 适合选题 | 事件分析、政策解读、观点输出 |

**必含模块**：基本信息、H-V-F标题设计、7页速查表+问题链条、评论区运营、制作规范、发布策略、合规检查

---

## B型（知识图谱图文）

| 项目 | 规格 |
|------|------|
| 图片数 | 1张 |
| 比例 | 3:4 竖版 1080x1440px |
| 文字量 | 50-150字总 |
| 核心驱动 | 收藏 |
| 创作者角色 | 整理者（"整理自留"） |
| 结构 | 摩尔7模块：身份牌→反直觉→拆骨架→本质句→爆发因→个人感→工具CTA |
| 适合选题 | 产业链拆解、知识框架、流程图、清单 |

---

## 口播脚本

| 项目 | 规格 |
|------|------|
| 时长 | 3-4分钟 |
| 句式 | 每句≤25字，句号断句 |
| 结构 | 黄金3秒→问题链→深入浅出→收尾 |
| 情绪词库 | 竟然、没想到、但是、更关键的是、说白了 |

---

## 共同规范

- 抖音安全区：顶部12%空白 + 底部12%免责
- 翻页钩子位置：~84%处
- 金句块：琥珀半透明底，圆角8px
- 禁止：手机mockup、emoji、硬阴影、人物照片、段落文字
