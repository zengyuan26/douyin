# 设计规则

> GPT Image2 生图规范。封面悬念法则 + 视觉铁律。

---

## 封面悬念法则（全类型强制）

封面 = 诱饵，不是摘要。

**封面可以放什么（不超过3样）**：
- 主标题：悬念/问句（如"卖车？还是。"）
- 副标题：机会暗示或数字诱饵
- 系列标签（如有，右上角小标）

**封面绝对不能放**：
- 金句（带背景色块的结论文字）
- 「所以呢」结论行
- 数据来源/文件编号/日期
- 核心数据解释
- 行动引导/CTA

**自检**：看完封面，观众脑子里应该是一个问号，不是一个句号。

## 三套视觉风格

> 内容温度决定风格。选题确定后自动匹配，不每次讨论。

---

### A · 财新周刊 dark（默认主力）

事件分析、政策解读、行业判断。权威感、有判断、封面像重磅报道。
参考：财新周刊 / Bloomberg Businessweek

```
底色：深炭黑 [#1a1a1e]
标题：暖白 [#f0ead6]，Bold
正文：暖灰 [#cccccc]，Regular
强调色：琥珀 [#C89050] / 警示红 [#cc5555] / 绿 [#5A8A6A] / 青 [#4D8F8F]
金句块：琥珀半透明底 [#C89050 12-18%]，圆角8px
分隔线：1px [#444444]
光影：琥珀光束斜切暗色背景，戏剧感。不是平的。
```

### B · 白皮书 light

知识框架、产业链拆解、流程图。干净理性、数据感。
参考：麦肯锡单页报告 / 36氪信息图

```
底色：暖白 [#f5f5f7]
标题：深蓝黑 [#2c3e50]，Bold
正文：深灰 [#555555]，Regular
强调色：琥珀 [#C89050] / 青绿 [#4D8F8F]
金句块：浅灰底 [#f0f0f0]，圆角6px
分隔线：1px [#dddddd]
光影：柔和自然光，无戏剧光。干净利落。
```

### C · 深夜对话 dark

合伙人教的、个人故事、真实进展。娓娓道来、有温度。
参考：晚点LatePost / 36氪深度

```
底色：更深黑 [#0d0f0f]
标题：暖白 [#f0ead6]，Bold
正文：暖灰 [#bbbbbb]，Regular
强调色：琥珀 [#C89050] / 青绿 [#4D8F8F]（双色）
金句块：青绿半透明底 [#4D8F8F 12%]，圆角8px
分隔线：1px [#2a2a2a]
光影：极暗调+单束暖光，像深夜书桌前。私密感。
```

### 自动匹配规则

| 内容线 | 内容类型 | 匹配风格 |
|--------|---------|---------|
| P线（政策翻译）| A型图文 | **A · 财新周刊 dark** |
| C线（互联网视角）| A型图文 / 口播 | **A** 或 **C**（个人观点→C） |
| T线（合伙人教的）| A型图文 / 口播 | **C · 深夜对话** |
| W线（我们在做什么）| A型图文 / 口播 | **C · 深夜对话** |
| R线（真实进展）| A型图文 / 口播 | **C · 深夜对话** |
| K线（概念图谱）| B型图文 | **B · 白皮书 light** |
| 其他 B型图文 | B型图文 | **B · 白皮书 light** |

**决策**：硬核分析→A，知识整理→B，个人表达→C。模糊时默认 A。

## GPT Image2 提示词原则

- 描述画面，不是罗列元素
- 给风格参考，不是给线框图
- 有材质、有光影、有视觉隐喻
- 每页底部有"所以呢"结论行（≤20字）
- 翻页钩子在~84%位置

## 禁止项

纯色块背景、段落文字、表格、照片、emoji、硬阴影、人物照片
