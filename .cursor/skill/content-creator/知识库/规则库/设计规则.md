# 设计规则

> GPT Image2 生图规范。封面悬念法则 + 视觉铁律。

---

## 封面悬念法则（全类型强制）

封面 = 诱饵，不是摘要。**必须同时做到两件事：让观众觉得"有蹊跷" + "这人有判断"。**

### 为什么反常识开头有效——五个心理学原理

用户决定继续看一个内容，最多只花 **2 秒**。反常识开头在这 2 秒内制造认知缺口，让大脑主动选择继续。

| 原理 | 机制 | 封面应用 |
|------|------|---------|
| **认知失调** | 两个相反信息同时进入大脑 → 产生不适 → 只有继续看才能消除 | 粉碎默认假设："200万吨扩产。不是看好信号。" |
| **好奇心缺口** | 大脑对不完整信息有生理性紧张，必须补全 | 给反直觉结论但不给解释 → 翻页 |
| **身份赋予** | "这个人在告诉我不知道的东西"→ 信息差关系瞬间建立 | 判断句暗示"我有你不具备的视角" |
| **蔡格尼克效应** | 人对未完成的事情记得更牢 | 反常识结论 = 未闭合的认知环 → 必须翻到解释页 |
| **2秒决策窗口** | 陈述句 = 已接收 → 划走。反常识 = 被挑战 → 翻页 | 不在封面上"讲完"，在封面上"撬开" |

### 封面三要素（新规则）

```
旧：锚点 + 判断
新：粉碎默认假设 + 锚点 + 判断方向
```

**粉碎默认假设**是可选但强烈推荐的——直接告诉观众他之前的认知是错的。这是认知张力最强的封面开头。

### 五种封面模式

| 模式 | 公式 | 心理学原理 | 示例 |
|------|------|-----------|------|
| **反常识结论式** | 你以为的是错的 + 给方向 | 认知失调 | "200万吨扩产。不是看好信号。" |
| **身份标签式** | 你是不是也以为... | 社会身份理论 | "如果你也觉得锂电过剩了——" |
| **数字对撞式** | 两个矛盾数字并置 + 反问 | 好奇心缺口 | "200万吨扩产 vs 34万吨叫停" |
| **概念折叠式** | 这是 X，不是 Y | 认知折叠 | "不是过剩。是上游配额重新分配。" |
| **ROMO式** | 你不需要焦虑 + 看这个就够了 | 反FOMO | "你不需要每天刷新闻。看三个数。" |

**模式选择规则**：
- 硬核分析/行业拆解 → 优先反常识结论式或数字对撞式
- 故事分享/个人经历 → 优先身份标签式
- 避坑/风险警示 → 优先概念折叠式
- 入门教程/科普 → 优先 ROMO 式

### 封面可以放什么（不超过3样）

- 主标题：悬念/问句/数字对撞/反常识结论
- 判断句：一句 AI 写不出来的立场（L4 有来处的判断）
- 系列标签（如有，右上角小标）

### 封面绝对不能放

- 纯悬念问句后面不跟判断（标题党感 → 划走）
- 金句块 + 副标题两个同时出现（重复信息，浪费封面位置）
- 「所以呢」结论行
- 数据来源/文件编号/日期
- 核心数据解释
- 行动引导/CTA

### 判断句规则

- 必须是 AI 写不出来的立场（L4 有来处的判断），不是"拆开看""深度分析"这种废话
- 给方向不给解释 → "为什么你这么判断？"→ 翻页
- 自检：删掉判断句，封面还成立吗？不成立 → 判断句不是可有可无的，是对主标题的"投名状"

### 自检

看完封面，观众脑子里应该是一句"嗯？什么意思？"和一句"这人有点判断"。

### 2026 趋势提醒

- **真实感 > 精致感**：判断力比设计感重要。粗糙但有判断 > 精致但像 AI 写的
- **反套路 > 套路**："家人们""千万别划走"全面失效。坦诚 = 新钩子
- **长期信任 > 单篇爆款**：观众关注不是因为一篇爆了，是连刷三篇都拿到了东西

## 三套视觉风格

> 内容温度决定风格。选题确定后自动匹配，不每次讨论。

---

### A · 财新周刊 dark（默认主力）

事件分析、政策解读、行业判断。权威感、有判断、封面像重磅报道。
参考：财新周刊 / Bloomberg Businessweek

```
底色：深炭黑 [#1a1a1e]
标题：暖白 [#f0ead6]，Bold
正文：暖灰 [#cccccc]，Regular
强调色：琥珀 [#C89050] / 警示红 [#cc5555] / 绿 [#5A8A6A] / 青 [#4D8F8F]
金句块：琥珀半透明底 [#C89050 12-18%]，圆角8px
分隔线：1px [#444444]
光影：琥珀光束斜切暗色背景，戏剧感。不是平的。
```

### B · 白皮书 light

知识框架、产业链拆解、流程图。干净理性、数据感。
参考：麦肯锡单页报告 / 36氪信息图

```
底色：暖白 [#f5f5f7]
标题：深蓝黑 [#2c3e50]，Bold
正文：深灰 [#555555]，Regular
强调色：琥珀 [#C89050] / 青绿 [#4D8F8F]
金句块：浅灰底 [#f0f0f0]，圆角6px
分隔线：1px [#dddddd]
光影：柔和自然光，无戏剧光。干净利落。
```

### C · 深夜对话 dark

合伙人教的、个人故事、真实进展。娓娓道来、有温度。
参考：晚点LatePost / 36氪深度

```
底色：更深黑 [#0d0f0f]
标题：暖白 [#f0ead6]，Bold
正文：暖灰 [#bbbbbb]，Regular
强调色：琥珀 [#C89050] / 青绿 [#4D8F8F]（双色）
金句块：青绿半透明底 [#4D8F8F 12%]，圆角8px
分隔线：1px [#2a2a2a]
光影：极暗调+单束暖光，像深夜书桌前。私密感。
```

### 自动匹配规则

| 内容线 | 内容类型 | 匹配风格 |
|--------|---------|---------|
| P线（政策翻译）| A型图文 | **A · 财新周刊 dark** |
| C线（互联网视角）| A型图文 / 口播 | **A** 或 **C**（个人观点→C） |
| T线（合伙人教的）| A型图文 / 口播 | **C · 深夜对话** |
| W线（我们在做什么）| A型图文 / 口播 | **C · 深夜对话** |
| R线（真实进展）| A型图文 / 口播 | **C · 深夜对话** |
| K线（概念图谱）| B型图文 | **B · 白皮书 light** |
| 其他 B型图文 | B型图文 | **B · 白皮书 light** |

**决策**：硬核分析→A，知识整理→B，个人表达→C。模糊时默认 A。

## GPT Image2 提示词原则

- 描述画面，不是罗列元素
- 给风格参考，不是给线框图
- 有材质、有光影、有视觉隐喻
- 每页底部有"所以呢"结论行（≤20字）
- 翻页钩子在~84%位置

## 禁止项

纯色块背景、段落文字、表格、照片、emoji、硬阴影、人物照片
