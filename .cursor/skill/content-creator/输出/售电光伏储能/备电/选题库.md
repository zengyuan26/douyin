> 定位：互联网产品经理 × 创业多次 × AI工具 × 投资学徒。跨界视角，学着讲着。
> 更新：2026-07-08 | 战略升级：从固定选题列表→生成式选题框架。内容=学到→连接→一个道理

## 一、两条内容线（重新定义）

| 线 | 不是 | 是 |
|------|------|------|
| **项目路** | 微电网进展汇报 | 我在学投资，发现了什么。翻译成对你有用的 |
| **来时路** | 我的个人故事 | 我过去的经历，让我今天怎么看问题。框架是怎么长出来的 |

**两条线的关系**：同一件事的两个面。学投资（项目路）产生新认知，过去的经历（来时路）解释为什么你这样看。

## 二、五种生成模式

选题不是"想一个话题"。选题是：**今天学到/看到/想到了什么 → 能连接什么经历 → 能讲出什么道理**。

| 模式 | 公式 | 触发条件 | 举例 |
|------|------|------|------|
| **学到+连接** | 今天学的概念 + 过去的经历 + 一个道理 | 每天学投资/商业，学到任何一个新概念 | 学"定价权"→ 电商被压价经历 → 你的工厂为什么不敢涨价 |
| **信号+拆解** | 行业信号 + 产品经理框架 + 判断 | 政策变化、行业新闻、市场动向 | 分时电价取消 → 用户痛点在哪 → 三类工厂受影响最大 |
| **踩坑+方法** | 过去的坑 + 现在怎么看 + 可用的方法 | 做项目时触发回忆，或复盘时发现规律 | 创业库存压死 → 投项目第一看现金流 → 工厂老板怎么看库存 |
| **AI+实操** | 用了AI做什么 + 过程分享 + 观众看到可能 | 每次用AI做工具、做内容、提效率 | 用AI拆电费单 → 3天做完 → AI还能这么用 |
| **进展+复盘** | 这一步 + 学到了什么 + 下次怎么做 | 项目每推进一步 | 巴南谈下来了 → 最大的认知：XXX |

**使用规则**：
- 每天问自己：我今天学到了什么？它让我想起什么？这个连接能讲出一个什么道理？
- 有连接 → 就是选题。没有连接 → 不硬做
- 模式1和模式2是主力（80%），模式3/4/5是补充（20%）

## 三、选题池（按模式组织）

### 学到+连接（项目路为主）

学投资/商业概念 → 连接过去经历 → 讲道理

| 编号 | 学到的概念 | 连接的经历 | 讲的道理 |
|------|------|------|------|
| X-01 | 定价权 | 做电商被渠道压价 | 你的工厂为什么不敢涨价——没有定价权 |
| X-02 | 护城河/壁垒 | 电商第一→合伙人被抓→什么都没留下 | 为什么有的厂利润是你的3倍——壁垒不是规模 |
| X-03 | 资本回报率 | 创业时算不过账，现金流断了 | 投一条新产线几年回本——大部分人算错了 |
| X-04 | 交易结构设计 | 腾讯项目合同被做手脚 | 签合同不是签价格，是签结构 |
| X-05 | 规模效应 vs 网络效应 | 互联网平台和工厂的区别 | 你的工厂做大了，成本为什么没降 |
| X-06 | 现金流 vs 利润 | 倒卖口罩——账面赚了，现金没了 | 工厂老板最容易犯的错：把利润当现金 |
| X-07 | 信息不对称 | 能源行业的内行和外行 | 你知道的和不知道的——中间差就是钱 |
| X-08 | 杠杆（资金/人力/技术） | 四次从零开始——每次靠什么起来的 | 你的工厂靠什么杠杆——大部分只有人力 |
| X-09 | 代理问题 | 合伙人被抓散——信任比合同重要 | 请人管厂最大的坑——利益不一致 |
| X-10 | 沉没成本 | 干了18年互联网说放就放 | 40岁转行最难的不是学新东西——是放掉旧的 |

## 七、选题自检

生成选题后问自己：

```
□ 这个选题是"我今天学到的/看到的/想到的"吗？
□ 它连接了什么过去的经历？
□ 能讲出一个什么道理？
□ 观众能拿走什么？（判断/方法/认知——不能只是"了解"）
□ 跟上一期内容角度不一样吗？
```

三个不要：
- 不要"讲知识"——没人缺知识。缺的是"你怎么看"
- 不要"汇报进展"——巴南落地不是新闻。落地过程中学到了什么才是内容
- 不要"讲道理"——道理要在故事里长出来。不要直接给结论
