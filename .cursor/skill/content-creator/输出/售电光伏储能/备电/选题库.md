# 选题库

> 定位：CVC投行投资人。看新能源、看AI、看人看项目。信息散落→拼成框架→给出判断。
> 更新：2026-07-21 | 战略升级：四象限选题池、来处判断45%冷启动底盘、受众拓宽到有资产的人

## 一、选题生成方式

选题不是"想一个话题"。选题是：**今天看到/学到/想到了什么信息 → 拼上已有的认知 → 能给出什么判断**。

| 方式 | 公式 | 适用象限 |
|------|------|------|
| **信号→判断** | 行业信号 + 框架拆解 + 判断 | 行业判断 |
| **概念→框架** | 学到的概念 + 跨界连接 + 方法论 | 框架判断 |
| **经历→判断** | 过去踩坑/项目进展 + 现在怎么看 + 可用的判断 | 实战判断 |
| **拆解→框架** | 一本书/一个案例/身边人故事 → 快速拆 → 整理成框架 | 来处判断 |
| **AI→实操** | 用了AI做了什么 + 过程分享 + 观众看到可能性 | 行业判断/框架判断 |

| K-09 | 代理问题——请人管厂最大的坑是利益不一致 | 合伙人被抓散→信任比合同重要 |
| K-10 | 沉没成本——18年互联网说放就放 | 40岁转行最难的不是学新东西，是放掉旧的 |
| K-11 | 安全边际——为什么便宜的时候买才是真本事 | 芒格核心框架→用在项目判断上：永远留有余地 |

## 六、选题自检

生成选题后问自己：

```
□ 这个选题给出了什么判断？（不能只是"介绍"或"分享"）
□ 判断的来处清楚吗？（经历/框架/身边案例/经典拆解——至少一个）
□ 观众能拿走什么？（判断/方法/认知——不能只是"了解"）
□ 跟上一期内容角度不一样吗？（同象限内也要差异化）
□ 这个判断只有你能给吗？（跨界视角是壁垒）
```

三个不要：
- 不要"讲知识"——没人缺知识。缺的是"你怎么判断"
- 不要"汇报进展"——巴南落地不是新闻。落地过程中学到了什么判断才是内容
- 不要"罗列信息"——AI能搜到的内容你讲没有价值。你的价值是拼成框架、给出判断
