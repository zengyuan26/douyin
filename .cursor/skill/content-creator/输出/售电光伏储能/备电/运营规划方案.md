## 一、核心定位

### 一句话

**CVC投行投资人。正在学投资，目前手里有一个微电网/新能源项目。把投行视角的行业判断和真实项目进展分享出来。**

不招居间人。不做居间人。目标客户是工厂/企业的老板和直接负责人。

### 信任来源

```
别人说"我懂这行" → 观众：装的吧
你说"我们在看这个方向，这是我们的判断" → 观众：这人有框架
你来时路的故事 → 观众：这人真实，靠得住
```

### 信任是第一 KPI

每一期内容结束后，只有一个检查标准：**观众更信任我了吗？** 不是播放量、不是点赞。

### 人设三角

```
          投行框架 + 真项目在跑
           /\
          /  \
         /    \
        /      \
  互联网18年 ──── 个人品质（不自己说）
  (跨界视角)    (让经历替你说话)
```

| 观众 | 想学能源知识 | 工厂老板/企业负责人。想省电费、想了解行业机会 |

## 三、两条内容线

| 线 | 方向 | 一句话 | 目的 |
|------|------|------|------|
| **来时路（职业线）** | 互联网→CVC投行的转行历程 | 怎么从18年互联网跨到投行。学到的、踩坑的、判断的 | 建立信任。双受众：工厂老板+同龄转行者 |

**两条线交替推进。来时路建立信任，项目路兑现信任。**

### 项目路三类工厂老板内容

| 方向 | 说明 | 信任逻辑 |
|------|------|---------|
| **商业理念** | 投行视角看商业——定价权、交易结构、信号噪声 | "这个人看问题的框架跟我认识的人不一样" |
| **直接解决** | 微电网算账。省多少、怎么省、坑在哪 | "他真能帮我省电费" |
| **育儿/家庭** | 工厂老板40-55岁，孩子教育接班是真正痛点 | "这人跟我一样，也是当爹的" |

### 来时路双受众设计

```
同一期来时路 →

  工厂老板看到 → "这人40岁转行，敢想敢干，靠得住"
  同龄转行者看到 → "我也40岁，我也想转。他是怎么走的"
```

## 四、内容原则

```
□ 信任是第一优先。每一期内容结束问：观众更信任我了吗？
□ "我们在看这个方向" > "我们在判断" > "我们在学" —— 投行视角切入
□ 项目进展持续更新。巴南签约、山东推进——每一条都是信任锚点
□ 政策翻译加"利好什么、利空什么"——投行视角才有
□ 商业模式拆解落到"信号怎么看、项目怎么选"——可判断
□ 互联网18年别浪费。跨界视角是独特武器
□ 省际对比是独特武器。陕西vs重庆vs湖北——大多数能源博主只讲一个省
□ 不招居间人。不提居间人。内容里不出现这个词
□ 记录即资产——来时路每一期是未来知识付费的素材
```

## 六、概念图谱系列（PPT+录音·项目路深度内容）

### 定位

每周一期长视频（5-10分钟），用PPT+录音的方式讲透一组能源概念的关系框架。不出镜，零拍摄成本。

### 对标参考

大山（抖音）"源网荷储虚拟电厂绿色微电网"——金字塔框架图+分层拆解+场景串联演示+总结问答。观众看完的感觉是"通透"。

### 与日常短视频的分工

| | 日常短视频 | 概念图谱系列 |
|------|------|------|
| 形式 | 分身出镜+AI画面 | PPT+录音 |
| 时长 | 2-5min | 5-10min |
| 目标 | 拉新、建立信任 | 深度收藏、建立专业信任 |
| 频率 | 日更/隔日更 | 每周1期 |
| 内容 | 单点观点/故事/事件 | 系统性框架拆解 |

### 系列选题（15期）

| 期 | 选题 | 核心框架 |

| 7 | 分时电价取消后，游戏规则怎么变了 | 旧规则vs新规则对比 |

### PPT制作规范

- 工具：Canva（搜索"教育培训PPT"模板，选白底+细线+单色点缀风格）
- 比例：9:16竖版（匹配抖音全屏）
- 配色：备电暖调——琥珀[#C89050]标题色，暖深灰[#1a1e24]正文色，白底
- 字号：标题≥48pt，正文≥28pt（手机小屏可读）
- 每页一个核心信息点，不堆砌

### 录音规范

- 剪映录音，逐页念稿
- 节奏：关键概念放慢，过渡段正常语速
- 语气：像在给朋友讲一个你刚学会的东西，不是在上课

## 七、运营节奏

### 第1个月
- 日更。来时路+项目路交替
- 来时路打信任基础：转行故事、合伙人教的、踩坑经历
- 项目路打专业基础：政策翻译、商业理念、概念图谱
- 不追数据。追"有没有人说这人有东西"
- 核心指标：有没有人说"关注了，真在干"

### 第2-3个月
- 基于数据调整两条线的配比
- 项目路深度内容出爆款——陕西重庆湖北对比、基金暗战、EMC接管模式
- 直接解决类内容开始吸引工厂客户私信
- 来时路积累知识付费素材

### 第3-6个月
- 项目路持续更新进展+深度拆解
- 在投行圈建立"用内容获客的CVC投资人"标签
- 人设自然过渡到"投了多个项目、有自己方法论的投资人"
- 来时路素材储备充足，可考虑知识付费产品形态

## 八、互动策略

```
□ 有人问专业问题：会的就答，不会就说"我也刚学到，我的理解是…"
□ 有人质疑投资人身份：不辩解。发一期"巴南项目是怎么谈下来的"
□ 有人问怎么合作：私信聊，不在评论区推销
□ 有人共鸣来时路：认真回复。这些人是未来知识付费的种子用户
□ 有人提居间人/代理：不讨论。你是投行投资人，不招居间人
```
