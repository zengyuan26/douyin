## 一、核心定位

### 一句话

**互联网产品经理出身 × 创业多次 × 会用AI做工具做内容 × 正在学投资。四层叠在一起，市面上没有第二个人。**

不是教工厂老板怎么做工厂。不是讲投资知识。不是AI教程。是**学习、判断、做事的过程**——今天学到什么 → 联系过去经历 → 讲一个道理。

不招居间人。不做居间人。目标客户是工厂/企业的老板和直接负责人。

### 为什么是你

| 层 | 能力 | 观众看到什么 |
|------|------|------|
| 产品经理 | 用户洞察、需求拆解、MVP思维 | 他看问题的角度跟别人不一样 |
| 创业多次 | 成过败过，知道什么生意会死 | 他说的有来处——踩过的坑就是框架 |
| AI+软件 | 会用AI做工具、做内容、提效率 | 他不是在讲AI概念，他是真在用 |
| 投行学徒 | 正在学投资。资本怎么想、交易结构 | 他的判断有投行框架，不是拍脑袋 |

四层叠起来的结果：**看到的东西天然跟别人不一样。** 内容池无限——每天学新东西，每个概念都能联系过去的经历，每个联系都是一期内容。

### 信任来源

```
别人说"我懂这行" → 观众：装的吧
你分享学习过程+跨界连接 → 观众：这人的框架不一样
你来时路的故事 → 观众：这人真实，靠得住
```

### 信任是第一 KPI

每一期内容结束后，只有一个检查标准：**观众更信任我了吗？** 不是播放量、不是点赞。

### 竞品差异

| | 汤老师新能源 | 增元AI棱镜 |
|------|------|------|
| 身份 | 能源行业专家 | **产品经理×AI×创业×投行。跨界学习者** |
| 讲什么 | 技术标准、政策原文 | **学到概念→联系经历→翻译成对你有用的** |
| 信任 | 行业经验 | **跨界框架 + 真实经历 + 真项目在跑** |
| 观众 | 想学能源知识 | 工厂老板/企业负责人。想省电费、想看懂行业 |

### 初期配额（前3个月）

| 象限 | 占比 | 是什么 | 说明 |
|------|------|------|------|
| **概念** | 40% | 行业概念、新闻分析、政策翻译 | 主力内容。量大、生产快、拉新 |
| **框架** | 40% | 投资判断、商业逻辑、交易结构 | **不限新能源。** 通用商业/投资框架，部分接项目讲 |
| **故事** | 10% | 项目进展、谈判、踩坑 | 前期项目少，配额低 |
| **价值观** | 10% | 来时路、人生观、40岁转行 | 初期没转化，轻量维持存在感 |

### 配额演进

```
前3个月：概念40% + 框架40% + 故事10% + 价值观10%
  ↓ 项目多了、信任建立后
3-6个月：概念30% + 框架35% + 故事20% + 价值观15%
```

概念+框架永远是底盘，不低于 60%。故事和价值观随项目积累自然增长。

## 四、内容原则

```
□ 信任是第一优先。每一期内容结束问：观众更信任我了吗？
□ 学着讲着。不装专家。今天学到的东西今天就翻译出来
□ 跨界视角是核心武器——产品经理+创业+AI+投行，看到的东西天然不一样
□ 每天问自己：今天学到了什么→能连接什么经历→能讲出什么道理。有连接就是选题
□ 层层追问到机制层。新闻是起点不是素材，不把新闻直接丢给AI生成
□ 不懂不发。自己没搞明白的内容不出。口播/图文/视频都一样
□ 认知差筛选：讲完观众会觉得"原来是这样"还是"不过如此"？后者不发
□ 项目进展持续更新。每一步学到的东西才是内容，不是进展本身
□ 不在内容里讲道理——道理要在故事里长出来
□ 互联网18年是底色不是标签——需要的时候天然在那里
□ 不招居间人。不提居间人。内容里不出现这个词
□ 记录即资产——每次发布后提取概念/数据/判断到 Obsidian 知识库
```

### PPT制作规范

- 工具：Canva（搜索"教育培训PPT"模板，选白底+细线+单色点缀风格）
- 比例：9:16竖版（匹配抖音全屏）
- 配色：备电暖调——琥珀[#C89050]标题色，暖深灰[#1a1e24]正文色，白底
- 字号：标题≥48pt，正文≥28pt（手机小屏可读）
- 每页一个核心信息点，不堆砌

### 录音规范

- 剪映录音，逐页念稿
- 节奏：关键概念放慢，过渡段正常语速
- 语气：像在给朋友讲一个你刚学会的东西，不是在上课

## 七、运营节奏

### 第1个月
- 日更。按内容象限配额走：概念40% + 框架40% + 故事10% + 价值观10%
- 概念象限主力形态：图文（A型分析）+ 口播短视频
- 框架象限主力形态：口播短视频，不限能源，商业/投资通用话题
- 不追数据。追"有没有人觉得这人视角不一样"
- 核心指标：有没有人说"关注了，这人看事情的角度有意思"

### 第2-3个月
- 基于数据调整四象限配比
- 框架象限最有机会出差异化——跨界视角+通用商业框架，别人讲不了
- 概念象限吸引行业搜索流量，框架象限留住人
- 直接解决类内容（概念象限的算账/避坑）开始吸引工厂客户私信

### 第3-6个月
- 项目路持续更新进展+深度拆解
- 在投行圈建立"用内容获客的CVC投资人"标签
- 人设自然过渡到"投了多个项目、有自己方法论的投资人"
- 来时路素材储备充足，可考虑知识付费产品形态

## 八、日更节奏：三层金字塔

### 核心理念

日更 ≠ 追热点。日更 = 每天学一点，每天记一点。爆款是偶尔的烟花，日常是地基。

| 目标 | 需要什么 | 频率 |
|------|---------|------|
| 引流（让人看到你） | 爆款，追热点 | 偶尔，月1-2条 |
| 留人（让人记住你） | 稳定、有规律、能预期 | 每天，轻量 |
| 建信（让人信任你） | 深度、有判断、有来处 | 每周，1条深度 |

### 三层结构

```
第一层  每日学习笔记（轻量·地基·日更）
        100-200字图文卡片，5分钟写完。
        每天学的东西记下来就是内容，不需要额外创作。

第二层  每周深度一条（中量·信任锚·周更）
        平行世界叙事/概念图谱/项目复盘/行业信号解读。
        30-60分钟，有框架有判断。

第三层  每月爆款一条（重量·引流口·月更）
        追大热点，用跨界框架给出别人给不了的判断。
        时机到了才发，不强求。
```

### 学习笔记四分类

学习笔记不限于新能源。每天学的所有东西都该记——跨界视角是核心武器。

| 类别 | 来源 | 给谁看 | 建什么信 |
|------|------|------|------|
| 行业概念 | 储能/售电/微电网/电力交易的新概念 | 工厂老板 | "这个人懂行" |
| 投行方法 | IRR怎么算、DCF假设怎么设、对赌条款怎么看 | 同行+潜在合作方 | "这个人专业" |
| 跨界工具 | AI做了什么、产品经理怎么拆需求、创业踩过什么坑 | 工厂老板+转行者 | "这个人视角不一样" |
| 生活感悟 | 跟孩子聊了什么、健身时想到什么、读了什么书 | 双受众 | "这个人真实，跟我一样" |

### 周节奏

```
周一  行业概念（地基·专业）
周二  投行方法（地基·深度）
周三  跨界工具（地基·差异化）
周四  行业概念（地基·专业）
周五  生活感悟（地基·真实）
周六  每周深度一条（锚点）
```

一周 6 条内容，只有 1 条需要认真做，其余 5 条是学习笔记。

### 为什么学习笔记是最佳日更形态

- 你每天本来就在学习，记下来就行，不需要额外创作
- 100 条笔记 = 一本"概念词典"，未来可以做知识付费
- 观众每天看到你在学、在想、在做，比一次爆款建立信任更持久
- 不追热点不焦虑——热点不是你的日常，学习才是
- 没人在做"投行学徒的每日学习笔记"——这个位置是空的

### 学习笔记固定格式

```
🔋 学习笔记 | #XX概念

一句话解释：XX就是……

今天学了XX → 想到以前做互联网/创业的时候遇到过类似的情况 →
给工厂老板一个判断：……

#学习笔记 #CVC投行 #新能源
```

### 与内容象限的关系

```
日常笔记（周一至周五）→ 拉满概念象限和框架象限，兼顾故事/价值观
每周深度（周六）→ 打象限锚点，周末流量靠深度留存
每月爆款（不固定）→ 看时机，不预设象限
```

## 九、互动策略

```
□ 有人问专业问题：会的就答，不会就说"我也刚学到，我的理解是…"

□ 有人问怎么合作：私信聊，不在评论区推销
□ 有人共鸣来时路：认真回复。这些人是未来知识付费的种子用户
□ 有人提居间人/代理：不讨论。不招居间人
```
