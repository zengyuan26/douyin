# Canva制作任务单：四张B型产业链图文

> 参考风格：摩尔产业观察（抖音搜"摩尔产业观察"看图）
> 格式：3:4比例，1080×1440px
> 工具：Canva
> 母版：先用模板做储能产业链图，调好后复制母版改内容做另外三张

---

## 全局设计规范

**配色（备电暖调，区别于摩尔冷蓝）：**
- 丝带主色：琥珀 [#C89050]
- 上游/重点层：琥珀 [#C89050]
- 中游/系统层：暖青 [#4D8F8F]
- 下游/赚钱层：暖绿 [#6A9E6A]
- 文字：深灰 [#333333]
- 连接线：浅灰 [#CCCCCC] 1px极细
- 背景：纯白 [#FFFFFF]

**节点样式：**
- 圆角矩形，圆角2-3px
- 1px细边框（跟它所在层的颜色一致）
- 填充：层颜色10%透明度（几乎看不出，但不是纯白）
- 文字12-13px，深灰 [#333333]

**连接线：**
- 1px极细，浅灰 [#CCCCCC]
- 拐弯处直角，不弯不斜

**字体层级：**
- 丝带标题：白色粗体，18-20px
- 副标题：深灰 [#555555]，12-13px
- 层级标签：白字在色块上，10-11px
- 节点文字：深灰 [#333333]，12-13px

**头部模板（每张图固定）：**
```
┌──────────────────────────────────┐
│     [彩色丝带横幅 · 白色粗体标题]    │  ← 丝带高60-80px
├──────────────────────────────────┤
│        副标题（深灰小字居中）       │
│    ───── 1px细线 ─────          │
└──────────────────────────────────┘
```

## 图3：工商业储能投产比模型

**丝带色：琥珀 [#C89050]**
**丝带标题：工商业储能投产比模型**
**副标题：1MW/2MWh 标准项目 · 工厂老板算账专用**

**内容结构（4行数据卡，数字大字突出）：**

```
ROW 1（琥珀左边条）：建一套要花多少 → ≈ 120-150 万 → 设备+安装+并网

ROW 4（琥珀左边条）：还没算的 → 15分钟现货电价落地→套利更大 / 碳双控→绿电+储能=碳排零
```

## 图4：售电交易流程图

**丝带色：暖青 [#4D8F8F]**
**丝带标题：售电交易全流程图**
**副标题：15分钟现货电价怎么玩**

**内容结构（3节点环形图，箭头串联成环）：**

```
       ┌──────────────────────┐
       │   ① 发电端（暖青）     │
       │ 火电/风电/光伏        │
       │ → 每15分钟报价        │
       └──────────────────────┘
              ↓ 报价
       ┌──────────────────────┐
       │  ② 交易层（琥珀）     │
       │ 现货：每15分钟定价    │
       │ 中长期：月度/年度合约  │
       │ 辅助服务：调频调峰    │
       └──────────────────────┘
              ↓ 价格信号
       ┌──────────────────────┐
       │ ③ 用电端（暖绿）     │
       │ 低价→多用电+充电     │
       │ 高价→少用电+放电     │
       │ 需求响应→少用拿钱    │
       └──────────────────────┘
         ↗ 反馈→下一轮报价
```

**底部：**
- 金句："交电费从被动付款变成主动交易"
- 免责声明
