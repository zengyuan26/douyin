## 文案（图片配文·摩尔7模块）

整理自留！磷酸铁锂、钠离子、全钒液流——三种电池一张图讲透。

很多人以为电池就分"固态"和"液态"两种。但真正的关键是——按电解液体系分，市面上主力是液态锂离子、液态钠离子、液流电池三条路，固态还在路上。

给你拆一下三条路

① 磷酸铁锂 — 绝对主力：液态电解液 / 160-200Wh/kg / 4000-6000次 / 0.3-0.5元/Wh
② 钠离子 — 新锐崛起：液态电解液 / 120-160Wh/kg / 3000-5000次 / 0.2-0.4元/Wh
③ 全钒液流 — 长时王者：水基电解液罐外循环 / 15-25Wh/kg / 15000-20000+次 / 2-3元/Wh

本质就是一句话——磷酸铁锂赢在性价比，钠离子赢在原材料，全钒液流赢在长时安全。没有谁通吃，场景选电池。

为什么现在必须搞懂这三种电池？
· 新能源装机爆增，4小时以上长时储能成了刚需
· 锂价大起大落，钠离子作为备份路线加速量产
· 2026年钠电量产元年+液流项目大规模并网，两种新路线同时起跑

我自己最近看储能项目的时候有个变化。以前只看锂电池方案，现在每个项目会先想：这个场景需要长时还是短时？空间够不够？然后倒推该用哪种电池。

建议大家下次看储能项目，按这四个指标套一遍——成本、安全、能量密度、循环寿命。你对这个项目该用什么电池，心里就有数了。

（以上内容仅作为行业知识科普，不构成任何建议）
