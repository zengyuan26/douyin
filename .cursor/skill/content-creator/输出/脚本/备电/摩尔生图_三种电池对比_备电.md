## 模块一：工作原理（三个窄条上下堆叠）

### 第一条 · 磷酸铁锂（液态锂离子电池）

正极磷酸铁锂 + 负极石墨 + 液态有机电解液 + 隔膜。锂离子在正负极间来回跑。充电正→负，放电负→正。
一句话：**锂离子在液体里，正负极之间来回跑。**

### 第二条 · 钠离子电池（液态钠离子电池）

结构跟锂电一模一样。唯一区别：把锂盐换成钠盐。钠离子比锂离子胖，能量密度低一些，但钠海里全是。
一句话：**把锂换成钠，结构一样，也是液态。**

### 第三条 · 全钒液流电池（液流电池）

完全不同体系。钒离子溶在硫酸里，装两个大罐子。充放电时泵进电堆，钒离子在膜两侧得失电子。功率靠电堆，容量靠罐子，分开设计。
一句话：**电解液在罐子和电堆之间泵来泵去，功率容量分开。**

## 模块二：四大指标对比

紧凑表格，每行高度约120-150px。四种电池用颜色小圆点区分：🟠磷酸铁锂 / 🔵钠离子 / 🟢全钒液流。

**成本潜力**
- 🟠 0.3~0.5 元/Wh — 工艺成熟，当前最便宜
- 🔵 0.2~0.4 元/Wh — 钠资源无限，理论低20-30%
- 🟢 2~3 元/Wh — 初始贵，用20年摊下来划算

**安全性**
- 🟠 较好 — 橄榄石结构稳定，但仍含可燃液
- 🔵 好 — 钠活性更低，可过放到0V，但仍含可燃液
- 🟢 最强 — 水基电解液不燃不爆，常温常压

**能量密度**
- 🟠 160~200 Wh/kg — 最高但到天花板
- 🔵 120~160 Wh/kg — 储能够用，移动场景偏低
- 🟢 15~25 Wh/kg — 极低，不适用于移动场景

**循环寿命**
- 🟠 4000~6000 次 — 够用
- 🔵 3000~5000 次 — 略短
- 🟢 15000~20000+ 次 — 碾压级，20年+，电解液可回收

## 模块三：应用与趋势（三个卡片上下堆叠）

**卡片1 · 磷酸铁锂**
储能绝对主力（占70%+），电动车中低端标配。地位稳固但天花板到顶。钠电蚕食低端，液流分流长时。现在的王者，份额会被切。

**卡片2 · 钠离子电池**
2025-2026量产元年。储能主战场，吃铅酸退出+低端LFP份额。跟LFP互补不是替代。拼便宜+低温场景。

**卡片3 · 全钒液流**
政策力推长时储能(4h+)。2025年乌海200MW/800MWh已并网。守长时赛道，不抢移动场景。
