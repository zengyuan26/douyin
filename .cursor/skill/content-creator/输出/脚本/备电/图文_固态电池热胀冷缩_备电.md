### P4 · 热胀冷缩卡死全固态 — 坚定/清晰 · matrix

**画面文字（金句）：** 一个初中物理问题。充一次电胀一次，放一次电缩一次。胀缩几次，路就断了。

**内容：**

```
液态为什么没这个问题？
电解液是液体，能变形、能流动。
正负极胀缩 → 液体自动填满缝隙。
锂离子永远有路走。

固态为什么有这个问题？
固体硬碰硬。压得再紧，一充一放——
热胀冷缩 → 缝隙越来越大 →
锂离子跑到一半，前面没路了。

堵死的地方，锂离子越积越多
长出尖刺一样的锂枝晶
把固态电解质戳穿。短路。
```
