## 第3页：到底在调什么

**标题**：每天傍晚六点。电网最危险的十分钟。

两个对比色块并排：

| 发电端 ↓ | 用电端 ↑ |
|----------|----------|
| 光伏十分钟归零 | 下班，空调全开 |
| 风电骤降 | 工厂夜班启动 |

电网铁律：**发多少 = 用多少。每时每刻必须相等。**

谁在调？火电爬坡半小时起步——**速度跟不上**。水电80%在西南，用电在东部——**距离够不着**。

> 口播：电网有一条铁律。电网必须实时平衡——发电+储能充放，每一刻都等于用电，频率才能稳在50Hz。傍晚六点。太阳落山，光伏十分钟从满发到归零，风电骤降。同时所有人下班回家，空调开机，工厂夜班。发电暴跌，用电暴涨。谁在调？火电在爬坡，半小时起步。光伏归零只要十分钟，跟不上。水电快一点，但80%在西南，用电在东部，隔了两三千公里，够不着。这就是虚拟电厂要解决的问题。

## 第4页：怎么调

**标题**：不调发电。调用电。一秒钟。

架构图：

```
  ┌─────────────────────────┐
  │   电网调度中心            │
  │   管大电站（高压侧）       │
  │   火电/大型风电/大型光伏   │
  └───────────┬─────────────┘
              │
  ┌───────────↓─────────────┐
  │   VPP  管分散资源（低压侧） │
  │   屋顶光伏/分布式风电/     │
  │   储能/充电桩/空调负荷     │
  └─────────────────────────┘
```

分界线：**大的、接高压侧的，电网调。小的、接低压侧的、分散的，VPP 攒成池子调。**

左右对比：

| 传统调度 | VPP调度 |
|---------|---------|
| 调发电端 | 调用电端 |
| 火电爬坡半小时 | 同一秒，几万条指令出去 |
| 水电两千里外 | → 储能：立刻放电 |
| **跟不上也够不着** | → 充电桩：暂停充电 |
| | → 空调：上调两度 |

大字：发电端没动。用电端压下来。供需重新平衡。**一秒钟。**

> 口播：先看 VPP 在电力系统中的位置。最上层是电网调度中心——管火电、大型风电、大型光伏，接在输电骨干网上，高压侧。VPP 在下一层——管的是分散的小资源。屋顶光伏、园区小风机、储能、充电桩、空调。单个太小，电网没法一个个管。VPP 把它们攒成池子，以一个整体参与调度。怎么调？不碰发电端，直接调需求端。同一秒，几万条指令出去——储能立刻放电、充电桩暂停、商场空调上调两度。发电端没变，用电端压下来了。供需重新平衡，一秒钟。

## 第5页：怎么赚

**标题**：一个资源池。同时吃四份钱。

四张数字卡片：

| ① 调峰辅助 | ② 需求响应 | ③ 现货套利 | ④ 容量备用 |
|-----------|-----------|-----------|-----------|
| 电网喊"帮我调" | 尖峰主动降负荷 | 低价充电 | 固定底薪 |
| **0.3-1.0元/度** | 广东≤**3.5元** 深圳**5元** | 高价放电 | 待命就拿钱 |

核心数字：同一个资源池，利用率 **30% → 80%+**

> 口播：虚拟电厂怎么赚钱？同一资源池，错峰吃四份钱。 第一份，调峰辅助——电网喊你调峰，一两毛到一块钱一度。 第二份，需求响应——尖峰时刻主动降负荷，广东最高三块五，深圳实时五块钱一度。 第三份，现货套利——低价充电，高价放电。 第四份，容量备用——按容量拿固定收益，不充放也能赚钱。 同一资源池，利用率从30%拉到80%以上。不是多建一个电站，是多赚三份钱。

## 第8页：储能的机会

**标题**：储能是子弹。虚拟电厂是枪。

三行，自下而上逐行放大：

储能响应**毫秒级** · 可充可放 · 容量确定
↓
接入 VPP 平台
↓
从自己套利 → 同时拿**四份收入**

底部：储能的未来价值，不看电芯卖多少钱。**看它能接入什么样的调度体系。**

> 口播：最后。储能跟虚拟电厂什么关系？
储能是VPP最优质的调度资源——响应速度快，可充可放，容量可控。

一个储能电站接入VPP平台，身份变了：从单打独斗自己套利，变成协同作战主动参与。以前只赚一份价差，现在可以错峰拿四份收入。

储能是子弹。虚拟电厂是枪。那金字塔再往下一层——VPP调度的这些资源，从哪来？微电网。工厂园区围墙里面，自己发电、自己储电、自己用电。怎么建、怎么算账、跟VPP怎么联动。下期拆。
