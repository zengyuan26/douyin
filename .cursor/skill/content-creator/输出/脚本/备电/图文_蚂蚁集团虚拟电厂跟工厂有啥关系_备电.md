## 五、制作规范

### 设计规格

```
✅ 图片数量：7张（9:16竖版）P1封面 → P7转化
✅ 文字大小：标题 ≥40pt，金句 ≥40pt，正文 ≥16pt
✅ 核心观点：必须醒目，每张图有1个金句视觉块
✅ 案例数据：必须具体（数字/来源/城市/企业名）
✅ 互动引导：必须明确（P7必须有CTA）
✅ 人物不出镜：7张图全部使用场景+信息图风格，不出现人物
✅ 光影约束：统一柔和自然光（High-key），禁止硬阴影
✅ 文字密度：每张图文字不超过画面60%，保持呼吸感
✅ 禁止项：手机Mockup、emoji、特殊符号、纯色/渐变背景
```

### 字体规范

```
✅ 字体选择（只使用这4种）：
   思源黑体（推荐）、微软雅黑、华文细黑、Arial

✅ 禁止特殊字符：
   代码块符号、数学符号、装饰符号、Emoji

✅ 中文标点（只使用）：
   逗号（，）、句号（。）、顿号（、）、冒号（：）、引号（「」"）

✅ 排版简化：
   文字尽量一行说完
   每张图文字不超过80字
   禁止竖排文字
   禁止表格、分隔线、边框
```
