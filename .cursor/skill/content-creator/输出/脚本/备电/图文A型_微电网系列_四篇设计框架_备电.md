# 图文A型_微电网系列_四篇设计框架

> 系列总题：**微电网：四份文件，一个窗口**
> 格式：A型图文，每篇7张 | 9:16 竖版 1080×1920px | GPT Image2 生图
> 发布节奏：每2天一篇，8天发完
> ⚠️ 广告合规：公司名≤5次，无"投资/研判/暴增"，无CTA，有免责声明

## 四、各篇详细设计

### 系列①：工厂老板用脚投票了

**目标圈层**：工厂老板（第一优先）。行业同行 + 泛人群（暴跌数据吸引）。

**封面标题**：工厂老板用脚投票了
**封面副标题**：Q1工商业光伏暴跌61.5%。山东-87%。同一个市场，户用暴涨89%。
**封面标签**：微电网 ①/④（右上角琥珀小标）

| P3 | 以前账算得过来 | 只装光伏，0.3-0.5元/W，6-7年回本。工厂老板愿意投 | 简化ROI时间轴：投入→第7年回本→后面净赚 |
| P4 | 现在账变了 | 全额上网变自发自用。周末停产光伏白发电→消纳率打折→回本8-9年 | 消纳率对比图：预期80% vs 实际50-60% |
| P5 | 还要配储能 | 黄区必须配储。+12万投入。储能回本6.5年。整体回本10-12年 | 总成本饼图：光伏60万 + 储能12万 = 72万 |
| P6 | 四个顾虑 | 回收期太长/技术迭代快/容量电价不明/"被逼迫"花钱 | 四个图标 + 一句话 |
| P7 | 下篇预告 | 账算不过来。但四份文件正在把账重新算过。下篇拆 | 琥珀钩子文字 |
