### 图片3：知识科普 / 为什么不是越白越好 ⚠️强制9:16

**【内容功能】**

- 用通俗语言解释：颜色、粗细、根须代表什么
- 不做专业检测科普，做"肉眼可见的避坑指南"

**【画面描述】**

- 左右对比表格式视觉：
  - 左栏标题：❌ 很多人买的"问题豆芽"
  - 右栏标题：✅ 自然生长的好豆芽
- 用3-4个对比点配简笔插画或实拍局部图（颜色、根须、粗细）
- 背景浅色，信息一目了然
- ⚠️ 禁止：手机外框、手机Mockup、任何电子设备外壳

**【文字内容】**

```
标题：豆芽不是越白越好，看这3点就够了

左侧（❌ 问题豆芽特征）：
1. 颜色
   · 白得发亮、发青，甚至有点透
2. 根须
   · 基本没根没须，切口非常整齐
3. 粗细
   · 粗得不正常，几乎一根粗杆
4. 手感
   · 软塌塌、不脆，掐一下容易出水

右侧（✅ 普通好豆芽长这样）：
1. 颜色
   · 微微发黄或乳白，不会"惨白"
2. 根须
   · 有根、有须，根部略有点"小毛毛"
3. 粗细
   · 粗细自然，有细有粗，看起来是"长出来的"
4. 手感
   · 掐起来脆、容易折断，不黏手

结论小结：
真正好豆芽，是"长得自然"的样子，
而不是"处理得太完美"的样子。
```

**【设计规格】**

- 做"对比表 + 小图标"式版面，方便截屏收藏
- 关键词加粗：颜色 / 根须 / 粗细 / 手感

**【内容功能】**

**【画面描述】**

- 画面左侧真实豆芽特写，右侧用简洁图标+文字列出"3步选豆芽"
- 氛围干净、安全、可信赖
- ⚠️ 禁止：手机外框、手机Mockup、任何电子设备外壳

**【文字内容】**

```
标题：买豆芽记住这3步，基本不踩坑

步骤1：先看颜色
- 微黄/乳白色，更接近自然
- 太白、发青、发亮的先放下

步骤2：再看根须
- 有根、有须，像"头发丝"才正常
- 完全没根、太整齐，要提高警惕

步骤3：最后掐一下
- 掐一下"咔嚓"脆断的是好豆芽
- 软塌塌、掐不动、手上黏黏的，别要

小提示：
- 不要贪"好看、好粗、好白"
- 食材是给家人吃的，安全永远排第一
```

**【设计规格】**

- 结构：标题 → 3步编号 → 底部温馨提示
- 用小图标标注"看颜色/看根须/掐一下"
