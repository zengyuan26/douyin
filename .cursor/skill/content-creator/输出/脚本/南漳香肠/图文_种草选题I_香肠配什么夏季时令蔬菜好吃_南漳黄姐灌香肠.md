  - 香肠切薄片更香

**【文字内容】**
```
主标题：香椿炒香肠做法

食材准备：
  - 香椿一小把（嫩芽部分）
  - 香肠2-3根（蒸熟切片）
  - 鸡蛋2个
  - 盐少许

做法步骤：
  1. 香椿只取嫩芽部分
  2. 焯水1分钟变绿捞出
  3. 切碎备用
  4. 香肠蒸熟切片
  5. 鸡蛋打散煎成蛋块
  6. 香肠香椿一起翻炒
  7. 加入鸡蛋加少许盐调味

小贴士：
  - 香椿一定要焯水去亚硝酸盐
  - 只用嫩芽，老叶子会苦
  - 鸡蛋可以吸收香肠的油

底部：香椿特殊香气+香肠油香绝了
```
