```
主标题：老师傅教你3招解决香肠太咸

方法1（浸泡减盐法）：
具体操作：
  - 清水浸泡2-3小时
  - 中间换水1-2次
  - 不要用热水，会流失香味
  - 适合：炒菜前处理

方法2（蒸煮减盐法）：
具体操作：
  - 先蒸熟再切片
  - 蒸的时候放点料酒姜片
  - 蒸汽会带走部分盐分
  - 适合：直接切片吃

方法3（搭配降盐法）：
具体操作：
  - 炒菜时搭配清淡蔬菜
  - 比如蒜苗炒香肠、芹菜炒香肠
  - 蔬菜会吸收部分盐分
  - 适合：做菜配饭吃

底部提醒：减咸不减香，这样吃才健康
```
