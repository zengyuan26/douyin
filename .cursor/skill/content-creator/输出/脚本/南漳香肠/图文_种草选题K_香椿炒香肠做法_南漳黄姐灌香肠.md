#### 图片4：做法步骤【强制9:16】

**【内容功能】** 详细展示制作步骤

**【画面描述】**
- 背景：炒菜过程场景虚化
- 布局：竖版9:16，标题+5个步骤
- 元素：步骤数字、关键提示
- 风格：清晰易学，便于操作

**【文字内容】**
```
主标题：详细做法步骤

步骤1（处理香椿）：
  - 只取香椿嫩芽部分
  - 老叶子不要（会苦）
  - 洗净备用

步骤2（香椿焯水）：
  - 烧一锅开水
  - 放入香椿焯水1分钟
  - 变色后捞出过凉水
  - 重要：去亚硝酸盐和苦味

步骤3（切碎备用）：
  - 焯好的香椿挤干水分
  - 切成小段
  - 约1厘米长

步骤4（处理香肠）：
  - 香肠冷水上锅蒸15分钟
  - 蒸熟后切成小丁
  - 或者切片也可以

步骤5（炒制出锅）：
  - 鸡蛋打散煎成蛋块盛出
  - 热锅少油炒香肠丁
  - 炒出香味
  - 加入香椿段快速翻炒
  - 加少许盐调味
  - 加入鸡蛋块翻拌均匀

底部：全程几分钟，简单快手！
```
