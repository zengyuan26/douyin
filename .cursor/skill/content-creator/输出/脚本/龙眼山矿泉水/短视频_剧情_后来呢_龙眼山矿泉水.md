## 设计思路

### 四线映射表

```
                       掐断点
                         ↓
 情节线 ──→ 冲突 ──→ 反转 ──→ 裂开 ──→ ✗（不演）
 人物线 ──→ 赶她走 ──→ 被问 ──→ 绷断 ──→ 沉默
 物件线 ──→ 凉水·照片 ──→ 杯沿茶渍 ──→ 挂钟滴答 ──→ 字幕
 观众线 ──→ 困惑 ──→ 心沉 ──→ 鼻酸 ──→ 沉默（填自己的遗憾）
```

| 秒 | 情节线 | 老头 | 妹儿 | 物件 | **观众感受** | **观众想起** |
|----|--------|------|------|------|------------|------------|
| 0-1 | "你走嘛" | 赶她走。抖。 | 僵住。 | — | 困惑 | — |
| 1-5 | 杯子+照片 | 转身。脸湿。 | 看见。 | 旧杯子·凉水·照片 | 警觉 | — |
| 5-10 | 觉得不对劲 | 沉默。 | 没走。在看。 | 杯沿茶渍 | 跟她一样不安 | — |
| 10-12 | "阿姨了？" | — | 随口问。以为在里屋。 | — | 没防备 | 每天一样的问候 |
| 12-15 | "昨天晚上……还笑起的" | 开始说。 | 在听。 | — | **心一沉** | "昨晚"——坏了 |
| 15-17 | **"我出去接……"断了** | 绷断。满脸。 | 愣住。 | 挂钟滴答 | **鼻酸** | **那个没在的瞬间。那通电话。** |
| 17-20 | 她看向杯子 | 说不下去。 | 突然明白。 | 旧杯子。照片。 | 震惊 | — |
| 20-23 | 字幕「后来喃」 | 沉默。 | 没走。 | 挂钟。字幕。 | **沉默。低头。** | **我得给谁打个电话。** |

### 四条线的分工

```
情节线 → 控制信息释放。「先冲突、藏原因、随口问引爆、不演结局」
人物线 → 控制共情入口。「观众通过妹儿的眼睛看这件事」
物件线 → 控制视觉锚点。「不说话的时候，杯子、照片、钟摆在说话」
观众线 → 控制最终落点。「不是看别人哭。是想起自己」
```

### 交汇点：掐断

```
四条线在"我出去接……"这一帧交汇。

  情节 → 信息在最关键的一刻戛然而止
  人物 → 想说。说不下去。
  物件 → 只有钟摆在响
  观众 → 鼻酸。想起自己没在的那个瞬间

  这一帧之后。什么都不需要了。字幕替观众问：后来喃。

## 故事逻辑自检

```
□ 前一秒怎么留人？
  → 黑屏。"你走嘛。"冲突先砸下来。然后才揭开。

□ 情绪怎么推？
  → 赶她走 → 她没走 → 随口问"阿姨了" → 他崩了 → 她突然明白。
  → 从"以为今天跟往常一样"到"阿姨昨晚走了"。反转。

□ 裂开在哪里？
  → "我出去接……"声音断了。说不下去了。
  → 就出去接个水的功夫。回来人没了。没说完比说完重。

□ 为什么叫「后来喃」？
  → 字幕替观众问。不是她在问。
  → 每个人心里都有一个"我出去接……"的瞬间。
  → 那个人。那通没打的电话。
```
