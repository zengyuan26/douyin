> ⚠️ **H-V-F 分析**
> - **H（钩子）**：「敲一下门」三个字——表面是请求，底下是「我怕我死在家里没人知道」

## 剧情脚本（一镜到底，4段情绪）

> 拍摄方式：固定机位，中景→近景缓推。一个镜头到底。不切机位，靠表演和光线推动情绪。
> 节奏：误会制造(8s) → 反转揭示(10s) → 情感爆发(13s) → 安静收尾(5s)
>
> **前3秒钩子逻辑**：婆婆一把抓住女主角，语气严肃。观众和女主角一起想——「哪里做错了？水有问题？要投诉？」然后翻过来。

---

### 段落1：误会。她以为自己闯祸了。（0-8秒）

```
固定机位，中景。老式客厅。挂钟滴答声贯穿。

女主角刚把桶装水放好，直起身。
顺手把门口一双歪了的旧拖鞋摆正。
转身准备走。

突然——
婆婆从画外伸出一只手。一把抓住她的手腕。
力气不轻。不像平时那个和和气气的婆婆。

婆婆（重庆话，声音严肃，眼神直直地看着她）：
"妹儿。你过来。"
（停半拍）
"我有事跟你说。"

女主角表情瞬间绷紧。
脑子里飞快转——水有问题？钱没给够？还是碰坏了什么东西？
嘴唇动了动。没敢说话。
（观众也在猜：怎么了？）
```

**钩子逻辑**：前 3 秒，观众看到一个老人突然抓住送水工的手，语气严肃。制造「要发难」的错觉。不会划走——想知道发生了什么事。

**台词**：婆婆 2 句 | 女主角 0 句

---

### 段落2：反转。不是指责。是求救。（8-18秒）

```
婆婆没有松开她的手。
另一只手从身后拿出一个旧铁盒。
打开。里面是叠好的零钱。
铁盒下面露出一角医院挂号单。

婆婆（重庆话，声音突然轻下来。不是刚才那个严肃的语气了）：
"你每天上班。来敲一下我屋头门。要得不。"

女主角低头。看见了铁盒。看见了挂号单。
再抬头看婆婆。
脸上从紧张变成了困惑——她在重新理解这句话。
这不是投诉。不是发难。
是别的事。

婆婆（停顿。声音更低）：
"要是我哪天没开门。钥匙在物业。墙上是我儿子电话。他在国外。"
（停）
"拜托老。"
```

**反转**：刚才的紧张感，在一瞬间被这句话化解了。婆婆不是在指责她——婆婆是在把自己最后的安全感，交给唯一每天见到的人。

**台词**：婆婆 3 句 | 女主角 0 句

---

### 段落3：不是同情。是回应。（18-31秒）

```
镜头从两人同框缓缓推到女主角脸部近景。

"拜托老"三个字落下后。
女主角没有立刻哭。

她先是眼眶红了。
然后用力咬了一下下唇。想忍住。
（停半拍）
没忍住。
眼泪慢慢从眼眶滚下来。顺着脸颊滑落。
她没擦。

然后她低下头。看了看婆婆拉着她的手——
布满皱纹的手。紧紧抓着她年轻的手腕。

她慢慢翻转手掌。反手握住婆婆的手。
两只手——一只年轻白皙，一只苍老枯瘦——握在一起。

（停2秒。挂钟滴答。两个人就这么握着手。谁也不说话。）

女主角（重庆话，声音很轻，有点哑）：
"婆婆。我每天都来。"
```

**情感落点**：不是「我答应你」。是「我每天都来」——把她的请求，变成了自己的承诺。前面 2 秒的沉默，比哭更有力量。

**台词**：女主角 1 句（"婆婆。我每天都来。"）

---

### 段落4：门里门外。（31-36秒）

```
固定机位。女主角走到门口。回头看。

婆婆站在客厅中间。阳光斜斜地照在她身上。
挂钟还在滴答。全家福在墙上。

女主角推开门。走出去。
门轻轻关上。

画面定格在门口的龙眼山桶装水。

字幕淡入：
「每天送到。每天看见。」

```

**台词**：无。
