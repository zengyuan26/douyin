# 短视频_考场外38°C送水_龙眼山矿泉水

## 📹 短视频脚本

### 标题
```
考场外38°C，家长站了3个小时，我送了100瓶水
```

### 拍摄形式
```
手机实拍 vlog · 小周本人出镜 · 不摆拍 · 真实记录
```

---

### 分镜脚本

#### 镜头1：开场（0-5秒）— 考场门口，手机自拍

```
📱 【画面】：小周站在考场门口，手持手机自拍。
背景是考场围栏、警戒线、等在外面的家长。

🗣️ 【口播】：
"今天高考第一天，我现在在XX中学考场门口。
现在温度38度，你看我身后这些家长——"
（侧身让镜头拍到身后的家长）

📝 【字幕】：今天高考第一天 · 重庆XX中学考点
```

**拍摄要点**：不需要美颜，不需要稳定器。手机拿稳就行。"活人感"就是：头发被汗粘在额头上，阳光刺眼睁不开眼，这些全留着。

---

#### 镜头2：现场画面（5-15秒）— 拍家长

```
📱 【画面】：手机镜头扫过考场外场景——
- 穿红旗袍的妈妈，手里攥着向日葵
- 坐在马路牙子上刷手机等孩子的爸爸
- 扇着广告扇子、不停看手表的大爷（爷爷）
- 树荫下站了一排人，谁也不说话

🗣️ 【口播/旁白】（边走边轻声说）：
"穿旗袍的妈妈，寓意旗开得胜。
攥向日葵的爸爸，盼一举夺魁。
他们从早上七点就站在这儿了，
中间不敢喝水，不敢上厕所，不敢走远。"

📝 【字幕】：早上7点站到现在 · 不敢喝水 · 不敢走远
```

**拍摄要点**：拍背影、侧面即可，不要正对家长脸，尊重隐私。用长焦或走近了拍背影/手/道具细节。

---

#### 镜头3：情绪高点（15-25秒）— 给家长送水

```
📱 【画面】：小周拿着一瓶水，走向一位家长。
（最好是看起来明显疲惫的家长，比如年纪大的爷爷奶奶、嘴唇干裂的）

🗣️ 【同期声】（收现场音，不要配乐压掉）：
小周："阿姨，喝瓶水吧。"
家长："谢谢你啊……"（可能摆手、可能接过、可能愣一下）
小周："没事没事，您等了多久了？"
家长："三个多小时了……"

（这段对话不勉强，有什么算什么，真实最重要）

📝 【字幕】：在考场外送了100瓶水 · 听到最多的话是"谢谢"
```

**拍摄要点**：
- 这段的"真实感"是核心。不要设计台词，不要演。
- 如果家长不愿意入镜，就用背影+打码，只收声音。
- 如果这段没拍到好的对话，不要硬拍——直接切旁白+空镜。

---

#### 镜头4：情感升华（25-35秒）— 自拍独白

```
📱 【画面】：小周把手机架在旁边，自己坐在路边台阶上。
满头汗，手里拿着一瓶龙眼山水，对着镜头说话。

🗣️ 【口播】：
"有一个妈妈说了一句话，我当时差点没绷住。
她说——
'不敢喝水，怕上厕所，怕刚好那几分钟，孩子出来找不到我。'
（停顿1秒）
12年。就等这两天。
家长连口水都不敢喝。"

📝 【字幕】：12年，就等这两天。家长连口水都不敢喝。
```

**拍摄要点**：这段是情绪核心。说话慢一点，不用稿，用你自己的话说也行。如果说到这儿情绪上来了，不用憋着——真实的情绪就是最好的内容。

---

#### 镜头5：品牌落点 + 收尾（35-45秒）— 回到水

```
📱 【画面】：小周站起来，镜头拍到旁边放着的龙眼山矿泉水箱子。
或者小周拿着一瓶水对着镜头。

🗣️ 【口播】：
"我是小周，在重庆做龙眼山矿泉水的。
綦江的富锶水，泡茶好喝，平时喝也好。
（举一下手里的水）
今天送了100瓶，不多。
但至少让100个家长，不用渴着等孩子出来。
明天继续送。"

📝 【字幕】：龙眼山天然涌泉 · 綦江富锶矿泉水 · 明天继续送
```

**拍摄要点**：品牌的出场要轻。前面情绪堆满了，这里不需要硬广，只需要让人知道"谁在做这件事"。Logo可以出现在：水瓶标签、纸箱、或者最后定格画面加一行小字。

### 🎵 背景音乐建议

| 段落 | 音乐 | 说明 |
|------|------|------|
| 镜头1-2 | **不用音乐** | 现场环境音（蝉鸣、广播声、远处车声）比任何音乐都好 |
| 镜头3 | 轻钢琴渐入 | 舒缓、不抢情绪（比如《萱草花》纯音乐、或者抖音热门温暖BGM） |
| 镜头4 | 钢琴延续 | 音量压低，人声为主 |
| 镜头5 | 音乐渐强到收尾 | 温暖、有力量感 |

> ⚠️ 绝对不要用激昂/快节奏的音乐，会毁掉情绪。
