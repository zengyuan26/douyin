## 六、内容体系

### 内容定位

**不是教工厂老板怎么做工厂。不是讲投资知识。不是AI教程。**

是**学习、判断、做事的过程**。今天学到一个概念 → 想起以前踩过的坑 → 这两件事连起来讲一个道理。观众跟着你看，不是被你拆。

### 为什么内容池无限

每天学新东西 → 每个概念都能联系过去的经历 → 每个联系都是一期内容。不是做好了再讲，学着讲着就是内容。

### 两条内容线

| 线 | 不是 | 是 |
|------|------|------|
| **项目路** | 微电网进展汇报 | 我在学投资，发现了什么。翻译成对你有用的 |
| **来时路** | 我的个人故事 | 我过去的经历，让我今天怎么看问题。你看问题的框架是怎么长出来的 |

**两条线的关系**：同一件事的两个面。学投资（项目路）产生新认知，过去的经历（来时路）解释为什么你这样看问题。

**信任在第一优先。信任在过程中自然建立。**

### 五种内容生成模式

| 模式 | 公式 | 举例 |
|------|------|------|
| 学到一个概念 | 今天学到的 + 联想到过去的经历 + 一个道理 | 学"定价权"→ 想起做电商时被压价 → 为什么你的工厂不敢涨价 |
| 看到一个信号 | 行业信号 + 产品经理框架拆解 + 判断 | 电改政策 → 用户痛点在哪里 → 谁会受益谁会死 |
| 踩过一个坑 | 过去的坑 + 现在做投资怎么避免 + 方法 | 创业时库存压死 → 投资看项目第一看现金流 |
| 做了一个工具 | AI做了什么 + 过程分享 + 观众看到可能性 | 用AI帮自己算电费 → 3天做完 → AI还能这么用 |
| 推进一步项目 | 这一步 + 学到了什么 + 复盘 | 巴南谈下来了 → 这一轮最大的认知：XXX |

### 内容定位演进

```
阶段1（已过去）：刚入行，跟你一样从零开始
阶段2（当前）  ：跨界学习者。产品经理×AI×创业×投资，学着讲着
阶段3（未来）  ：投了多个项目、有自己方法论的投资人
```

### 核心受众

**首要目标：工厂老板 / 企业直接负责人**

他们为什么看你：因为你的视角跟他们认识的所有人都不一样。产品经理的拆解习惯 + 创业的实战经验 + AI的工具能力 + 投行的判断框架——这个组合，看到的东西天然不一样。

**次要目标：同龄转行者**（35-50岁，想转行但不知道往哪走的人）

来时路内容对他们天然有共鸣。但不是煽情——是我做到了什么、怎么做到的、你也能。

**同一期内容，两种人看，两种感受。不矛盾。**

### 项目路：给工厂老板的内容

不是教他做工厂。是**你的跨界框架让他看到了他之前没看到的东西**。

| 方向 | 说明 | 信任逻辑 |
|------|------|---------|
| **商业理念** | 学到投资概念 → 翻译成工厂老板听得懂的 | "这个人看问题的框架比我高一个维度" |
| **直接解决** | 算账、避坑、政策翻译。能直接用的 | "他真能帮我省电费" |
| **育儿/家庭** | 工厂老板40-55岁，孩子教育接班是真正痛点 | "这人跟我一样，也是当爹的" |

### 来时路的双受众设计

```
同一期来时路 →

  工厂老板看到 → "这人40岁转行，敢想敢干，靠得住"
  同龄转行者看到 → "我也40岁，我也想转。他是怎么走的"
```

### 个人品质：不自己说，让故事长出来

| 品质 | 让哪个故事来说 |
|------|-------------|
| 敢作敢当 | 40岁从互联网跨到投行，从头开始 |
| 执行力强 | 巴南落地、山东推进——每一步都真实的 |
| 坚韧不屈不挠 | 14年互联网起伏——四次从零开始 |
| 靠得住 | 合伙人说"看人先看做人"——让合伙人替你说 |
| 价值观正 | 投行看长期、看结构，不做一锤子买卖 |

### 内容原则

- **信任优先**：每一期内容结束后，观众更信任你了吗？这是唯一标准
- **学着讲着**：不装专家。今天学到的东西，今天就翻译出来
- **跨界视角是核心武器**：产品经理+创业+AI+投行——这个组合看到的东西天然不一样
- **项目进展持续更新**。巴南、山东——每一条都是信任锚点
- **18年互联网是底色不是标签**——需要的时候天然在那里，不需要每期提
- **不招居间人。不提居间人。**
- **记录即资产**——每一期都是未来知识付费的素材

### 情绪锚点

每条视频一个"魂"——观众看完心里留下的那一个感觉。先定锚点，再动笔。详见 [情绪锚点.md](知识库/方法论库/情绪锚点.md)。

**来时路常用**：破除恐惧 / 被理解的踏实 / 我也能 / 这人真实 / 敬佩 / 被看见 / 释然 / 期待
**项目路常用**：刷新认知 / 实用惊喜 / 紧迫感 / 后怕 / 优越感 / 被尊重 / 心安 / 兴奋
**育儿/家庭**：会心一笑 / 被理解 / 温暖

### 情绪调性

松弛真实。投行视角但不装大佬。像朋友分享"我们最近在看的一个方向"。笑着讲，但内容有分量。
