## 六、内容体系

### 内容定位演进

```
阶段1（已过去）：刚入行，跟你一样从零开始
阶段2（当前）  ：CVC投行投资人。看项目、学投资、拆解行业信号
阶段3（未来）  ：投了多个项目、有自己方法论的投资人
```

### 两条内容线

| 线 | 方向 | 一句话 |
|------|------|------|
| **来时路（职业线）** | 互联网→CVC投行的转行历程 | 怎么从18年互联网跨到投行。学到的、踩坑的、判断的。短期学投资，长期做投行 |
| **项目路（项目线）** | 具体项目的真实进展 | 当前是微电网。怎么选、怎么谈、怎么落地。以后可能还有其他项目 |

**两条线的关系**：来时路让人认识你、信任你。项目路让人看到你真在干、每一步都看得见。交替推进。

**信任是第一优先。信任在来时路中取得。**

**时间维度**：
- 短期 → 项目路为主。微电网项目进展、算账、落地
- 长期 → 来时路为主。职业轨迹、投资认知迭代、投行方法论

### 核心受众

**首要目标：工厂老板 / 企业直接负责人**

**次要目标：同龄转行者**（35-50岁，想转行但不知道往哪走的人）。来时路内容对他们天然有共鸣——"40岁从头开始，他怎么做到的"。

同一期来时路内容，两种人看，两种感受。不矛盾。

### 项目路：给工厂老板的三类内容

| 方向 | 说明 | 信任逻辑 |
|------|------|---------|
| **商业理念** | 投行视角看商业——定价权、交易结构、信号和噪声。帮工厂老板换一副眼镜看自己的生意 | "这个人看问题的框架跟我认识的人不一样" |
| **直接解决** | 微电网算账。省多少、怎么省、坑在哪 | "他真能帮我省电费" |
| **育儿/家庭** | 工厂老板40-55岁，孩子在上学或刚工作。教育、接班——是他们半夜睡不着的问题。你在家带孩子、陪母亲，有素材 | "这人跟我一样，也是当爹的。他说的我信" |

### 来时路的双受众设计

```
同一期来时路 →

  工厂老板看到 → "这人40岁转行，敢想敢干，靠得住"
  同龄转行者看到 → "我也40岁，我也想转。他是怎么走的"
```

### 个人品质：不自己说，让故事长出来

线下受老板器重的品质——敢作敢当、执行力强、坚韧、靠得住、价值观正——**不在内容里自己说。让每一步经历替你说。**

| 品质 | 让哪个故事来说 |
|------|-------------|
| 敢作敢当 | 40岁从互联网跨到投行，从头开始 |
| 执行力强 | 巴南落地、山东推进——每一步都真实的 |
| 坚韧不屈不挠 | 14年互联网的起伏——电商第一→合伙人被抓→倒卖口罩→腾讯败诉→父亲去世→爬起来 |
| 靠得住 | 合伙人说"看人先看做人"——让合伙人替你说 |
| 价值观正 | 不是唯利是图——投行看长期、看结构，不做一锤子买卖 |

### 内容原则

- **信任优先**：每一期内容结束后，观众更信任你了吗？这是唯一标准
- "我们在看这个方向" > "我们在判断" > "我们在学" —— 投行视角切入
- 投行角度不是炫技，是看问题的框架不一样——领先指标、定价权、交易结构
- 项目进展持续更新。巴南、山东——每一条都是信任锚点
- 18年互联网是底色不是标签——需要的时候天然在那里，不需要每期提
- 投行看的是信号和结构，媒体报道看的是事件和故事
- **不招居间人。不提居间人。内容里不出现这个词。**

### 记录即资产

来时路的每一期内容都是未来知识付费的素材。不需要现在就决定产品形态（课程/社群/书）。只需要知道：**每一次记录都是有价值的。** 积累够了，产品形态会自然出来。

### 情绪锚点

每条视频一个"魂"——观众看完心里留下的那一个感觉。先定锚点，再动笔。详见 [情绪锚点.md](知识库/方法论库/情绪锚点.md)。

**来时路常用**：破除恐惧 / 被理解的踏实 / 我也能 / 这人真实 / 敬佩 / 被看见 / 释然 / 期待
**项目路常用**：刷新认知 / 实用惊喜 / 紧迫感 / 后怕 / 优越感 / 被尊重 / 心安 / 兴奋
**育儿/家庭**：会心一笑 / 被理解 / 温暖

### 情绪调性

松弛真实。投行视角但不装大佬。像朋友分享"我们最近在看的一个方向"。笑着讲，但内容有分量。
