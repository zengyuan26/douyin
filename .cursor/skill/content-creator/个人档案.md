## 四、商业模式：发牌人

### 角色定位

基金朋友做多个项目。我帮他承担居间人网络这一块——找居间人、培训、给工具、最后成交。这是我现在手上的起步项目，后面可能有别的。

### 三层结构

```
基金朋友（管钱、看方向、投多个项目）
  └── 你（CVC新人，帮他负责居间人这块）
        ├── 找居间人（线上内容获客）
        ├── 培训 + 给销售工具
        ├── 最后成交（跟基金朋友一起）
        └── 融资 + 项目落地（基金出钱）

居间人（你从线上找来的）
  ├── 负责：用本地人脉接触工厂老板
  ├── 负责：用你给的培训+工具去谈
  └── 谈差不多了交给你。靠佣金吃饭。

工厂（居间人找的）
  └── 签合同、装设备、省电费
```

| 角色 | 职责 |
|------|------|
| CVC合伙人 | 资金+行业认知+方向判断+最后成交 |
| 增元 | 线上获客（找居间人）+ 培训体系 + 销售工具 + 品牌 |
| 居间人 | 本地获客 + 初步洽谈 + 交棒给我们成交 |

### 关键逻辑

- 你是发牌人，不是居间人。居间人是你下面的渠道。
- 你的内容做两件事：① 吸引居间人来找你 ② 让工厂老板认识你
- **信任锚点**：不说"我投了"，说"我们在做"。不说"我懂"，说"合伙人刚教我的"。

## 六、内容体系

### 六条内容线

| 编号 | 线 | 占比 | 一句话 |
|------|------|------|------|
| **T** | **合伙人教的** | **25%** | 合伙人刚教会我的。他说的话、他的判断、他看项目的方法 |
| **W** | **我们在做什么** | **20%** | 我们在投什么、为什么、项目进展。不装"我在投"，说"我们在做" |
| **K** | **概念图谱（PPT+录音）** | **15%** | 每周一期长视频（5-10min），讲透一组能源概念的关系框架 |
| P | 政策翻译 | 15% | 新政策出来，我们的判断——利好什么、利空什么 |
| C | 互联网视角 | 15% | 18年互联网看能源。电商、平台、算法这些词，能源行业的人不会用 |
| R | 真实进展 | 10% | 项目推进、踩坑、签约。让观众看到你是真的在干 |

### 受众漏斗（内容服务的核心目标）

```
泛人群（看热闹）
  ↓
有工厂人脉的人（居间人潜客）← 内容首要目标
  ↓
私信"居间" → 领操作手册 → 进培训体系
  ↓
出单 → 我们成交 → 分佣金
```

**内容做两件事：**
1. 吸引居间人：你在找有工厂资源的人。让他们看到这个机会。
2. 建立信任：真项目（巴南山东）、真合伙（CVC）、真培训体系。

### 内容原则

- "我们在做" > "我在学" > "我在看" —— 优先从投资角度切入
- "刚入行"不是弱点，是真实。大佬说自己不懂是谦虚，新人说自己刚学是真实
- 项目进展持续更新。巴南签约、山东推进——每一条都是信任锚点
- 18年互联网是底色不是标签——不需要每期都提，需要的时候它天然在那里
- CVC合伙人是信任锚点不是炫耀资本——自然提及，不刻意强调
- 每期内容的隐藏钩子：让有工厂资源的人觉得自己也能干

### 情绪调性

松弛真实。像朋友分享"我们老板今天又教了我什么"。笑着讲，但内容有分量。
