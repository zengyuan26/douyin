## 六、内容体系

### 内容定位

**AI让信息平权了。但散落的信息拼成框架、给出判断——这个能力AI替代不了。**

你的内容不是讲知识。是把你看到的信息拼成判断，让别人拿走就能用。每天学到的东西 → 整理成框架 → 给出判断。

### 为什么内容池无限

你天然在收集信息——新能源行业信号、投资框架、AI工具、身边人案例、读到的书。把这些散落的信息拼起来、给出判断，就是内容。不需要额外创作。

### 内容四象限

```
                          看外部（项目/机会/行业）
                ┌──────────────────────────┐
         框架   │  行业判断               │  框架判断              │
         拆解   │  新能源/AI信号          │  投资框架/商业判断     │
                │  政策翻译/趋势拆解      │  跨领域思维模型        │
                │                          │  芒格式认知工具        │
                ├──────────────────────────┤
         故事   │  实战判断               │  来处判断              │
         经历   │  项目进展/谈判           │  来时路/身边人故事     │
                │  帮人看项目的案例        │  经典拆解→框架        │
                │                          │  读书快拆/认知分享     │
                └──────────────────────────┘
                          看内部（人/自己/认知）
```

**两条轴**：
- 横轴：看外部（项目/机会/行业）↔ 看内部（人/自己/认知）
- 纵轴：框架拆解 ↔ 故事经历

**四个象限**：

| 象限 | 是什么 | 内容举例 |
|------|------|------|
| **行业判断** | 新能源/AI的信号和趋势。政策翻译。用投行视角看行业变化 | 15分钟电价意味着什么、CBAM对出口工厂的影响 |
| **框架判断** | 投资框架、商业底层逻辑、芒格式思维模型。不限行业 | 定价权/护城河/规模效应vs网络效应/反过来想 |
| **实战判断** | 项目真实进展、谈判细节、帮人看项目的案例 | 巴南落地学到了什么、朋友工厂怎么踩的坑 |
| **来处判断** | 判断力从哪来——经历、身边人、经典拆解、认知分享 | 四次从零开始、读芒格的拆解、身边人怎么选错的 |

### 来处判断 · 四个来源

| 来源 | 是什么 | 举例 |
|------|------|------|
| 来时路 | 个人经历，框架怎么长出来的 | 四次从零开始→韧性不是坚持，是知道自己擅长什么 |
| 身边人 | 观察到的案例，别人的选择和结果 | 朋友工厂踩坑→提炼出共性判断 |
| 经典拆解/读书分享 | 一本书/一个模型→快速拆→整理成框架。读得快拆得快，等于帮你读了 | 读芒格→提炼"反过来想"→用在项目判断上 |
| 认知分享 | 信息散落时代，你怎么收集整理形成判断 | "这个结论不是看了一篇新闻，是我把三件事拼在了一起" |

### 各阶段配比

| 阶段 | 行业判断 | 框架判断 | 实战判断 | 来处判断 | 逻辑 |
|------|---------|---------|---------|---------|------|
| **阶段1** 冷启动 | 25% | 15% | 15% | **45%** | 先让人知道你是谁、判断力哪来的、框架怎么不一样 |
| **阶段2** 信任建立 | 25% | 30% | 20% | 25% | 框架判断加重，展示专业深度 |
| **阶段3** 私域转化 | 20% | 35% | 25% | 20% | 框架+实战为主，高净值人群从这两块筛出来 |

### 内容定位演进

```
阶段1（冷启动）：让观众知道你是谁 → 来处判断45%，先用经历和框架建立信任底盘
阶段2（信任建立）：让观众信你的判断 → 框架判断提到30%，行业+框架双线并重
阶段3（私域转化）：高净值人群筛选出来 → 框架+实战为主，价值进私域
```

### 核心受众

**有资产的人**——工厂老板、企业主、高净值个人（可投资产100万+）。

他们为什么看你：你对信息的判断力跟他们认识的所有人都不一样。实战喂出来的直觉 + 芒格式框架整理 + AI工具能力 + 投行视角——这个组合，看到的东西天然不一样。

**次要目标**：同龄转行者（35-50岁，想转行但不知道往哪走的人）。

来时路内容对他们天然有共鸣。不是煽情——是我做到了什么、怎么做到的、你也能。

**同一期内容，两种人看，两种感受。不矛盾。**

### 情绪锚点

每条内容一个"魂"——观众看完心里留下的那一个感觉。

**来处判断常用**：破除恐惧 / 被理解的踏实 / 我也能 / 这人真实 / 敬佩 / 被看见
**行业判断常用**：刷新认知 / 紧迫感 / 后怕 / 原来如此
**框架判断常用**：实用惊喜 / 被尊重 / 兴奋 / 优越感
**实战判断常用**：心安 / 这人靠得住 / 期待后续

### 情绪调性

直言直语，做人硬派。投行视角但不装大佬。像朋友分享"最近我们看到了什么"——不是讲课，是给你看我的判断框架。
