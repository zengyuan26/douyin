# 发牌人行动路线图

> 上次更新：2026-06-27 | 常回来对进度

---

## 阶段1：验证模式（第1-4周）

**目标：验证"内容吸引的陌生人"能变成居间人**

- [ ] 调整内容方向：每期视频结尾明确"找有工厂资源的人"
- [ ] 制作居间人操作手册（已有基础，更新）
- [ ] 私信流程标准化（"居间" → 自动回复 → 手册 → 筛选）
- [ ] 每天回复私信，找有意向的人聊

**里程碑：**
- [ ] 10个有意向
- [ ] 5个进培训
- [ ] 1个开始跑客户

---

## 阶段2：居间人培训体系（第5-8周）

**目标：居间人进来后有标准路径可走**

- [ ] 录制8节培训课（已有草稿 → 视频版）
- [ ] 做销售工具包：计算器 + 提案模板 + FAQ
- [ ] 跑通第一个居间人全流程（培训 → 出单 → 成交 → 分佣）
- [ ] 把全流程写成 SOP

**里程碑：**
- [ ] 1个非熟人居间人成交
- [ ] 培训体系试跑成功

---

## 阶段3：复制（第9-16周）

**目标：从1个变成5个活跃居间人**

- [ ] 内容切换：从"我们在做什么" → "居间人成功故事"
- [ ] 居间人分级：新人 → 活跃 → 城市合伙人
- [ ] 制作居间人成功案例系列（最强的招募内容）

**里程碑：**
- [ ] 5个活跃居间人
- [ ] 月均项目线索 ≥ 10条

---

## 阶段4：规模化（第17周+）

**目标：网络自运转，你抽身**

- [ ] 居间人带居间人（层级分佣）
- [ ] 培训完全标准化（不用你讲）
- [ ] 基金朋友其他项目复用这张渠道网
- [ ] 你从"搭渠道" → "管渠道"

**里程碑：**
- [ ] 10+ 活跃居间人
- [ ] 你的时间投入下降 50%

---

## 三条核心原则

1. **你是发牌人不是跑渠道的。** 内容找居间人 → 他们跑客户 → 你成交。
2. **媒体杠杆是引擎。** 没有内容就没有居间人。
3. **先验证再铺量。** 第一个陌生居间人出单，才算跑通。
