## 一、机会本质

**你做的事，本质是"用内容建销售渠道"。**

基金的资源（资金+行业判断+项目落地能力）是供给。你通过抖音内容找到"有工厂人脉但不知道怎么变现"的人，把他们变成居间人。你给他们培训+工具，他们负责本地获客。谈差不多后，你和基金朋友去成交。

这不卖设备、不跑工地。**你在牌桌中间收分发牌。**

## 二、项目筛选（先用标准筛一遍）

用 skill 的"加盟流量引擎"标准过一下，看你这个项目匹配度：

| 标准 | 判断 | 匹配？ |
|------|------|--------|
| 客户决策快（几周） | 居间人决策快（几天），工厂端慢（3-8月） | ⚠️ 部分 |
| 有复购 | 居间人可出多单，后续服务有持续收益 | ✅ |
| 客户群体大 | 5-8万家出口工厂，潜在居间人群体也大 | ✅ |
| 视觉可展示 | 数字、图表、案例可见，但非视觉冲击型 | ⚠️ 部分 |
| 不需要专业背景 | 居间人需要工厂人脉 + 基础行业认知 | ⚠️ 需要筛选 |
| 客单价适中 | 居间费20-50万/单，对应工厂改造600-1000万 | ⚠️ 偏大 |

**结论：不完全匹配"加盟流量引擎"模型，更像"销售代理 + 平台赋能"模型。** 这个偏离不一定是坏事——门槛高意味着竞争少。skill 说"不匹配就换项目或换模式"，你的情况是：项目定了，模式调一下就行。

## 三、纳瓦尔框架分析

### 1. 定位分析

```
                    高价值输出              低价值输出
              ┌─────────────────┬─────────────────┐
      擅长    │  ★ 发牌人        │   培训执行      │
              │  内容获客+找渠道 │   做课件/写手册  │
              │  品牌建设        │   可AI辅助       │
              ├─────────────────┼─────────────────┤
      不擅长  │  行业深度判断    │   施工/售后      │
              │  合伙人补位      │   不应该是你做   │
              └─────────────────┴─────────────────┘
```

**核心定位：发牌人**

你的甜蜜区在第4象限（擅长且高价值）：用18年互联网经验+AI能力做内容获客，找到居间人。行业判断交给基金朋友，施工售后有团队。

你不是居间人，你不是工厂老板，你不是投资人。**你是建渠道的人。**

价值层级：

│ ★ 层级4：认知差价值  ← 你的核心                              │

你是层级3+4的结合——认知差让你能被居间人信任，匹配能力让你能撮合交易。

### 3. 经验复利分析

```
复利判断：

✅ 内容资产复利
   → 每发一条视频，就是一块积木。100条视频 = 100条"销售员"
   → 越积累越多，新居间人会翻你以前的内容

✅ 培训体系复利
   → 培训材料迭代越做越好。案例越多越有说服力
   → 第一个居间人最难，第十个就容易了

✅ 行业认知复利
   → 跟基金朋友看得越久，判断越准
   → 3年后你不再是"新人"，你是有实操履历的人

✅ 案例复利
   → 巴南落地 = 1个案例。10个案例 = 不可撼动的信任
   → 案例是最好的销售工具

⚠️ 但有一个复利陷阱
   → 你的"成交能力"如果变成居间人依赖你成交
   → 你就抽不了身。这个需要注意分离
```

**结论：这是复利业务。做得越久越值钱。** 客户会为"你帮多少人挣到了佣金"付更多信任。

**你的杠杆组合很健康。** 四个杠杆都在用，不是单腿走路。

## 四、八条关键问题

```
□ 0. 通过筛选了吗？
   → 不是完美匹配加盟流量模型，但作为销售代理+平台模式成立。
     高客单价、长周期对应的是高佣金，逻辑对。

□ 1. 机会的本质是什么？
   → 用内容把"散落在各地有工厂人脉的人"组织成一张销售网络。

□ 2. 你的定位是什么？
   → 发牌人。建渠道的不是跑渠道的。

□ 3. 你能提供的独特价值？
   → AI获客能力 + 基金背书 + 培训体系 + 成交能力。
     居间人自己做不了的是：没项目、没资金、没品牌、不会谈。

□ 4. 能用的杠杆？
   → 媒体（抖音）> 人力（居间人）> 代码（AI工具）> 资本（基金）
     媒体杠杆最优先，因为它是所有其他杠杆的起点。

□ 5. 子弹能否剥离？
   → 工具可剥离，培训可半剥离，成交和品牌暂时不能。
     现阶段不要强求剥离。先把渠道跑通再考虑。

□ 6. 关键决策点？
   → 第一：第一个非熟人居间人什么时候出单？（验证模式）
     第二：你和基金朋友的分账比例和权责边界。（必须书面化）

□ 7. 风险是什么？
   → 资金：不是你出钱，低风险
     声誉：居间人乱承诺害你的品牌 → 需要培训体系兜底
     时间：如果居间人不成交，你的时间投入没回报
     关系：和基金朋友的角色边界模糊 → 必须明确

□ 8. 退出机制？
   → 如果不行：内容资产是你的（可以复用），培训体系是你的，品牌是你的
     如果行：居间人网络可以层级化，你升到更上层
     两种情况你都不会赔。保护好自己的IP。
```

## 六、最终建议

### ✅ 值得做的理由

- **你的能力匹配**：18年互联网获客 + AI工具能力 + 基金背书，这个组合别人很难复制
- **杠杆结构健康**：四根杠杆都在用，媒体做主引擎，代码做差异化
- **经验值钱**：这是复利业务。50个案例比5个案例强10倍，品牌越来越值钱
- **天花板高**：你现在做居间人网络，后面基金朋友的其他项目也可以复用这张网
- **风险可控**：不是你出钱。你的投入是时间和内容

### ⚠️ 需要谨慎的

- **第一单验证**：从"熟人介绍"到"内容找来的陌生人"，这个跳跃需要验证。你的第一个非熟人居间人什么时候出单？
- **权责边界**：你和基金朋友之间，分账比例和决策权要书面明确。不是不信任，是避免以后纠纷
- **不要亲自下场做居间**：你的定位是发牌人。一旦你开始自己跑客户，就退回了子弹在身体里的阶段

### 💡 最佳策略

```
第一步：跑通一个非熟人居间人（验证模式）
  → 用内容吸引 > 私信筛选 > 进培训 > 他出单 > 你成交
  → 目标：3个月内跑通第一个非熟人案例

第二步：标准化培训 + 工具（降低复制成本）
  → 录制成套培训视频（居间人自己看）
  → 做销售工具包（计算器、提案模板、FAQ话术）
  → 目标：新居间人进来，90%问题自己搞定

第三步：复制（规模化）
  → 居间人带居间人（层级化分佣）
  → 你的内容从"找居间人"扩展为"晒居间人成功案例"
  → 目标：10个活跃居间人同时跑

第四步：平台化（远期）
  → 基金朋友其他项目也可以用你的渠道网络
  → 你不是备电的增元，你是"帮项目建渠道"的增元
```

### 🔑 复利判断

**这是经验值钱的业务。** 内容积累、案例积累、培训体系积累，每一样都在帮下一单降低成本。你不需要重新发明自己。1年后的增元和今天的增元，品牌价值和成交能力会有质变。

## 七、行动路线图

### 阶段1：验证模式（第1-4周）

```
目标：验证"内容吸引的陌生人"能变成居间人

任务：
□ 调整内容方向：每期镜头前+结尾明确说"找有工厂资源的人"
□ 制作居间人操作手册（已有基础，更新后可用）
□ 私信流程标准化（"居间"→自动回复→手册→筛选）
□ 每天回复私信，找到3个有意向的人聊

关键指标：私信"居间"的人数
里程碑：10个有意向 + 5个进培训 + 1个开始跑客户
```

### 阶段2：居间人培训体系（第5-8周）

```
目标：居间人进来后有标准路径可走

任务：
□ 录制8节培训课（已有草稿，录制成视频版）
□ 做销售工具包：计算器 + 提案模板 + FAQ
□ 跑通1个居间人全流程（培训→出单→成交→分佣）
□ 把流程写成 SOP

里程碑：1个非熟人居间人成交 + 培训试跑成功
```

### 阶段3：找居间人 → 复制（第9-16周）

```
目标：从1个变成5个活跃居间人

任务：
□ 内容重点切换：从"我们在做什么"到"居间人故事"
  （居间人自己出镜讲"我是怎么挣到这个佣金的"）
□ 居间人分级：新人→活跃→城市合伙人
□ 制作居间人成功案例系列（最强的招募内容）

里程碑：5个活跃居间人，月均项目线索≥10条
```

### 阶段4：规模化（第17周+）

```
目标：网络自运转

任务：
□ 居间人带居间人（层级分佣）
□ 培训完全标准化（不用你做）
□ 基金朋友其他项目也可以用这张网
□ 你从"搭渠道"自然过渡到"管渠道"

里程碑：10+活跃居间人，你的时间投入下降50%
```

---

**核心记住三件事：**

1. **你是发牌人不是跑渠道的。** 内容找居间人 → 他们跑客户 → 你成交。这个链条不要乱。
2. **媒体杠杆是引擎。** 没有内容就没有居间人。没有居间人你就是在做销售代理。
3. **路走对了但还没证明。** 巴南和山东是熟人。你需要一个从零找来的陌生人出单，才算真正验证模式。
