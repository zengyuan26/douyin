# 图文内容模板

> **【选题】** 茶楼茶具损耗降本方案
>
> **【核心文案】** 普洱茶杯摔碎了别扔 老师傅有办法修好
>
> **【目标客户】** 茶楼、茶室、茶馆
>
> **【行业】** 茶饮 / 茶文化
>
> **【强制规格】** 图文 | 不限张数 | 9:16竖版 1080×1920px
>
> **【核心要求】** 严禁纯色背景！每张图背景必须与内容相关

---

## 一、选题策略

### 1.1 内容定位

| 项目 | 内容 |
|------|------|
| **【核心痛点】** | 茶楼茶具损耗大，紫砂壶、普洱杯等名贵茶具破损后成本高 |
| **心理策略** | 实用价值 + 文化认同 - 懂茶的人都惜器 |
| **内容类型** | B端获客 + 干货价值 |
| **目标人群** | 茶楼老板、茶馆主理人、茶室管理者 |
| **内容目的** | 痛点场景 → 文化价值 → 方案展示 → 咨询转化 |

### 1.2 选题逻辑

```
茶楼茶具痛点 → 损耗原因 → 文化价值 → 修复方案 → 信任转化
```
