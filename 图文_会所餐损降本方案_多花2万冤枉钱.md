# 图文内容模板

> **【选题】** 会所/私房菜馆餐损降本方案
>
> **【核心文案】** 一个月多花2万？你不知道的会所餐损真相
>
> **【目标客户】** 高级会所、私房菜馆、私人餐厅
>
> **【行业】** 高端餐饮 / 私人会所
>
> **【强制规格】** 图文 | 不限张数 | 9:16竖版 1080×1920px
>
> **【核心要求】** 严禁纯色背景！每张图背景必须与内容相关

---

## 一、选题策略

### 1.1 内容定位

| 项目 | 内容 |
|------|------|
| **核心痛点** | 高端会所/私房菜馆瓷器损耗大，成本居高不下 |
| **心理策略** | 利益诱惑 + 身份认同 - 高端场所就该用对方案 |
| **内容类型** | B端获客 + 干货价值 |
| **目标人群** | 会所老板、私房菜馆主理人、高端餐饮管理者 |
| **内容目的** | 痛点场景 → 方案展示 → 专业信任 → 咨询转化 |

### 1.2 选题逻辑

```
会所餐损痛点 → 损耗真相 → 省钱方案 → 身份匹配 → 咨询转化
```
