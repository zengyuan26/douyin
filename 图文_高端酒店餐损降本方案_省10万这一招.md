# 图文内容模板

> **【选题】** 高端酒店餐损降本方案
>
> **【核心文案】** 这一招一年至少帮你省10万以上
>
> **【目标客户】** 高端酒店（五星/精品酒店）
>
> **【行业】** 酒店餐饮 / 餐饮管理
>
> **【强制规格】** 图文 | 不限张数 | 9:16竖版 1080×1920px
>
> **【核心要求】** 严禁纯色背景！每张图背景必须与内容相关

---

## 一、选题策略

### 1.1 内容定位

| 项目 | 内容 |
|------|------|
| **核心痛点** | 高端酒店餐饮损耗大，成本居高不下 |
| **心理策略** | 利益诱惑 + 损失厌恶 - 别人都在省了你还在亏 |
| **内容类型** | B端获客 + 干货价值 |
| **目标人群** | 高端酒店管理者、餐饮总监、酒店老板 |
| **内容目的** | 痛点场景 → 方案展示 → 专业信任 → 咨询转化 |

### 1.2 选题逻辑

```
高端酒店餐损大 → 省钱方案 → 专业展示 → 信任建立 → 咨询转化
```
