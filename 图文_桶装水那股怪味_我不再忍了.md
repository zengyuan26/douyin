## 一、"人-货-场"内容逻辑

> **人在场景下用产品/服务 → 遇到问题 → 苦恼不能解决 → 情绪变化 → 两个解决方案可选**

| 要素 | 内容 |
|------|------|
| **人** | 注重生活品质的人、泡茶爱好者、家庭用户 |
| **货** | 循环桶清洗检测服务 + 一次性桶装水（龙眼山天然涌泉） |
| **场** | 日常饮水场景、泡茶场景、家庭厨房场景 |

## 八、制作规范

### 设计规格

```
图片数量：7张（9:16竖版）
主标题：7字以内
副标题：10字以内
图表为主，文字精简
每张图文字不超过30字/行
每张必须有场景图（禁止纯色/渐变背景）
背景与内容相关
禁止出现手机外框/手机Mockup
禁止使用emoji和特殊符号
严禁堆砌文字，保持简洁易读
```
