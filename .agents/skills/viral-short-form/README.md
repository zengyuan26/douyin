# viral-short-form

Write short-form that people actually save and finish. This skill brainstorms hooks, shapes scripts for retention, drafts carousels, and gives an honest read on where a draft loses people. Install it once, then ask in plain language inside the AI tool you already use. Instead of a blank page, you get several hooks across different angles, a script built to hold attention, or a straight critique of your draft.

It works across every format: talking head, demo, unboxing, before/after, tutorial, storytime, listicle, carousel, and meme. The hooks come from patterns that tend to work, and scripts follow hook, escalation, payoff, and CTA, adapted to the platform you're posting on. It tells you what tends to work and why. It won't tell you something will go viral.

## Install

```
npx skills add vyralcontent/content-skills --skill viral-short-form
```

Works in Claude, Cursor, and any agent that supports the [SKILL.md standard](https://agentskills.io).

## Use it

Ask in plain language once it's installed:

- "Give me 8 hooks for a video launching my cold-brew maker."
- "Write a 30-second talking-head script on why most morning routines fail."
- "Here's my script. Where does it lose people?"
- "Turn this blog post into a 6-slide carousel."

## What it does, and doesn't

- Does: turn a rough idea into hooks, a retention-shaped script or carousel, and an honest critique of a draft, fitted to the platform.
- Doesn't: promise virality. This is pattern-matching, not prediction. It improves the odds and gives you a structure to start from, never a guarantee.
