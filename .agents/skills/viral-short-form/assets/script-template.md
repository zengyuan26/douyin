# Short-form script template

Fill this in for a single vertical video. Keep the audio column tight (people
talk faster than they read), and make the on-screen text reinforce the spoken
line instead of repeating it. Aim for a cut/reset every 2-3 seconds.

```
WORKING TITLE:
FORMAT:            (talking head / demo / before-after / tutorial / storytime / listicle / meme)
PLATFORM:          (TikTok / Reels / Shorts)
GOAL:              (saves / reach / conversion)
TARGET LENGTH:     (sec)

──────────────────────────────────────────────────────────────────────
TIME   | SPOKEN / AUDIO                  | ON-SCREEN TEXT        | VISUAL / SHOT
──────────────────────────────────────────────────────────────────────
0-3s   | HOOK (lead with payoff/stake)  | hook restated, big    | in media res / face
       |                                |                       |
3-Xs   | ESCALATION beat 1              | key phrase            | reset shot/angle
       | (raise tension / specificity)  |                       |
       | ESCALATION beat 2              |                       | reset shot/angle
       |   ↑ use "but / therefore"      |                       |
       |                                |                       |
X-Ys   | PAYOFF (deliver the promise)   | the takeaway          | show the result
       |                                |                       |
end    | CTA (one ask)                  | CTA text              | hold / loop to start
──────────────────────────────────────────────────────────────────────

OPEN LOOP (opened in hook → closed at payoff):
RE-HOOK LINE (for the mid-video sag):
SOUND:             (trending/fitting, esp. TikTok)
PROMISE↔PAYOFF CHECK:  does the body deliver what the hook promised?  [ ]
```

Notes:
- If the hook needs a sentence of setup, the setup is probably the real hook.
  Start there.
- Cut the first and last lines last: they're usually the most over-written.
