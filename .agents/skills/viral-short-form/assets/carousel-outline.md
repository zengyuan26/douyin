# Carousel outline template

For static, swipeable formats (Instagram, LinkedIn). The job of every slide is to
earn the swipe to the next. One idea per slide, big readable text, generous
margins inside the safe zone.

```
TOPIC:
AUDIENCE:
GOAL:              (saves / shares / follows)
SLIDE COUNT:       (5-10)

SLIDE 1: HOOK
  • One bold idea or promise, large type. No logo, no intro.
  • This is a hook (see references/hooks.md), not a title page.

SLIDE 2: THE STAKES / WHY CARE
  • Why this matters now, or the cost of getting it wrong.

SLIDES 3..n-2: ONE POINT PER SLIDE
  • Slide 3: strongest practical point (front-loaded to survive early drop).
  • Slide 4: …
  • Slide 5: …
  • Keep each to a single idea + one supporting line.

SLIDE n-1: SUMMARY / RECAP
  • The whole thing distilled to a saveable list. This is the "save" slide.

SLIDE n: CTA
  • One ask: save, follow for more, or comment a keyword.
```

Notes:
- Caption (post text): repeat the hook in the first line. It shows in the feed
  before the image fully registers.
- If a slide has two ideas, split it. If a slide has none, cut it.
